Technology Detection Using Wappalyzer

El complemento Technology Detection utiliza las reglas de Wappalyzer (http://wappalyzer.com/) para detectar las tecnologias utilizadas por las aplicaciones.

Funciona de forma muy similar a los complementos del navegador Wappalyzer con las siguientes excepciones:

  • No utiliza las "Variables JavaScript Globales" debido a que son difíciles de probar sin un navegador "completo"
  • No muestra las versiones - todavía esta por hacer
  • It does not not show the confidence - this is still todo
  • Permite ver la "evidencia" utilizada para detectar las tecnologias

La pestaña Tecnología

Esta pestaña muestra todas las tecnologías detectadas en el sitio seleccionado.
Hacer clic derecho en una tecnología mostrara un menú "Mostrar evidencia" debajo del cual están todos los regexes utilizados para detectarlo.
Seleccionar un regex cambiará la pestaña "Buscar" y buscará en el historial ese regex.
