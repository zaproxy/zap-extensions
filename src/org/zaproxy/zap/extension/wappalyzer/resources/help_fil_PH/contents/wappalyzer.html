Ang Pagtuklas ng Teknolohiya Gamit ang Wappalyzer

Ang Pagtuklas ng Teknolohiya na add-on ay gumagamit ng Wappalyzer (http://wappalyzer.com/) na mga patakaran upang malaman ang mga teknolohiyang ginamit ng mga aplikasyon.

Ito ay gumagana sa paraang katulad ng Wappalyzer na browser na mga add-on na may sumusunod na mga eksepsyon:

  • Hindi ito gumagamit ng 'Global JavaScript na mga variable' habang ang mga ito ay mahirap subukan kung walang isang 'ganap' na browser
  • Ito ay hindi nagpapakita ng mga bersyon - ito ay gagawin pa
  • Ito ay hindi nagpapakita ng tiwala sa sarili - ito ay gagawin pa
  • Ito ay nagbibigay-daan sa iyo upang tingnan ang 'katibayan' na ginamit sa pagtuklas ng mga teknolohiya

Ang tab ng teknolohiya

Ang tab na ito ay nagpapakita ng lahat ng mga natukoy na mga teknolohiya para sa napiling site.
Kanang pag-klik sa isang teknolohiya ay nagpapakita ng isang 'Magpakita ng katibayan' na menu sa ilalim na kung saan ang lahat ng mga regexes na ginagamit ay makita ito.
Pagpili ng isang regex ay lumipat sa 'Paghahanap' na tab ay hanapin ang buong kasaysayan para sa regex.
