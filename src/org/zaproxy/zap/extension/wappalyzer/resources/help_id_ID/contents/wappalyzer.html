Teknologi Deteksi Menggunakan Wappalyzer

Teknologi Deteksi add-on menggunakan aturan wappalyzer (http://wappalyzer.com/) untuk mendeteksi teknologi yang digunakan dengan aplikasi.

Berfungsi dalam cara yang sama dengan pengaya browser Wappalyzer dengan pengecualian berikut:

  • Tidak menggunakan 'variabel global JavaScript' karena ada kesulitan untuk menguji tanpa browser lengkap
  • Tidak menampilkan versi - ini masih todo
  • Tidak menampilkan keyakinan - ini masih todo
  • Memungkinkan anda untuk melihat 'bukti' yang digunakan untuk mendeteksi teknologi

Tab Teknologi

Tab ini menampilkan semua teknologi yang terdeteksi untuk situs yang dipilih.
Klik kanan pada teknologi akan menampilkan menu 'Tampilan bukti' dimana semua regexes yang digunakan untuk mendeteksinya.
Memilih regex akan beralih ke tab 'penelusuran' dan menelusuri melalui riwayat untuk regex tersebut.
