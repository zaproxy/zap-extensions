Tela de opções de navegação forçada

Esta tela permite que você configure as opções de Navegação Forçada:

Threads de varredura simultâneos por host

o número de threads que a varredura usará por host. Aumentar o número de threads vai acelerar o escaneamento mas pode sobrecarregar tanto o computador onde o ZAP está rodando quando o host alvo.

Navegação Recursiva

Se marcada, então a varredura será recursiva através de todos os sub-diretórios encontrados.
Isto pode levar um longo tempo.

Arquivo default

O arquivo padrão selecionado quando o ZAP inicia.

Adicionar arquivo personalizado de navegação forçada

Permite que você adicione seus próprios arquivos para serem usados com brute force sobre arquivos e diretórios.
Eles devem ser arquivos de texto com um nome de arquivo ou diretório em cada linha.
Arquivos são adicionados ao diretório 'dirbuster' abaixo do diretório home do ZAP.
