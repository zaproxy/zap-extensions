Mendukung spesifikasi Pembuka API

Add-on ini memungkinkan anda untuk laba-laba dan impor OpenAPI (Swagger) definisi (versi 1.2 dan 2.0).

Add-on akan mendeteksi secara otomatis setiap definisi API Terbuka dan menggerakkannya selama berada di dalam jangkauan.

UI

2 item menu ditambahkan pada menu perkataan:
  • Impor sebuah definisi Pembuka API dari sistem file lokal
  • Impor sebuah definisi Pembuka API dari sistem sebuah URL
Dialog URL Impor mendukung bidang penambahan opsional yang memungkinkan anda untuk menolak Host (dan port) di definisi dengan satu alternatif yang anda suplai.
Pilihan ini tidak disediakan untuk impor berkas karena mudah untuk merubah berkas lokal untuk menggunakan host yang berbeda.

API

Operasi berikut ini ditambahkan ke API:
  • ACTION importFile (file)
  • ACTION importUrl (url, hostOverride)
Definisi akan diimpor secara serentak dan setiap peringatan akan dikembalikan.

Garis Perintah

Pilihan Garis Perintah berikut ini ditambahkan:
  • -openapifile <filename> : impor sebuah definisi Pembukaan API dari nama file yang ditetapkan
  • -openapifile <url> : impor sebuah definisi Pembukaan API dari URL yang ditetapkan
Definisi ini akan diimpor secara serentak dan peringatan apapun akan ditampilkan pada garis perintahnya.

Nilai yang Ditetapkan Pengguna

Nilai default akan digunakan ketika mengimpor definisi Pembukaan API.
Ini dapat diganti menggunakan add-on Bentuk Handler yang memungkinkan anda untuk menentukan nilai anda sendiri.
Dalam kebanyakan kasus ini akan menjadi nilai yang sederhana (seperti senar dan bilangan bulat) namun dalam beberapa kasus anda mungkin perlu untuk menentukan nilai terstruktur, misalnya.
