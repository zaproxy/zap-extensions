Scripts da comunidade

Uma coleção de scripts ZAP fornecida pela Comunidade, armazenada em https://github.com/zaproxy/community-scripts

Favor enviar seus scripts através de pull requests

Para obter mais informações sobre ZAP scripts ver:
  • https://github.com/zaproxy/zap-core-help/wiki/HelpAddonsScriptsScripts
  • https://github.com/zaproxy/zaproxy/wiki/InternalScripting
Para discutir qualquer aspecto do ZAP scripting junte-se ao grupo zaproxy-scripts: https://groups.google.com/group/zaproxy-scripts

Por favor, certifique-se de que os scripts enviados tenham a extensão correta para a linguagem na qual estão escritos.

Todos os scripts no repo são publicados sob a licença Apache v 2.0.

Você pode obter uma cópia da licença em https://www.apache.org/licenses/LICENSE-2.0

Submetendo seus scripts para este repo você os está liberando sob a licença Apache v 2.0. No entanto, você pode opcionalmente também liberá-los sob licenças mais branda, através de comentários nos próprios scripts.
