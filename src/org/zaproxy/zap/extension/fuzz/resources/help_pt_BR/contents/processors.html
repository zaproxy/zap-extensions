Diálogo de processadores de carga

Isso permite que você selecione os processadores de carga, para usar com geradores de carga específicos.

Processadores de carga do ZAP incluem:
  • Decodificação Base 64
  • Codificação Base 64
  • Expandir (para um comprimento mínimo especificado)
  • JavaScript Escape
  • JavaScript Unescape
  • Hash MD5
  • String de sufixo
  • String de prefixo
  • Hash SHA1
  • Hash SHA-256
  • Trim
  • URL Decode
  • URL Encode
Você pode também personalizar scripts de carga - eles podem realizar qualquer manipulação de carga que você precise.

Complementos também podem definir processadores adicionais de carga.
