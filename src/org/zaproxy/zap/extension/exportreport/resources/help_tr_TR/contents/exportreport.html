Komut Satırı

Eklenti komut satırından da çalıştırılabilir ve bir rapor oluşturmak için aşağıdaki argümanların yazılması gerekir.

OWASP ZAP Komut Satırı Seçenekleri

  • -session: ZAP'i başlattıktan sonra belirtilen oturumu açar
  • -cmd: ZAP'i 'satır içinde' çalıştırır, örn. kullanıcı arayüzünü ve ya bir hayalet modunu başlatmadan
Bakınız Wiki özgün olarak desteklenen komut satırı seçenekleri için daha fazla detay için.

Dışa Aktarma Rapor Uzantısı Komut Satırı Seçenekleri

  • -raporu_dışa aktar:

    • Açıklama:

      • Belirtilen oturumu yükledikten sonra belirtilen dosyaları kaydeder.

    • Şu anda Mevcut

      • ASCII 1.0 Strict Compliant XHTML Files (.xhtml )
      • XML Dosyaları (.xml )
      • JSON Dosyaları (.json )

    • Gelecek Aşama

      • Bootstrap HTML Dosyaları(.bootstrap.html )
      • PDF Belgeleri (.pdf )
      • Google Belgeleri (.doc )

  • -source_info:

    • Açıklama:

      • Raporun aşağıdaki ayrıntılarını belirtir:
        1. Başlık
        2. Tarafından
        3. İçin
        4. Tarama Tarihi
        5. Rapor Tarihi
        6. Tarama Doğrula
        7. Rapor Doğrula
        8. Açıklama

    • Örnek:

      • -source_info "Vulnerability Report of MyApp.com;JordanGS;Lost Souls;August 15, 2016;August 18, 2016;ZAP_D-2016-08-15;ZAP_D-2016-08-15;Lorem ipsum dolor sit amet, pri corpora ancillae adolescens in. Ea usu atomorum tincidunt, ne munere regione has. Ne sea summo tation, et sed nibh nostrum singulis. Nec causae viderer discere eu."

    • Doğrulama:

      • İçerik kontrol edilmedi, dilerseniz boş alanlara girebilirsiniz, tek koşul 8 öğenin hepsinin listede olmasıdır.

  • -alert_severity:

    • Açıklama:

      • Hangi uyarı şiddetlerinin rapora dahil edileceğini belirtir:
        1. Yüksek
        2. Orta
        3. Düşük
        4. Bilgilendirme

    • Giriş formatı:

      • Sadece ';' sınırlayıcısı olan bir dize listesi kabul eder

    • Giriş sınırlaması:

      • Listedeki her öğe için sadece 't' ve 'f' kabul eder

    • Örnek:

      • -alert_severity "t;t;f;t"

    • Açıklama:

      • Yukarıdaki örnekte, sadece Yüksek, Orta ve Bilgilendirici Uyarılar oluşturulan rapora dahil edilecektir.

    • Doğrulama:

      • Doğrulama: İçerik, 't' veya 'f' olarak geçerli ve bu 4 öğenin hepsi listede bulunur.

  • -alert_details:

    • Açıklama:

      • Hangi uyarı bilgilerinin rapora dahil edileceğini belirtir:
        1. CWE Kimliği
        2. WASC Kimliği
        3. Açıklama
        4. Diğer Bilgiler
        5. Çözüm
        6. Referans
        7. İstek Başlığı
        8. Cevap Başlığı
        9. İstek Gövdesi
        10. Cevap Gövdesi

    • Giriş formatı:

      • Yalnızca ';' sınırlayıcısı içeren bir dize listesini kabul eder

    • Giriş sınırlaması:

      • Listedeki her öğe için yalnızca 't' ve 'f' değerlerini kabul eder

    • Örnek:

      • -alert_details "t;t;t;t;t;t;f;f;f;f"

    • Açıklama:

      • Yukarıdaki örnekte, sadece CWE ID, WASC ID, Açıklama, Diğer Bilgi, Çözüm ve Referans Uyarı Bilgileri oluşturulan rapora dahil edilecektir.

    • Doğrulama:

      • İçerik, 't' veya 'f' olarak onaylandı ve 10 öğenin hepsi listede.

  • Hepsini bir araya koy! (Windows)

    1. Bir komut istemi açın
    2. Zap.bat dosyasının olduğu konuma gidin
    3. aşağıdakileri çalıştırın:
      • zap.bat -export_report "/path/to/save/report.xhtml" -source_info "Vulnerability Report of MyApp.com;JordanGS;Lost Souls;August 15, 2016;August 18, 2016;ZAP_D-2016-08-15;ZAP_D-2016-08-15;Lorem ipsum dolor sit amet, pri corpora ancillae adolescens in. Ea usu atomorum tincidunt, ne munere regione has. Ne sea summo tation, et sed nibh nostrum singulis. Nec causae viderer discere eu." -alert_severity "t;t;f;t" -alert_details "t;t;t;t;t;t;f;f;f;f" -session "/path/to/existing/session" -cmd

API Çağrıları

Eklenti API çağrıları ile erişilebilir ve bir rapor oluşturmak için aşağıdaki argümanların yazılması gerekir.

Kısıtlamalar yukarıdaki Komut Satırı için olanlar ile aynıdır.

Kullanım örneği:


Map<String, String> map = new HashMap<String, String>();
String API_KEY = "ZAPROXY-PLUGIN"
String absolutePath = "/path/to/save/report.xhtml";
String extension = "xhtml";
String sourceDetails = "Vulnerability Report of MyApp.com;JordanGS;Lost Souls;" +
    "Ağustos 15, 2016;August 18, 2016;ZAP_D-2016-08-15;ZAP_D-2016-08-15;" + 
    "Lorem ipsum dolor sit amet, pri corpora ancillae adolescens in. Ea " + 
    "usu atomorum tincidunt, ne munere regione has. Ne sea summo tation, " +
    "et sed nibh nostrum singulis. Nec causae viderer discere eu.";
String alertSeverity = "t;t;f;t";
String alertDetails = "t;t;t;t;t;t;f;f;f;f";

map.put("apikey", API_KEY);

// Raporda hedef kaydetmek için mutlak yolu temsil eden dize.
map.put("absolutePath", absolutePath)

// Raporun uzantısını temsil eden dize
map.put("fileExtension", extension);

// Her girdi için ayrılmış dize
map.put("sourceDetails", sourceDetails);

// Her girdi için ayrılmış dize
map.put("alertSeverity", alertSeverity);

//Her girdi için ayrılmış dize
map.put("alertDetails", alertDetails);
ApiResponseElement val = (ApiResponseElement) clientApi.callApi("exportreport", "action", "generate", map);
