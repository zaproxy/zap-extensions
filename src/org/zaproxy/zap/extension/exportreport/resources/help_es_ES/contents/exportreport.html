Informe de exportacion

Modulo de exportacion de informe permite a los usuarios personalizar el contenido y exportar en un formato deseado.
  • Creado por jordanGS.
  • Minimo compatible version: Semanal version ZAP_D-2016-09-05
  • Proyecto en GitHub. Tambien puede encontrar un zip de la version compatible minimo en la carpeta de descarge de descarga de zap.
  • Apoyo y en el Oficial OWASP Zed ataque Proxy Jenkins Plugin.

ZAP UI

Impormasyon sa Pinagmulan

Ang gumagamit ay pumasok at awtomatikong makuha ang data na may kaugnayan sa ulat. Mag-hover sa bawat field sa extension para sa tip ng tool.
  • Pamagat - Mga Default sa Pangalan ng Session
  • Por
  • Para
  • Petsa ng Pag-scan - Petsa ng ipinasok ng user ng AScan, mga default sa kasalukuyang petsa ng oras

  • I-scan ang Bersyon - Placeholder para sa hinaharap
  • Ulat ng Bersyon - Mga Default sa kasalukuyang bersyon ng ZAP tool

UI

Dialog ng Tagapili ng File

Ang diyalogo nagpapakita ng mga folder at tinanggap na mga uri ng file. Ang listahan ng drop down na 'Mga File ng Uri' ay i-filter upang ipakita lamang ang mga folder at mga file ng tinukoy na extension.

Magagamit na Kasalukuyang

  • ASCII 1.0 Mahigpit na Sumusunod XHTML Files (.xhtml )
  • XML Files (.xml )
  • JSON Files (.json )

Hinaharap Milestone

  • PDF Dokumento (.pdf )
  • Google Dokumento (.doc )

Command Line

Ang extension ay maaaring tumakbo mula sa command line pati na rin at nangangailangan ng mga sumusunod na mga argumento na ipasa sa upang bumuo ng isang ulat.

OWASP ZAP Command Line Option

  • -session : Binubuksan ang ibinigay na sesyon pagkatapos simulan ang ZAP
  • -cmd : Nagpapatakbo ng inline na ZAP, ibig sabihin, nang hindi nagsisimula ng UI o isang demonyo
Tingnan ang Wiki para sa higit pang mga detalye sa natively suportadong mga pagpipilian sa command line.

I-export ang Ulat sa Mga Pagpipilian sa Linya ng Command ng Extension

  • -export_report:

    • Descripcion:

      • Makakatipid sa tinukoy na file pagkatapos na mai-load ang ibinigay na sesyon.

    • Magagamit na kasalukuyang

      • ASCII 1.0 Mahigpit na Sumusunod XHTML Files (.xhtml )
      • XML Files (.xml )
      • JSON Files (.json )

    • Hinaharap Milestone

      • Bootstrap HTML Files(.bootstrap.html )
      • PDF Documento (.pdf )
      • Google Documento (.doc )

  • -source_info:

    • Descripcion:

      • Tinutukoy ang mga sumusunod na detalye ng ulat:
        1. Titulo
        2. Por
        3. Para
        4. Scan ng petsa
        5. Petsa ng ulat
        6. I-scan ang Ver
        7. Iulat ang Ver
        8. Descripcion

    • Halimbawa:

      • -source_info "Ulat ng kahinaan ng MyApp.com; JordanGS; Nawala ang mga Kaluluwa; Agosto 15, 2016; Agosto 18, 2016; ZAP_D-2016-08-15; ZAP_D-2016-08-15; Lorem ipsum dolor sit amet, pri corpora ancillae adolescens sa. Ang mga ito ay may isang atomicum, may isang rehiyon na may. Ang dagat ng dagat, at ang mga numeral na lugar. Nec causae viderer discere eu."

    • Validacion:

      • Ang nilalaman ay walang check, maaaring pumasok sa walang laman na mga patlang kung gusto mo, tanging kondisyon ay lahat ng 8 item ay nasa listahan.

  • -alert_severity:

    • Descripcion:

      • Tinutukoy kung aling mga severities alerto ang isasama sa ulat:
        1. Alto
        2. Medio
        3. Bajo
        4. Informacion

    • Format ng input:

      • Tinatanggap lamang ang isang listahan ng string na may ';' delimiter

    • Paghihigpit sa pag-input:

      • Tinutukoy kung aling mga severities alerto ang isasama sa ulat

    • Halimbawa:

      • -alert_severity "t; t; f; t"

    • Paliwanag:

      • Sa halimbawa sa itaas, ang Mga Alerto sa Mataas, Katamtaman at Kaalaman lamang ang isasama sa nabuong ulat.

    • Pagpapatunay:

      • Pagpapatunay: Ang nilalaman ay napatunayan na alinman sa 't' o 'f' at lahat ng 4 na item ay nasa listahan.

  • -alert_details:

    • Descriptcion:

      • Tinutukoy kung aling mga detalye ng alerto ang isasama sa ulat:
        1. CWE ID
        2. WASC ID
        3. Descripcion
        4. Iba pang impormasyon
        5. Solusyon
        6. Referencia
        7. Humiling ng Header
        8. Response Header
        9. Humiling ng Katawan
        10. Tugon ng katawan

    • Format ng input:

      • Tinatanggap lamang ang isang listahan ng string na may ';' delimiter

    • Paghihigpit sa pag-input:

      • Tanging tumatanggap ng 't' at 'f' para sa bawat item sa listahan

    • Halimbawa:

      • -alert_details "t; t; t; t; f; f; f; f"

    • Paliwanag:

      • Sa halimbawa sa itaas, tanging ang CWE ID, WASC ID, Paglalarawan, Iba pang Impormasyon, Solusyon at Mga Detalye ng Alerto ng Reference ay isasama sa nabuong ulat.

    • Pagpapatunay:

      • Ang nilalaman ay napatunayan na alinman sa 't' o 'f' at lahat ng 10 item ay nasa listahan.

  • Paglalagay ng lahat ng sama-sama! (Windows)

    1. Buksan ang command prompt
    2. Mag-navigate sa lokasyon kung saan matatagpuan ang zap.bat file
    3. execute ang mga sumusunod:
      • zap.bat -export_report "/path/to/save/report.xhtml" -source_info "Report kahinaan ng MyApp.com; JordanGS; Nawala ang mga Kaluluwa; Agosto 15, 2016 Agosto 18, 2016; ZAP_D-2016-08-15; ZAP_D-2016-08-15; Lorem ipsum dolor sit amet, pri corpora ancillae adolescens in. Ang mga ito ay may isang atomicum, may isang rehiyon na may. Ang dagat ng dagat, at ang mga numeral na lugar. Nec causae viderer discere eu. " -alert_severity " t; t; f; t " -alert_details " t; t; t; t;; f " -session " / path / to / existing / session " -cmd
