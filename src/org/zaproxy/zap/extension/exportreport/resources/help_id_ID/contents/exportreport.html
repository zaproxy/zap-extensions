Laporan Ekspor

Laporan Ekspor modul yang memungkinkan pengguna untuk menyesuaikan konten dan ekspor dalam format yang diinginkan.
  • Dibuat oleh JordanGS.
  • Minimum Versi Yang Didukung: mingguan rilis ZAP_D-2016-09-05
  • Proyek di GitHub. Anda juga dapat menemukan sebuah .zip dari rilis minimal yang didukung di folder download-zap.
  • Didukung dan dimasukkan kedalam resmi OWASP Zed Attack Proxy Jenkins Plugin.

Daftar Isi

  • ZAP UI
  • Baris Perintah
  • API panggilan

ZAP UI

Sumber Info

Pengguna masuk dan secara otomatis mengambil data yang relevan untuk laporan. Arahkan kursor ke masing-masing bidang dalam ekstensi untuk tool tip.
  • Judul - Default untuk Sesi Nama
  • Oleh
  • Untuk
  • Upload Scan - Pengguna memasukkan tanggal AScan, default untuk saat ini tanggal-waktu
  • Tanggal Laporan - Default untuk saat ini tanggal-waktu
  • Versi Scan - Placeholder untuk masa depan
  • Laporan Versi - Default untuk versi saat ini dari ZAP alat

UI

Pemilih Dialog File

Dialog hanya menampilkan folder dan file yang diterima. Yang 'Tipe file daftar drop-down akan filter untuk hanya menampilkan folder dan file-file dengan ekstensi tertentu.

Tersedia saat ini

  • ASCII 1.0 Ketat sesuai File XHTML (.xhtml )
  • File XML (.xml )
  • File JSON (.json )

Tonggak Masa depan

  • File Bootstrap HTML (.bootstrap.html )
  • Dokumen PDF (.pdf )
  • Dokumen Google (.doc )

UI

  • Tampilkan semua file yang diterima

  • Daftar format file yang diterima

  • Penyaring oleh format file tertentu

Garis komando

Ekstensi dapat dijalankan dari baris perintah serta dan membutuhkan argumen berikut yang harus dilalui untuk menghasilkan laporan.

Opsi Baris Perintah OWASP ZAP

  • -session: Membuka sesi yang diberikan setelah memulai ZAP
  • -cmd : Menjalankan 'inline ZAP', yaitu tanpa memulai UI atau daemon
Lihat Wiki untuk lebih detail tentang native didukung opsi baris perintah.

Ekspor Laporan Ekstensi-Opsi Baris Perintah

  • -export_report:

    • Deskripsi:

      • Menyimpan ke file yang ditentukan setelah loading yang diberikan sesi.

    • Tersedia saat ini

      • ASCII 1.0 Compliant Ketat XHTML File (.xhtml )
      • File XML (.xml )
      • JSON File (.json )

    • Tonggak masa depan

      • File HTML Bootstrap(.bootstrap.html )
      • Dokumen PDF (.pdf )
      • Google Dokumen (.doc )

  • -source_info:

    • Deskripsi:

      • Menentukan berikut rincian laporan:
        1. Judul
        2. Oleh
        3. Untuk
        4. Tanggal Pindai
        5. Tanggal laporan
        6. Pindai Ver
        7. Laporan Ver
        8. Deskripsi

    • Contoh:

      • -source_info "kerentanan LAPORAN MyApp.com; JordanGS: Lost Souls: 15 Agustus 2016; 18 Agustus 2016; ZAP_D-2016/08/15, 2016/08/15 ZAP_D-: lorem ipsum wortel, badan-badan swasta budak di masa muda. Ea usu atomorum tincidunt, ne munere regione memiliki. Tidak merusak bagian atas latihan, tapi pelacakan kami, setiap. Tidak ada alasan harus belajar komputer."

    • Validasi:

      • Konten tidak dicentang, dapat memasukkan bidang kosong jika anda ingin, satu-satunya syarat adalah bahwa semua 8 item dalam daftar.

  • -waspada_tingkat keparahan:

    • Deskripsi:

      • Menentukan mana peringatan severities akan dimasukkan dalam laporan:
        1. Tinggi
        2. Sedang
        3. Rendah
        4. Informasi

    • Masukkan format:

      • Hanya menerima daftar string dengan ';' pembatas

    • Masukan pembatasan:

      • Hanya menerima 't' dan 'f' untuk setiap item dalam daftar

    • Contoh:

      • -waspada_tingkat keparahan "t;t;f;t"

    • Penjelasan:

      • Dalam contoh di atas, hanya lansiran tinggi, menengah dan Informasi yang akan disertakan dalam laporan yang dihasilkan.

    • Validasi:

      • Validasi: konten divalidasi menjadi 't' atau 'f' dan semua 4 barang ada dalam daftar.

  • -waspada_detail:

    • Deskripsi:

      • Menentukan rincian peringatan mana yang akan disertakan dalam laporan:
        1. ID CWE
        2. ID WASC
        3. Deskripsi
        4. Info Lainnya
        5. Solusi
        6. Referensi
        7. Permintaan Atasan
        8. Respon Atasan
        9. Permintaan Tubuh
        10. Respon Tubuh

    • Masukkan format:

      • Hanya menerima daftar string dengan ';' pembatas

    • Masukkan larangan:

      • Hanya menerima 't' dan 'f' untuk setiap barang dalam daftar

    • Contoh:

      • -rincian_waspada "t;t;t;t;t;t;f;f;f;f"

    • Penjelasan:

      • Dalam contoh di atas, hanya CWE ID, WASC ID, Deskripsi, Info Lainnya, Solusi dan Referensi Rincian Waspada akan dimasukkan dalam laporan yang dihasilkan.

    • Validasi:

      • Konten divalidasi menjadi 't' atau 'f' dan semua 10 item ada dalam daftar.

  • Menempatkan semuanya bersama! (Windows)

    1. Buka perintah cepat
    2. Menavigasi ke lokasi di mana zap.bat yang terletak
    3. lakukan hal berikut:
      • zap.bat -export_report "/path/to/save/report.xhtml" -source_info "Vulnerability Report of MyApp.com;JordanGS;Lost Souls;August 15, 2016;August 18, 2016;ZAP_D-2016-08-15;ZAP_D-2016-08-15;Lorem ipsum dolor sit amet, pri corpora ancillae adolescens in. Bagi atom, penggunaan keyboard, sehingga fungsi dari wilayah dua. Tidak merusak bagian atas latihan, tapi pelacakan kami, setiap. Tidak ada alasan harus belajar komputer." -alert_severity "t;t;f;t" -alert_details "t;t;t;t;t;t;f;f;f;f" -session "/path/ke/ada/sesi" -cmd

Panggilan API

Ekstensi dapat diakses dengan panggilan API dan memerlukan argumen berikut yang harus dilalui untuk menghasilkan laporan.

Pembatasan yang sama untuk Baris Perintah di atas.

Contoh Pemakaian:


Peta<String, String> map = new HashMap<String, String>();
String API_KEY = "ZAPROXY-PLUGIN"
String absolutePath = "/path/ke/menyimpan/laporan.xhtml";
String ekstensi = "xhtml";
String sourceDetails = "Kerentanan Laporan MyApp.com;JordanGS;jiwa-Jiwa yang Hilang;" +
"15 agustus 2016;18 agustus 2016;ZAP_D-2016-08-15;ZAP_D-2016-08-15;" + 
"Desain Website, pri tubuh tapi aku pemuda di. Ini " + 
"penggunaan atom keyboard, bukan peran negara. Ne sea summo tation, " +
    "et sed nibh nostrum singulis. Nec causae viderer discere uni eropa.";
String alertSeverity = "t;t;f;t";
String alertDetails = "t;t;t;t;t;t;f;f;f;f";

peta.put("apikey", API_KEY);

// String yang mewakili path absolut untuk menyimpan tujuan untuk laporan
peta.put("absolutePath", absolutePath)

// String yang mewakili perpanjangan laporan
peta.put("fileExtension", perpanjangan);

// String yang dipisahkan oleh ; untuk setiap entri
peta.put("sourceDetails", sourceDetails);

// String yang dipisahkan oleh ; untuk setiap entri
peta.put("alertSeverity", alertSeverity);
ApiResponseElement val = (ApiResponseElement) clientApi.callApi("exportreport", "aksi", "menghasilkan", peta);
