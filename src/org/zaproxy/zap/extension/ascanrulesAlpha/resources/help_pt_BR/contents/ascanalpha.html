Regras de Varredura Ativ - alfa

Este complemento traz as seguintes regras de varredura ativa em qualidade alfa:

Buffer Overflow

procura por indícios de estouro de pilha (buffer overflow) no código. Isso é feito com o lançamento de grandes seqüências de caracteres de texto de entrada e busca por código de crash e encerramento anormal da sessão.

Detector de cookie inútil

Testa cookies para detectar se alguns não têm nenhum efeito no tamanho da resposta quando omitidos, especialmente os que contém o nome "sessão" ou "userid"

Exemplo de arquivo de varredura ativo

Esta regra implementa um exemplo de varredura ativa que carrega strings de um arquivo que o usuário pode editar.
Para mais detalhes veja: http://zaproxy.blogspot.co.uk/2014/04/hacking-zap-4-active-scan-rules.html

Exemplo de Varredura ativa simples

Isso implementa um exemplo de regra de varredura muito simples.
Para mais detalhes veja: http://zaproxy.blogspot.co.uk/2014/04/hacking-zap-4-active-scan-rules.html

Injeção de linguagem de expressão

O software constrói todo ou parte de uma instrução de linguagem de expressão (EL) em uma página de servidor de Java (JSP); para isso usa uma entrada vinda de um componente upstream, mas não neutraliza elementos especiais que poderiam modificar a instrução EL antes que seja executada. Em certas versões do Spring 3.0.5 e anteriormente, houve uma vulnerabilidade (CVE-2011-2730) na qual tags de EL podiam ser avaliadas duas vezes, que expunha qualquer aplicação a uma EL injection. No entanto, mesmo para versões posteriores, esta vulnerabilidade é ainda possível, dependendo da configuração. +

Format String Error

Looks for indicators of format string handling errors in compiled code. It does this by putting out strings of input text based upon characters compiled C code anticipates to produce formatted output and look for code crash and abnormal session closures.

Source Code Disclosure - File Inclusion

Uses local file inclusion techniques to scan for files containing source code on the web server.

Source Code Disclosure - Git

Uses Git source code repository metadata to scan for files containing source code on the web server.

HTTPS As HTTP Scanner

This active scanner attempts to access content that was originally accessed via HTTPS (SSL/TLS) via HTTP.
