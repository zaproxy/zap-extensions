Mga pagpipilian sa AJAX Spider iskren

Ang iskren na ito ay nagpapahintulot sayo para maisaayos ang AJAX Spider ang mga pagpipilian. Ang AJAX Spider ay Ang add-on para sa crawler ang tawag dito ay crawljax. Ang add-on sets up ay ilokal proxy sa ZAP para makapagsalita sa crawljax. Ang AJAX Spider ay nag papahintulot sa iyo na isulat ang crawl web aplikasyon sa AJAX para masmapalalim pa kesa sa native spider. Gamitin ang AJAX Spider Kung meron kang mga web aplikasyon isulat sa AJAX. Maaari mo lamang gamitin Ang native spider para makumpleto ang application ng coverage web (e.g., to cover HTML comments).

Field Mga detalye Default
Browser Ang AJAX spider ay umaasa sa panglabas na browser para ma krawl ang targeted site. Pwede mong matukoy ang anumang gusto mung gamiten. para sa ibapang detalye sa mga supported browser pumunta sa "Selenium" add-on pangtulong na pahina. Firefox
Bilang ng mga windows sa browser upang buksan Maaari mong I-configure ang bilang ng mga window para magamit sa AJAX Spider. Pag masmaraming windows, masmabilis ang magiging proseso. 1
Pinakamataas na lalim ng crawl Ang pinakamataas na lalim ng mga crawl ay hindi maabot. Zero ibig sabihin walang hangan ang lalim nito. 10
Ang pinakamataas na estado ng crawl Ang pinakamataas na estado ng crawler ay crawl. Zero ibigsabihin walang kataposan ang estado ng crawl. 0(walangkatapusan)
Pinakamataas na durasyon Ang pinakamataas na oras ng crawler ay pinapahintulutan tumakbo. Zero ibigsabihin walanghangan ang pagtakbo ng oras. 60 minuto
Ang kaganapan ay naghihintay sa oras Matapos maghintay ng oras ang panig ng kliyente Ang kaganapan ay inalis na. 1000 ms
Ang oras ay naghihintay sa reload Matapos maghintay sa oras ang URL ay naloadan na. 1000 ms
I-klick ang mga elemento minsan Kapag pinagana, ang crawler susubukan nitong makipag-ugnayan sa bawat elemento (e.g., by clicking) isang beses lamang. Kung hindi ito naiset, ang crawler ay susubukan ito maikliek ng maraming beses. Unsetting itong opsiyon ay magiging masmahigpit ngunit magiging mas matagal. True
Gumamit ng mga random na halaga sa mga form field Kung pinagana, isinget ang halaga ng random sa mga form ng field. Kung hindi, ito ay gagamit ng walang laman na halaga. True
Iklik ang mga default elemento lamang When enabled, only elements "a", "button" at "input" pweding mag-klik sa crawl. Kung Sana, ang mga ito ay gigamit sa ilalim ng table para matukoy Kung anung mga elemento ang gagamitin para maiklik. Para sa mas malalim na pagsusuri, huwag paganahin ito at isaayos ang mga clickable na elemento sa table. True
Pumili ng mga elemento na magkiklik sa crawl (table) Ang listahan ng mga elemento ng crawl. Ang table lamang ang nag aaplay sa "click default elements only" ito ay Hindi pinagana. Gamitin "ang lahat ng hindi pinagana" para sa higit pang pag papalalim ng pagsusiri, bagaman maaari itong bumagal ng matagal. Lahat ng pinagana
