Pantalla de opciones del Spider AJAX

Esta pantalla permite configurar las opciones AJAX Spider . AJAX Spider es un complemento para un rastreador llamado Crawljax. El complemento configura un proxy local en ZAP para comunicarse con Crawljax. AJAX Spider permite rastrear aplicaciones web escritas en AJAX en más profundidad que el rastreador nativo. Utilice AJAX Spider si tiene aplicaciones web escritas en AJAX. Debe también utilizar el rastreador nativo para abarcar completamente una aplicación web (por ejemplo, abarcar comentarios HTML).

Explicación de la opción de configuración



Opciones de configuración
Campo Detalles Por defecto
Navegador AJAX Spider utiliza un navegador externo para rastrear el sitio destino. Se puede especificar cuál se desea utilizar. Para mas detalles sobre los navegadores compatibles consulte las páginas de ayuda del complemento "Selenium". Firefox
Numero de ventanas de navegador para abrir Se puede configurar el número de ventanas para ser utilizadas por AJAX Spider. Mientras más ventanas abiertas, más rápido será el proceso. 1
Profundidad máxima de rastreo La profundidad máxima que el rastreador puede alcanzar. Cero significa profundidad ilimitada. 10
Estados de rastreo máximos El número de estados máximos que el rastreador debe rastrear. Cero significa estados de rastreos ilimitados. 0 (ilimitado)
Duración máxima El duración máxima permitida para que el rastreador se ejecute. Cero significa duración ilimitada de ejecución. 60 minutos
Tiempo de espera del evento El tiempo de espera luego de que se ejecuta el evento del lado del cliente. 1000 ms
Tiempo de espera de recarga El tiempo de espera después de que se carga URL. 1000 ms
Hacer clic a los elementos una vez Cuando se habilita, el rastreador intenta interactuar con cada elemento (por ejemplo haciendo clic) solo una vez. Si no se configura, el rastreador intentará hacer clic varias veces. Desconfigurar esta opción es mas riguroso pero puede tomar considerablemente más tiempo. True
Utiliza valores aleatorios en campos de formularios Cuando se habilita, inserta valores aleatorios dentro de campos de formularios. De lo contrario, utiliza valores en blanco. True
Hacer clic solo en los elementos por defecto Cuando se habilita, se dará clic solo los elementos "a", "botón" y "salida" durante el rastreo. De lo contrario, utiliza la tabla de abajo para determinar cuáles elementos se cliquearán. Para más información sobre profundidad del análisis, deshabilite esto y configure los elementos cliqueables en la tabla. True
Seleccione los elementos para hacer clic durante el rastreo (tabla) La lista de elementos que se rastreará. Esta tabla solo aplica cuando "Hacer clic solo en los elementos por defecto" no está habilitado. Utilice "habilitar todos" para más información sobre análisis en profundidad, aunque puede tardar un poco más. All enabled
