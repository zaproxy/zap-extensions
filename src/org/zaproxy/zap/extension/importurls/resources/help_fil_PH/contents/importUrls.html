Pag-import ng mga URL

Ang add-on na ito ay magdadagdag ng pagpipilian upang ma-import ang dokumento ng mga URL gamit ang 'Kagamitan' na menu. Ang dokumento ay kailangang maging payak na teksto na may isang URL kada line.

Ang add-on na ito ay naglalantad ng ZAP API na endpoint /importurls/importurls (filePath*) upang mapadali ang programatik na paggamit ng pangsyonalidad.
