Tela de opções Selenium

Esta tela permite a você configurar os requisitos de alguns dos WebDrivers.

Explicação das opções de configuração

Opções de configuração
Campo Detalhes Padrão
WebDrivers
ChromeDriver Isso permite que você selecione o local do ChromeDriver. O caminho para o pacote WebDriver, se disponível.
geckodriver Isso permite que você selecione o local do geckodriver (driver de Firefox). O caminho para o pacote WebDriver, se disponível.
IEDriverServer Isso permite que você selecione o local do IEDriverServer. O caminho para o pacote WebDriver, se disponível.
Binários
Firefox Isso permite que você selecione a localização do binário do Firefox (por exemplo, para usar uma versão diferente do padrão do sistema). (Nenhum)
PhantomJS Isso permite que você selecione o local do binário do PhantomJS. (Nenhum)
Nota: Também é possível definir os locais acima (binários e WebDrivers) usando as propriedades de sistema do Java, caso em que as opções acima serão substituídas e os valores mostrados em vez disso.

More details about the bundled WebDrivers can be found on the overview page.
