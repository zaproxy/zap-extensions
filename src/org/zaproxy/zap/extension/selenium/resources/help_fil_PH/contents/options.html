Mga opsyon ng Siliniyum sa Iskrin

Pinahihintulutan ka ng iskrin nato na iayos ang lahat ng kinakailangan ng ilan sa mga WebDrivers.

Ang pagpapaliwanag sa kumpigurasyon ng opsyon

Mga opsyon ng kumpigurasyon
Field Mga Detalye Ang Default
Mga WebDriver
ChromeDriver Pinahihintulutan ka nito na piliin ang lokasyon ng ChromeDriver. Ang daan para sa bundled ng WebDriver, kung magagamit.
geckodriver Pinahihintulutan ka nito na piliin ang lokasyon ng geckodriver (Firefox driver). Ang daan para sa bundled ng WebDriver, kung magagamit.
IEDriverServer Pinahihintulutan ka nito na piliin ang lokasyon ng IEDriverServer. Ang daan para sa bundled ng WebDriver, kung magagamit.
Ang mga Binary
Firefox Pinahihintulutan ka nito na piliin ang lokasyon sa binary ng Firefox (para sa halimbawa, ang paggamit ng isang bersiyon maliban sa sistema ng default). (Wala)
PhantomJS Pinahihintulutan ka nito na piliin ang lokasyon sa binary ng PhantomJs. (Wala)
Paalala: Ito din ay posible na mailagay ang mga lokasyon sa itaas sa mga (binary at WebDriver) gamit ang mga pag aari ng sistema ng Java, sa alin mang kaso na mga opsyon na nasa itaas ay magiging walang halaga at sa halip na ipinapakita ang mga halaga nito.

Iba pang mga detalye para sa bundled ng mga WebDriverna maaaring makita sa pangkalahatang ideya ng pahina.

Maaaring tingnan sa

     Ang Siliniyum para sa pangkalahatang ideya ng add-on ng Siliniyum
     API para sa pangkalahatnag ideya ng siliniyum sa API
