Ma-kustomays na HTML na Report

Ang Ma-kustomays na HTML na Report na add-on ay magpapahintulot sa iyo na kontrolin ang impormasyon na mapupunta sa loob ng report sa pagscan ng ZAP. ito ay mayroong tatlong mga tab na magbibigay ng mga detalye patungkol sa report: Saklaw, Mga Alerto at Mga Detalye ng Alerto.

Saklaw

Maaring makatalaga ng pangalan ng report, deskripsyon ng report at pumili ng template sa labas ng HTML. Bukod pa rito, maaring makatukoy kung isasama ba lamang ang mga nilalaman sa loob ng saklaw.
Mga template ng Report:
  • Tradisyonal: mga listahan ng lahat ng mga alerto at pag-aayos sa kanila sa kada lebel ng panganib mula sa Pinakamataas hanggang sa Pinakamababa.
  • Pinaghihiwalay na Mga Sayt: pareho sa Tradisyonal, pero gumugrupo din kada pangalan ng domain.
  • Maiksi: ginugrupo ang mga alerto kada uri (upang maiwasan ang pag-uulit) at pag-aayos mula sa Pinakamataas hanggang sa Pinakamababa.

Mga Alerto

Maaring makapili kung anong mga uri ng alerto ang isasali sa loob ng report. Bukod pa dito, ang mga alerto ay maaring mapili o hindi mapili kada lebel ng panganib (Pang-impormasyon, Mababa, Katamtamanan, Mataas).

Mga Detalye ng Alerto

Magpapahintulot na piliin kung anong mga katangian ng alerto ay isasali sa loob ng Report.
Ito ay naka-tsek at default na naisasali:
  • Deskripsyon
  • Ibang Impormasyon
  • Solusyon
  • Reperensiya
  • ID ng CWE
  • ID ng WASC
Ito ay hindi naka-tsek at default na hindi naisasali:
  • Header ng Pagrekwest
  • Header ng Pagtugon
  • Katawan ng Parekwest
  • Katawan ng Pagtugon
