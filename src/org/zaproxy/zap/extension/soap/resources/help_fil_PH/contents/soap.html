SOAP na Scanner

Ang add-on na ito ay nag-iimport at nag-scan ng WSDL na mga file na naglalaman ng SOAP na mga endpoint.

Importing

Ang add-on ay awtomatikong magtutuklas ng kahit anong mga katuturan at spider sa kanila habang sila ay nasa saklaw.

dalawang menu na mga item ang nadagdag sa Tools na menu:
  • Mag-import sa WSDL file mula sa lokal file na sistema
  • Ang pag-import ng isang WSDL file mula sa isang URL
Ang mga operasyon na ito ay magagamit din sa pamamagitan ng API.

Pag-scan

Ang patakaran ng pag-scan na idinagdag sa add-on na ito ay:
  • Aksyon Spoofing: http://www.ws-attacks.org/index.php/SOAPAction_Spoofing
  • XML na injection : http://www.ws-attacks.org/index.php/XML_Injection
