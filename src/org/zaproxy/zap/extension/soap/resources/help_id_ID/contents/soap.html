Pemindai SOAP

Add-on ini mengimpor dan memindai berkas WSDL yang mengandung endpoints SOAP.

Mengimpor

Add-on akan secara otomatis mendeteksi setiap definisi SOAP dan spider mereka selama mereka termasuk kedalam cakupan.

2 menu item juga ditambahkan ke menu Tools:
  • Impor berkas WSDL dari sistem berkas lokal
  • Impor berkas WSDL dari URL
Operasi tersebut juga tersedia melalui API.

Pemindaian

Aturan pemindaian yang ditambah oleh add-on ini adalah:
  • Tindakan Spoofing : http://www.ws-attacks.org/index.php/SOAPAction_Spoofing
  • Injeksi XML : http://www.ws-attacks.org/index.php/XML_Injection
