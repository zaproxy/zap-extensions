Escáner SOAP

Este complemento importa y escanea archivos WSDL que contienen puntos de conexión SOAP.

Importar

Este complemento detectará automáticamente cualquier definición SOAP y las rastreará siempre que estén en el ámbito de la aplicación.

Se agregan también dos elementos de menú en el menú de Herramientas:
  • Importar un archivo WSDL desde el sistema de archivos locales
  • Importar un archivo WSDL desde una dirección URL
Estas operaciones también están disponibles a través de la API.

Escanear

Las reglas de escaneo agregadas por este complemento son:
  • Action Spoofing : http://www.ws-attacks.org/index.php/SOAPAction_Spoofing
  • XML Injection : http://www.ws-attacks.org/index.php/XML_Injection
