JxBrowser

JxBrowser adalah browser yang tersemat berdasarkan dari Chromium. ini tidak memiliki dependensi lainnya jadi anda yakin bahwa ini bekerja diluar kotak.
Anda dapat memulai karena banyak kasus seperti yang anda suka dan mereka semua akan secara otomatis dikonfigurasi untuk proxy via ZAP.
Hal ini juga sekarang menerima 'tidak valid' sertifikat secara default, yang meliputi ZAP sertifikat root.

Anda masih dapat menggunakan browser lain untuk proxy via ZAP tetapi anda akan perlu untuk secara manual mengkonfigurasi browser tersebut untuk proxy via ZAP.

Menggunakan JxBrowser untuk otomatisasi

Kurangnya dependensi eksternal membuat JxBrowser sangat berguna untuk percobaan otomatis. JxBrowser dapat dipilih, untuk digunakan dengan fungsi lain (misalnya AJAX Spider), melalui baris perintah atau ZAP API menggunakan ID jxBrowser.

Hal ini bukan tanpa kepala browser tetapi dapat dijalankan tanpa menggunakan teknologi seperti virtual frame buffer. ZAP docker gambar berisi script yang dimulai ZAP dengan xvfb.

Perizinan

JxBrowser http://www.teamdev.com/jxbrowser adalah sebuah perangkat lunak berpemilik. Penggunaan JxBrowser diatur oleh perjanjian lisensi produk JxBrowser http://www.teamdev.com/jxbrowser-licence-agreement. Jika anda ingin menggunakan JxBrowser dalam perkembangan anda, silahkan hubungi TimDev.

Sebagai ZAP pengguna anda akan dapat menggunakan ZAP JxBrowser add-on untuk tujuan apapun.
Namun jika anda mengubah ZAP kode sumber dan memanggil JxBrowser API dari salah satu dari produk anda sendiri kemudian anda akan perlu untuk mengatur cocok lisensi dengan TimDev.
Hal ini juga dicatat bahwa sementara JxBrowser tidak 'telepon rumah' internal Kromium fungsi dapat menghubungi layanan pihak ke-3 (mantra memeriksa, geolokasi, dll) dalam cara yang mirip dengan browser yang lain.

Banyak terima kasih untuk TimDev untuk bermurah hati memberikan kita tetap lisensi gratis untuk memungkinkan kita untuk mendistribusikan JxBrowser dengan ZAP.
