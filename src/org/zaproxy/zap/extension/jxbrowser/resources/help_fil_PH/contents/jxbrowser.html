JxBrowser

Ang JxBrowser ay isang naka-embed na browser na binase sa Chromium. Ito ay maryoon walang ibang external na mga dependency kaya ikaw ay makakasigurado na ito ay gagana palabas ng kahon.
Maari kang maglunsad ng kahit ano karaming mga pagkakataon na gusto mo at silang lahat ay awtomatikong makokompigyura sa proxy gamit ang ZAP.
Ito rin ay tumatanggap ng 'hindi balidong' mga sertipiko bilang default, na magpapasali ng ugat na sertipiko ng ZAP.

Maari ka pa ring gumamit ng ibang browser upang maka-proxy gamit ang ZAP pero kailangan mong manuwal na ikompigyura ang mga browser na iyon upang maka-proxy gamit ang ZAP.

Gamit ang JxBrowser para sa Awtomasyon

Ang pagkukulang ng external na mga dependency ay makakagawa ng JxBrowser na partikular na magagamit para sa pang-awtomate na pagtest. Ang JxBrowser ay maaring mapili, para sa kagamitan ng ibang mga pangsyonality (e.g. AJAX na Ispayder), gamit ang komand layn o ang ZAP na API gamit ang ID jxbrowser.

Ito ay hindi isang browser na walang pinuno pero ito ay maaring mapatakbo kahit wala itong pinuno gamit ang mga teknolohiya kagaya ng birtuwal na mga buffer ng frame. Ang mga imahe ZAP ng docker ay naglalaman ng script na nagpapasimula ng ZAP gamit ang zvfb.

Paglilisensiya

JxBrowser http://www.teamdev.com/jxbrowser ay isang proprayeti na software. Ang paggamit ng JxBrowser ay pinamamahalaan ng Kasunduan ng Lisensya sa Produkto ng JxBrowser http://www.teamdev.com/jxbrowser-licence-agreement. Kung gusto mong gumamit ng JxBrowser sa iyong pagdebelop, mangyari lamang na kontakin ang TeamDev.

Bilang isang tagagamit ng ZAP ikaw ay maaring makagamit ng ZAP JxBrowser na add on para sa kahit anong layunin.
Gayunpaman kung babaguhin mo ang source code ng ZAP at magtatawag ng JxBrowser na API mula sa isa sa iyong mga produkto ay kinakailangan mong isaayos ang angkop na lisensya sa TeamDev.
Mahalaga ding isaisip na kahit ang JxBrowser ay hindi nagpino-'phone home' ang internal na Chromium na pangsyonalidad maaring tumawag ng 3rd parti na mga serbisyo (pagsisiyasat ng ispeling, geolokasyon, etc) na parehong paraan sa ibang mga browser.

Maraming pasalamat sa TeamDev para sa bukas-palad na pagpapakaloob sa amin ng permanenteng libreng lisensiya upang magpahintulot sa atin na magredistribyut ng JxBrowser gamit ang ZAP.
