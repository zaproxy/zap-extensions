JxBrowser

JxBrowser es un navegador integrado basado en Chromium. No tiene otras dependencias externas así que puede asegurarse de que estará listo para usarse.
Se pueden iniciar tantas instancias como se quiera y todas ellas serán configuradas automáticamente al proxy mediante ZAP.
Ahora también acepta por defecto certificados "inválidos", de los cuáles incluye el certificado raíz de ZAP.

Todavía se puede usar cualquier otro navegador al proxy mediante ZAP pero se necesitará configurar manualmente esos navegadores al proxy mediante ZAP.

Utilizar JxBrowser para automatización

La falta de dependencias externas hace que JxBrowser particularmente útil para pruebas automatizadas. El JxBrowser puede ser seleccionado, para utilizar con otras funcionalidades (por ejemplo AjAX Spider), a través de la línea de comando o la API ZAP utilizando el ID jxbrowser.

No es un navegador sin cabeza pero puede ser ejecutado sin cabeza utilizando tecnologías como búferes de trama virtual. Las imágenes de docker ZAP contienen un script que inicia ZAP con xvfb.
