Exterminador SNI

O Terminator SNI permite que você use o ZAP como um proxy HTTPS transparente para os clientes que suportam SNI.

Com o ZAP já funciona muito bem como um proxy HTTP transparente, isso permite a você configurar uma LANs (ou VMs) de teste onde todo o tráfego HTTP e HTTPS é intermediado por proxy, independentemente das configurações de software.

Por exemplo, se você tiver uma máquina Linux que você usa para testes, pode fazer algo como o seguinte para encaminhar o tráfego de todos os HTTP e HTTPS para sua instância ZAP rodando em 192.168.0.14::
iptables -t nat -A OUTPUT -p tcp --dport 443 -j DNAT --to-destination 192.168.0.14:8443
iptables -t nat -A OUTPUT -p tcp --dport 80 -j DNAT --to-destination 192.168.0.14:8080
