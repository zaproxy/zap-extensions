Id de sessão na reescrita de URL

Este scanner verifica a existência de parâmetros do tipo token de sessão sendo reescritos na URL. Para ajudar a reduzir falsos positivos, o scanner verifica o comprimento do valor do token; se o valor desse parâmetro não for maior que 8 caracteres de comprimento, o parâmetro é ignorado (ou seja: survey? sId = 5 não seriam sinalizado como vulnerável).
