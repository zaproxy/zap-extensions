Estadísticas de Autenticación

Este complemento registra estadísticas relacionadas con autenticación en todos los contextos que estén al alcance.
Las estadísticas son guradadas en base a sitio y cobertura:
stats.auth.component.state.nothtmlRespuestas no HTML (que es poco probable que tengan indicadores de inicio y cierre de sesión)
stats.auth.component.state.notsuccessRespuestas HTML fallidas (que es poco probable que tengan indicadores de inicio y cierre de sesión)
stats.auth.component.state.noindicatorNo hay indicadores de inicio y cierre de sesión definidos
stats.auth.component.state.loggedinandoutHay indicadores tanto de inicio como de cierre de sesión
stats.auth.component.state.loggedinSolo se encuentra disponible el indicado de inicio de sesión
stats.auth.component.state.loggedoutSolo se encuentra disponible el indicado de cierre de sesión
stats.auth.component.state.unknownNinguno de los indicadores de inicio o cierre de inicio de sesión están presentes
El componente puede ser:
ascanEscaneo Activo
authAutenticación
fuzzFuzzer
manualManual de solicitudes
proxySolicitudes redirigidas mediante proxy
spiderRastreador web
idEl número entero que identifica algún otro componente que no esté listado más arriba

las estadísticas son accesibles mediante la API de ZAP.

El ícono para este complemento viene de los Fugue icons chart.png y locl-small.png
