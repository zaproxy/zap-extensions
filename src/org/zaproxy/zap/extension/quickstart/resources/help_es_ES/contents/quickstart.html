Inicio Rápido

La pestaña Inicio rápido le proporciona una manera fácil de probar rápidamente una aplicación web.

Ingrese la URL de su aplicación de destino y presione el botón 'Ataque'.

ZAP spider analizará el objetivo y luego lo atacará con el escáner activo.

Para una prueba más profunda, debe explorar su aplicación usando su navegador o Pruebas de regresión automática mientras se realiza el proxy a través de ZAP.

Iniciar Navegador

Siempre que se tenga instalado el complemento Selenium se podrá ver la opción de iniciar navegadores.
Seleccione el navegador deseado desde el menú desplegable y presione el botón "Iniciar Navegador".
Los navegadores iniciados de esta manera serán configurados al proxy mediante ZAP e ignorará los errores de certificado, por lo que no es necesario importar el certificado ZAP Root CA.
ZAP recordará el último navegador seleccionado cuando lo reinicies.

Tenga en cuenta que los navegadores iniciados se detendrán si se cambia la dirección o el puerto escuchado por ZAP.

Plug-n-Hack

Si se tiene el complemento Plug-n-Hack instalado verá un botón para configurar Firefox con Plug-n-Hack.
Esto configura automáticamente Firefox al proxy mediante ZAP e importará y confiará en el ZAP Root CA.
