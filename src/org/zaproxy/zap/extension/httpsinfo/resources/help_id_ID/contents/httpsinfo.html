HTTPS Info Add-on

Add-on HTTP Info diakses melalui menu konteks didalam situs pohon atau tabel riwayat. Menampilkan dialog dimana berbagai informasi ringkas ditampilkan mengenai target sertifikat server HTTP, dan rangkaian chiper SSL/TLS yang ditawarkan.

Add-on ini memanfaatkan OWASP proyek lain: Deep Violet untuk melakukan itu sertifikat dan rangkaian sandi pengumpulan informasi.

Umum

Bagian atas dialog dikhususkan untuk rincian umum sertifikat SSL/TLS yang dihadirkan oleh server. Seperti:
  • Subjek DN
  • Algoritma Penandatanganan
  • Sertifikat Sidik Jari
  • Emiten DN
  • Validitas tanggal
  • Status ditandatangani sendiri
  • dan lain-lain.

Cipher suite

Bagian bawah dialog ini dikhususkan untuk pencacahan/daftar spesifik Chiper Suites target server, dikelompokkan berdasarkan protokol jabat tangan (SSLv2, SSLv3, TLSv1, TLSv1.1, dll.).
