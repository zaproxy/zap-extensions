Code Dx

Ang Code Dx na add-on ay magpapahintulot sa iyo na gumawa ng magkabagay na mga report at i-upload ang mga ito sa Code Dx na server. Mayroong dalawang pagpipilian sa ilalim ng Report na menu.

Code Dx: Pag-upload ng Report

Ang pagpipiliang ito ay gagawa ng isang report para sa iyo at i-upload ito sa Code Dx na server. Ang pagpili nito ay magpapasabi sa iyo para sa URL ng Server at API na Key. Pagkatapos maparami ang mga field, ang pagpindot ng refresh na button upang makuha ang listahan ng mga proyekto na magagamit ng API na Key. Pagkatapos ay ipindot ang Upload na button upang ipadala ang mga resulta sa Code Dx na server.

Code Dx: Gumawa ng XML na Report

Ang pagpipiliang ito ay gagawa ng pamantayang XML na report na magpapasama ng HTTP na rekwest at pagtugon na datos.
