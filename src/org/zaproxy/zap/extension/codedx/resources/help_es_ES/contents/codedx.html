Code Dx

El complemento Code Dx le permite generar reportes compatibles y subirlos a un servidor Code Dx. Hay dos opciones bajo el menú de Reportes.

Code Dx: Subir Reporte

Esta opción generará un reporte y lo subirá a un servidor de Code Dx para usted. Seleccionarla le pedirá una URL de Servidor y una clave de API. Después de rellenar los campos, haga clic en el botón actualizar para obtener una lista de proyectos disponibles para la clave API. Luego haga clic en el botón subir para enviar resultados al servidor Code Dx.

Code Dx: Generar reporte XML

Esta opción generará un reporte XML estándar que incluye la solicitud HTTP y los datos de respuesta.
