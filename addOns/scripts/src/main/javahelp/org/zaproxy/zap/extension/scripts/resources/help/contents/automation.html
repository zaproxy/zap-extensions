Scripts Automation Framework Support

This add-on supports the Automation Framework.

Job: script

The script job allows you to execute various actions with scripts. Run standalone scripts is currently the only action, but more actions are planned (add/remove/enable/disable) 
  - type: script
    parameters:
      action:                    # String: The executed action - available actions: run
      type:                      # String: The type of the script - available script type for action run: standalone
      name:                      # String: The name of the script
