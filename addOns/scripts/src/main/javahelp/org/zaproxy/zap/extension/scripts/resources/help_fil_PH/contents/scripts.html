Ang mga Iskrip

Ang ZAP Iskrip add-on ay hinayaan ka upang patkabuhin ang mga iskrip na maaaring naka-embed sa loob ng ZAP at mai-access ang mga istraktura ng datos ng ZAP.
Ito ay mga suporta sa kahit anung ini-iskrip na lenggwahe na sumusuporta sa JSR 223 (http://www.jcp.org/en/jsr/detail?id=223) , kabilang ang:

  • ECMAScript / Javascript ( gamit ang Nashorn engine, kasama sa pamamagitan ng default)
  • Zest https://developer.mozilla.org/en-US/docs/zest (kasama sa pamamagitan ng default)
  • Groovy http://groovy-lang.org/
  • Python http://www.jython.org
  • Ruby - http://jruby.org/
  • at marami pang iba...
BABALA - ang mga iskrip ay tumatakbo kasama ng parehong mga permisyon bilang ZAP, kaya huwag magpatakbo ng kahit na anung mga iskrip na hindi mo pinagkakatiwalaan!

Mga tipo ng iskrip

Iba't ibang mga tipo ng mga iskrip na suportado:
  • Mag-isang manindigan - Ang mga iskrip ay nasa sarili at ito ay tumatakbo lamang kapag inumpisahan mo sila ng mano-mano
  • Aktibong mga Patakaran - itong pagpapatakbo ay isang parte ng Aktibong mga nagsusuri at maaari ring gumana ng isa-isa
  • Pasibong mga Patakaran - itong pagpapatakbo ay isang parte ng Pasibong Nagsusuri at maaari ring gumana ng isa-isa
  • Mga Patakaran ng Kinatawan - itong tumatakbong 'nasa linya', ay maaaring magbago ng bawat kahilingan at tumugon at maaari ring gumana ng isa-isa. Sila ay maaari ring mag-trigger ng mga punto ng break
  • Nagpadala ng HTTP - ang mga iskrips na tumatakbo laban sa bawat kailingan/pagtugon na naipadala/natanggap na sa pamamagitan ng ZAP. Ito ay mga kabilang ang mga mensahe na kinatawan, mga pinadalang mensahe sa panahon ng aktibong pagsusuri, fuzzer, ...
  • Na-target na mga patakaran - ang mga iskrip na nagsasagawa kasama ang isang target URL at ito ay papatakbuhin lamang kapag inumpisahan silang imano-mano
  • Awtentikasyon - ang mga skrip na nagsasagawa habang angawtentikasyon ay ginagampanan para sa konteksto. Upang magamit, kailangan nilang mapili habang kinukumpigura ang Sript-Based awtentikasyong pamamaraan para sa isang konteksto.
  • Iskrip Input ng mga Vector - ang mga iskrip na para sa pagtukoy ng eksakton na kung anung ZAP ang kinakailangang atakihin
  • Ekstender - ang mga iskrip na maaaring idagdag sa bagong pag-andar, kasama ng grapikong mga elemento at ng bagong API ng mga puntong pagtatapos
Ang lahat ng mga iskrip ay tumatakbo ng awtomatikong unang ' hindi pinapagana' - kinakailangan na paganahin sila sa pamamagitan ng Ang mga Iskrip na 'puno' ng tab bago sila tumakbo.
Kung ang isang mali ay nangyayari habang sila ay tumatakbo hindi na sila gagana pagkatapos.
Kapag pinili mo ang iskrip ang huling pagkakamali ay magpapakita pagkatapos sa Tab ng iskrip console.
Ang mga iskrip na na-target ay maaaring ma-invoke sa pamamagitan ng tamang pagpindot sa isang nakatala sa, ga site at mga tab ng kasaysayan at piliin ang 'mag-invoke kasama ang iskrip...' pagpipilian ng gamit.

Lahat ng mga lenggwahe na in-iskrip at maaaring magamit sa lahat ng mga tipo ng iskrip, pero ang mga lenggwahe lamang na nai-download na mula sa Marketplace ng ZAP ay karaniwang may mga template. Gayunpaman maaari mo ring mai-angkop ang isang template para sa ibang pang lenggwahe.
Kung ang paborito mong lenggwahe ay hindi makuha sa Marketplace ay mangyaring ma-raise ng isang isyu sa pamamagitan ng " Online/Report ng isang isyu" pagpipilian ng gamit.
Kasabay nito ay maaari mo na lamang ilagay sa kaugnay na mga garapon sa 'lib' na panutuan (hindi ang 'plugin' na panutuan) at muling i-restart ang ZAP.

Mga Pabagu-bagong Global

Mga pabagu-bago ay maaaring ibahagi sa pagitan ng lahat ng mga iskrip sa pamamagitan ng klase ng org.proxy.zap.extension.script.ScriptVars.
Halimbawa sa Javascript ay maaari mong magamit itong klase tulad ng sumusunod:

org.zaproxyzap.zap.extension.script.ScriptVars.setGlobalVar("var.name","value")
org.zaproxy.zap.extension.script.ScriptVars.getGlobalVar("var.name")

Mga Pabagu-bagong Iskrip

Mga pagu-bago ay maaaring ibahagi sa pagitan ng hiwalay na mga pagsang-ayon sa parehong iskrip sa pamamagitan ng parehong org.zaproxy.zap.extension.script.ScriptVars na klase.
Halimbawa sa Javascript ay maaari mong magamit itong klase tuland ng mga sumusunod:

org.zaproxy.zap.extension.script.ScripVars.setScriptVar(this.context, "var.name","value")
org.zaproxy.zap.extension.script.ScriptVars.getScriptVar(this.context, "var.name")
Tandaan na ang mga pamamaraan na ito ay magagamit lamang mula sa pag-iskrp ng mga lenggwahe na nagbibigay ng access sa ScriptContext (tulad ng Javascript).
