Вкладка дерева скриптов

На вкладке дерева сценариев показаны все загруженные в настоящее время сценарии, упорядоченные по типу.
Он также показывает вам, какие шаблоны у вас есть — их нельзя запустить напрямую, вы используете их для создания новых скриптов.
Он также позволяет добавлять новые сценарии, загружать, сохранять и удалять их.
Вкладка содержит панель инструментов, которая позволяет:

  • Загрузить скрипт из файлового хранилища
  • Сохранить скрипт в файловом хранилище
  • Создать новый скрипт
Сценарии, которые запускаются в компонентах ZAP, таких как Active Scanner, можно включать и отключать с помощью пункта меню, вызываемого правой кнопкой мыши.
Все сценарии можно удалить из пользовательского интерфейса, щелкнув правой кнопкой мыши пункт меню «Удалить сценарий».
