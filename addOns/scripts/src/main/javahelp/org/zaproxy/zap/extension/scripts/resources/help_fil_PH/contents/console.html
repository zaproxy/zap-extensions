Ang Tab ng Script console

Ang mga script console ay nagpapahintulot sa iyo na magsulat ng mga script na tumatakbo sa loob ng ZAP.
Ito ay binubuo ng:

  • Isang toolbar
  • Isang lugar na kung saan ay maisusulat mo ang iyong mga script
  • Isang lugat para sa debug at maling mga mensahe
Maari kang magpatakbo ng 'Stand alone' na mga script mula sa loob ng tab na ito gamit ang 'Run' na pindutan sa tab toolbar.
Lahat ng ibang mga uri ng mga script ay tatakbo kapag pinagana o kung pinagtitiwalaan mahihingi.

Para lumikha ng isang bagong script sa pagkarga o pagpalit ng mga script tingnan ang Ang mga script 'tree' na tab.

Kanang-klik sa script na lugar para i-displey at mag-edit sa mga opsyon.

Mga template ay maaaring makikita sa console pero hindi maaaring i-edit.

Awtong-kumpleto

Ang console ay sumusuporta ng isang limitadong anyo sa awto-kumpleto, kinokontrol gamit ang isang pindutan sa toolbar.
Kapag pinagana ang isang popup ay makikita kapag ikaw ay magtype ng pangalan sa isa sa ipinahayag na mga parameter sumunod gamit ang isang dot- ito ay nagpapakita sa iyo ng lahat ng mga paraan na magagamit para sa iyo para tawagan. Pagpili ng isa sa kanila ay ipinasok ang mga pamamaraan at mga parameter sa iyong script. Kung i-type mo ang isa pang tuldok kaagad matapos ang paraan sa pagtawag pagkatapos mo ay ipapakita ang isa pang udyok para sa mga pamamaraan magagamit sa pagbalik na uli, kung mayroon man. Ikaw ay maaaring magpanatili ng malawak na mga uri ay bumalik hanggang sa simulan mo ang pag-type sa ibang bagay.
