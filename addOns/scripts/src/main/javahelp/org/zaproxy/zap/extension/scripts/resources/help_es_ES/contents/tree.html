Pestaña de árbol de scripts

La pestaña del árbol de scripts muestra todos los scripts que están actualmente cargados organizados por tipo.
It also shows you which templates you have available - these cannot be run directly, you use them to create new scripts.
También permite agregar nuevos script, cargar, guardar y eliminarlos.
The tab includes a toolbar which allows you to:

  • Cargar un script desde un almacén de archivos
  • Guardar un script en un almacén de archivos
  • Crear un nuevo script
Los scripts que se ejecutan como componentes ZAP, tal como el Escaneo Activo, pueden ser habilitados o deshabilitados desde el menú de opciones del clic derecho.
Todos los scripts pueden ser removidos de la interfaz a través de la opción del clic derecho "Remover Script".
