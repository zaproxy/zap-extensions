Mga script sa tree na tab

Ang mga script sa tree na tab ay nagpapakita sa iyo ng lahat ng mga script na kasalukuyang mong kinarga ang organisado ayon sa uri.
Nagpapakita din ito ng mga template na kung saan mayroon kang magagamit - ito ay hindi pwede pero makatakbo ng direkta, gamitin mo ito upang lumikha ng mga bagong mga script.
Ito ay pumapayag din sa iyo para magdagdag ng bagong mga script, magkarga, mag-save at alisin ang mga ito.
Ang tab ay kinabibilangan ng isang kasangkapan na kung saan ay nagbibigay-daan sa iyo upang:

  • I-load ang isang script mula sa filestore
  • I-save ang isang script sa filestore
  • Lumikha ng bagong script
Ang mga script na tatakbo sa loob ng ZAP na mga komponent, tulad ng Aktibong Scanner, ay maaaring mapagana at hindi mapagana sa pamamagitan ng kanang klik na menu opsyon.
Ang lahat ng mga script ay maaaring alisin mula sa isang UI sa pamamagitan ng isang kanang klik 'Alisin ang Script' menu na opsyon.
