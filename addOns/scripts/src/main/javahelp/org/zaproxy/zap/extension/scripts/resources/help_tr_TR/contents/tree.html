Komut dosyaları ağacı sekmesi

Komut dosyası ağacı sekmesi, türe göre sıralanmış yüklediğiniz tüm komut dosyalarını gösterir.
It also shows you which templates you have available - these cannot be run directly, you use them to create new scripts.
Ayrıca yeni komut dosyaları eklemenize, yüklemenize, kaydetmenize ve çıkarmanıza olanak verir.
The tab includes a toolbar which allows you to:

  • Dosya deposundan komut dosyası yükleme
  • Dosya deposuna komut dosyası kaydetme
  • Yeni bir komut dosyası oluştur
Aktif Tarayıcı gibi ZAP bileşenleri içinde çalışan komutlar sağ tık menü seçeneğini kullanarak etkinleştirilebilir ve devre dışı bırakılabilir.
Tüm komut dosyaları sağ tık 'Komut Dosyası Kaldır' menü seçeneğini kullanarak kullanıcı arayüzünden kaldırılabilir.
