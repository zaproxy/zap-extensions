Scripts tree tab

Script pohon tab menunjukkan anda semua script anda saat ini telah dimuat diselenggarakan oleh jenis.
Hal ini juga menunjukkan anda yang template yang telah tersedia ini tidak bisa dijalankan secara langsung, anda menggunakan mereka untuk membuat script baru.
Hal ini juga memungkinkan anda untuk menambahkan script baru, beban, menyimpan dan menghapus mereka.
Tab termasuk toolbar yang memungkinkan anda untuk:

  • Beban script dari filestore
  • Beban script dari filestore
  • Buat sebuah script baru
Skrip yang dijalankan didalam komponen ZAP, seperti Pemindai Aktif yang bisa diaktifkan dan dinonaktifkan melalui opsi pada menu klik kanan.
Semua script yang dapat dihapus dari UI melalui klik kanan 'Menghapus Naskah ini' menu pilihan.
