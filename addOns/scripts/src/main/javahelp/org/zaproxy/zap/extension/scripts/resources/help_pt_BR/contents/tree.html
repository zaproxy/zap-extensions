Aba da Árvore de Scripts

A aba Árvore de Scripts mostra todos os scripts que estão disponíveis, organizados por tipo.
Ela também mostra quais templates você tem disponíveis - não podem ser executados mas servem para criar novos scripts.
Ela também permite que você adicione novos scripts, carregue, salve e remova.
A aba inclui uma barra de ferramentas que permite a você:

  • Carregar um script do disco
  • Salvar um script no disco
  • Criar um novo script
Scripts que são executados dentro dos componentes de ZAP, tais como o Scanner ativo, podem ser habilitados e desabilitados através do menu de botão direito do mouse.
Todos os scripts podem ser removidos da interface do usuário através de um clique direito na opção de menu 'Remover Script'.
