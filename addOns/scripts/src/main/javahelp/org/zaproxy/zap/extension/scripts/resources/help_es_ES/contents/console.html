Pestaña de Consola de Script

La Consola de Scripts le permite escribir scripts que se ejecutan en ZAP.
Está compuesta por:

  • Una barra de herramientas
  • Un área en el que puede escribir sus scripts
  • Un área para mensajes de depuración y error
You can run 'Stand alone' scripts from within this tab using the 'Run' button on the tab toolbar.
Todos los demás tipos de scripts serán ejecutados cuando sean habilitados o si son explícitamente invocados.

Para crear un nuevo script de cargar o cambiar scripts vea la The Scripts 'tree' tab.

Haga click derecho en el área del script para opciones de visualización y edición.

Plantillas pueden verse en la consola pero no pueden ser editadas.

Auto-completar

La consola soporta una forma limitada de auto-completado, controlada por un botón en la barra de herramientas.
Cuando está activado, aparecerá una ventana emergente cuando usted escriba el nombre de uno de los parámetros definidos, seguidos por un punto - esto le mostrará todos los métodos disponibles para que los llame. Seleccionas uno de ellos insertará el método y los parámetros en su script. Si usted escribe otro punto inmediatamente después de la llamada al método, entonces le será mostrado otro aviso para los métodos disponibles para el tipo de devolución, si hubiera. Usted podrá seguir ampliando los tipos devueltos hasta que empiece a escribir otra cosa.
