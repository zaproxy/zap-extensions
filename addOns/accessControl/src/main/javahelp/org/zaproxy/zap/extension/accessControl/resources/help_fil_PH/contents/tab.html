Access Kontrol ng Istatus Tab

Ang Access Kontrol ng Istatus Tab ay nag papahintulot sa pagpapanimula ng bagong Access Control na sinusubukan at ipinapakita ang mga resulta na nakuha. Ang bawat pag gamit para sa bawat URL na inaatake ng ZAP, ang isang entry ay idinagdag sa may impormasyon tungkol sa:

  • Ang mga id ng ZAP ng mga mensahe na naipadala
  • ang mga ginamit na paraan ng HTTP
  • ang mga pinagkukunan ng URL
  • ang mga istatus kowd ng mga tugon ng HTTP
  • ang gumamit na kung kanino ang pinupunto ng pinagkukunan ay na-access
  • kung ang kahilingan ay natukoy bilang pagiging awtorisado o hindi
  • ang patakaran ng access ay ginamit kung saan alinman sa dalawa ang naka direkta ang tinukoy o ang ipahiwatig na nakabatay sa pinagmulan na tinukoy na patakaran
  • ang nakuhang resulta: matagumpay (berdeng check) kung ang gumaganang patakaran ay sinusundan ng nabigo (pulang cross) kung hindi

     Pagsubok sa mga konsepto ng acess kontrol para sa isang maiksing panimula para sa Pagsubok ng Access Kontrol
     Ang konteksto ng mga opsyon sa Access Kontrol matutunan ang tungkol sa mga opsyon na konektado sa mga konteksto
