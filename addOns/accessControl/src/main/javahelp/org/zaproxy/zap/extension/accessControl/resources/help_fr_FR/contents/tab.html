Onglet État du contrôle d’accès

L’onglet État du contrôle d’accès permet de lancer de nouveaux tests de contrôle d’accès et affiche les résultats obtenus. Pour chaque utilisateur et pour chaque URL attaquée par ZAP, une entrée est ajoutée avec des informations sur :

  • Identifiant du message envoyé par ZAP
  • La méthode HTTP utilisée
  • L'URL de la ressource
  • Le code d'état HTTP de la réponse
  • L’utilisateur à partir duquel la ressource a été consultée
  • Si la requête a été identifiée comme autorisée ou non
  • La règle d’accès utilisée, définie directement ou déduite des règles définies au niveau parent
  • Le résultat obtenu : réussi (coche verte) si la règle d’accès a été respectée, ou échec (croix rouge) dans le cas contraire
