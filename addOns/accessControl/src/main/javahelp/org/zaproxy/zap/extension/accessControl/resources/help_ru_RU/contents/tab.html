Вкладка «Статус контроля доступа»

Вкладка «Состояние контроля доступа» позволяет начать новое тестирование контроля доступа и отображает полученные результаты. За для каждого Пользователя и для каждого URL-адреса, атакованного ZAP, добавляется запись с информацией о:

  • Идентификатор ZAP отправленного сообщения
  • используемый метод HTTP
  • URL-адрес ресурса
  • код состояния HTTP ответа
  • Пользователь, с которого был осуществлен доступ к ресурсу
  • был ли запрос идентифицирован как авторизованный или нет
  • используемое правило доступа, которое было либо напрямую определено, либо выведено на основе правил, определенных родителем
  • полученный результат: успешно (зеленая галочка), если правило доступа было соблюдено или не удалось (красный крест) в противном случае
