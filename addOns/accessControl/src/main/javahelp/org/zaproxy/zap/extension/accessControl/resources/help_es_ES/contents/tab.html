Pestaña de Estado de Control de Acceso

La Pestaña de Estado de Control de Acceso permite iniciar una prieba de Control de Acceso y muestra los resultados obtenidos. Por cada Usuario y por cada URL atacada por ZAP, se agrega una entrada con información sobre:

  • ID de ZAP del mensaje enviado
  • el método HTTP utilizado
  • la URL del recurso
  • el código de estado HTTP de la respuesta
  • el usuario desde cuyo punto se accesó al recurso
  • si la solicitud fue identificada como autorizada o no
  • la regla de acceso utilizada, la cual fue o definida directamente o inferida en base a las reglas definidas de los padres
  • el resultado obtenido: exitosa (marca de verificación verde) si se siguió la regla de acceso o fallida (equis roja) si ocurrió lo contrario
