Параметры контекста управления доступом

Параметры контекста управления доступом представлены в виде панели для каждого контекста при открытии свойств сеанса. диалог. Эта панель позволяет пользователям ZAP определять правила доступа для каждого пользователя каждого контекста.

Как упоминалось на странице концепции, ZAP использует древовидную структуру. структура URL-адресов. Таким образом, при настройке правил доступа необходимо явно задать только одно правило для всего поддерево, а для остальных узлов выводятся правила. Три возможных значения могут быть установлены для любого узла в Context для каждый Пользователь:

  • Разрешено — правило для узла явно установлено как разрешено.
  • Запрещено — правило для узла явно установлено как Запрещено.
  • Унаследовано. Правило для узла выводится на основе ближайшего родителя с явно заданным правилом.

По умолчанию правила для любого узла имеют значение Унаследовано, а корневой узел имеет значение Неизвестно. Таким образом, для любого узла без родителя с явно настроенным правилом предполагаемое правило будет по умолчанию равно Неизвестно .
