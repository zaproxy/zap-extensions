Aba de Status de controle de acesso

A aba de Status de controle de acesso permite que o início de novos testes de controle de acesso e exibe resultados obtidos. Para cada usuário e para cada URL atacada pelo ZAP, uma entrada é adicionada com informações sobre:

  • ID da mensagem enviada, fornecido pelo ZAP
  • o método HTTP usado
  • a URL do recurso
  • o código de status HTTP da resposta
  • o usuário que acessou o recurso
  • se a solicitação foi identificada como sendo autorizado ou não
  • a regra de acesso usada, se foi definida diretamente ou inferida com base nas regras-pai
  • o resultado obtido: sucesso (check verde) se a regra de acesso foi seguida ou falha (cruz vermelha) se não foi
