访问控制状态标签

访问控制状态标签允许开始新的访问控制测试并显示测试结果。 对于每个用户和每个受到 ZAP 攻击的 URL，都会添加一个条目，其中包含相关信息：

  • 发送信息的 ZAP ID
  • 使用的 HTTP 方法
  • 该资源的 URL
  • 响应的 HTTP 状态代码
  • 访问资源的用户
  • 申请是否被确认为已获授权
  • 使用的访问规则，该规则可以是直接定义的，也可以是根据父级定义的规则推断出来的
  • 获得的结果：如果遵循了访问规则，则成功（绿色勾选）；否则失败（红色叉号）。
