Opções de contexto de controle de acesso

As opções de contexto de controle de acesso estão presentes como um painel para cada contexto, ao se abrir a caixa de diálogo Propriedades da sessão. Este painel permite aos usuários do ZAP definirem as regras de acesso para cada usuário de cada contexto.

Como mencionado na página de conceitos, o ZAP está fazendo uso da estrutura baseada em árvore de URLs. Então, ao configurar as regras de acesso, apenas uma regra precisa ser definida explicitamente para uma subárvore inteira, enquanto para os outros nós regras são inferidas. Três valores possíveis podem ser definidos para qualquer nó de contexto para cada usuário:

  • Permitido - a regra para o nó é explicitamente definida como Permitido
  • Negado - a regra para o nó é explicitamente definida como negado
  • Herdada - a regra para o nó é inferida com base no pai mais próximo com uma regra definida explicitamente

Por padrão, as regras para qualquer nó possuem um valor herdado e o nó de raiz tem um valor desconhecido. Assim, para qualquer nó sem um pai com uma regra explicitamente configurada, a regra inferida padrão será desconhecido.
