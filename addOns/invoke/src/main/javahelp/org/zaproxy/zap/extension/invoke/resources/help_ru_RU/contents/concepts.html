Вызов приложений

Другие приложения могут быть вызваны с передачей контекстной информации, такой как URL-адрес выбрано сообщение.
Так, например, Nmap можно вызвать, передав сайт который вы хотите, чтобы он сканировал.

Приложения настраиваются с помощью Опций Приложения экран .

Доступ через

     Вкладка Сайты Пункт контекстного меню «Запустить приложение»
     Вкладка История Пункт контекстного меню «Запустить приложение»
