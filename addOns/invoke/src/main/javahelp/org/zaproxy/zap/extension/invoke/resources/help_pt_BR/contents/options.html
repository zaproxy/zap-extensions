Tela de opções de aplicações

Esta tela permite que você configure aplicações that can be invoked.
By default there are no applications available, you need to add all of the applications that you want to use.

Nome para exibição

O nome que será usado para esta aplicação no ZAP.

Comando completo

O caminho completo do aplicativo que você deseja chamar.
Only one command can be specified. If you want to run multiple commands in one go, or set up environmental variables, then you should create a script or batch file and then invoke this from ZAP.

Parâmetros

Os parâmetros que ZAP deve passar para o comando.
As seguintes tags podem ser usadas para passar parâmetros:
     %url% a url completa - por exemplo, 'http://localhost/test?a=b'
     %site% o site - por exemplo, ' http://localhost:8080/ /'
     %host% o nome do host - por exemplo, 'localhost'
     %port% a porta - por exemplo, '80'
     %cookie% o primeiro campo de cookie do cabeçalho da solicitação (se houver)
     %postdata% os dados de postagem enviados, caso existam, com newlines substituídas por "\n"
     %msgid% the HTTP message id to fetch data from the API (/JSON/core/view/message/?id=$msgid), does not alway exist, will return -1 if no message id
     %header-{{ header }}% the request header by name (if any), e.g. %header-user-agent% would return the value of the User-Agent header
Então, se um comando aceita uma URL, usando '-u' você pode especificar parâmetros como: '-u %url%'

Capturar as Saídas

Se marcada, o comando é executado e qualquer saída gerada é exibida na Aba da saída.
You should always use this option when running scripts or commands that do not have a UI.
You can also use this option for troubleshooting - if an application does not run as expected then copy the command run and try it from a command line prompt.

Saída para Anotação

Se marcada, em seguida, qualquer saída será anexada a uma Nota anexada ao registro da Aba de histórico .
Note que alguns nós na Aba de Sites não têm uma Aba de histórico com registros associados - para esses nós esta opção não tem efeito, não funciona.
