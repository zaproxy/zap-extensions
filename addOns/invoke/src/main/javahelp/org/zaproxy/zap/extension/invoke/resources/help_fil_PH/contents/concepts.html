Pagtawag ng mga Aplikasyon

Ang ibang mga aplikasyon ay maaring matawag sa pagpasa ng impormasyon ng konteksto, kagaya ng URL sa napiling mensahe.
Kaya, halimbawa, ang Nmap ay maaring matawag sa pagpasa ng sayt kung saan mo gusto itong ma-scan.

Ang mga aplikasyon ay nakompigyura gamit ang Mga Pagpipilian ng mga Aplikasyon na skrin.

Ma-akses gamit ang

     Mga sayt na tab 'Patakbuhin ang apikasyon' kanang pagpindot ng menu na item
     Pangkasaysayan na tab 'Patakbuhin ang aplikasyon' kanang pagpindot ng menu na item
