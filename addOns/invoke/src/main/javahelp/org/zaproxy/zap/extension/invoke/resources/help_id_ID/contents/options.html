Pilihan Aplikasi layar

Layar ini memungkinkan anda untuk mengkonfigurasi aplikasi that can be invoked.
By default there are no applications available, you need to add all of the applications that you want to use.

Tampilan Nama

Nama yang akan digunakan untuk aplikasi ini didalam ZAP.

Perintah Penuh

Path lengkap dari aplikasi yang anda inginkan untuk memanggil.
Only one command can be specified. If you want to run multiple commands in one go, or set up environmental variables, then you should create a script or batch file and then invoke this from ZAP.

Parameter

Parameter yang harus dilalui ZAP ke perintah.
Tag berikut dapat digunakan untuk melewati parameter:
     %url% url lengkap, misalnya 'http://localhost/test?a=b'
     %site% situs, misalnya ' http://localhost: 8080 /'
     %host% nama host, misalnya 'localhost'
     %port% pelabuhan, misalnya '80'
     %cookie% field cookie pertama dari header permintaan (jika ada)
     %postdata% data POST yang dikirimkan, jika ada, dengan setiap baris diganti dengan "\n"
     %msgid% the HTTP message id to fetch data from the API (/JSON/core/view/message/?id=$msgid), does not alway exist, will return -1 if no message id
     %header-{{ header }}% the request header by name (if any), e.g. %header-user-agent% would return the value of the User-Agent header
Jadi jika perintah diterima menggunakan URL '-u' Anda akan menentukan parameter sebagai: '-u %url%'

Menangkap Output

Jika dicek lalu perintah berjalan dan setiap output yang dihasilkan akan ditampilkan di Tab keluaran.
You should always use this option when running scripts or commands that do not have a UI.
You can also use this option for troubleshooting - if an application does not run as expected then copy the command run and try it from a command line prompt.

Keluaran ke Catatan

Jika dicentang kemudian output akan ditambahkan Catatan melekat ke yang relevan Riwayat tab rekam.
Perhatikan bahwa beberapa node dalam Tab situs tidak berhubungan Tab riwayat catatan - untuk node tersebut pilihan ini tidak akan memiliki efek.
