Mga pagpipilian ng mga Aplikasyon na skrin

Ang iskrin na ito ay magpapahintulot sa iyo na ikompigyura ang mga aplikasyon that can be invoked.
By default there are no applications available, you need to add all of the applications that you want to use.

Ipakita ang Pangalan

Ang pangalan na magagamit sa aplikasyong ito sa ZAP.

Ang Kabuuang Utos

Ang kabuuang daanan ng aplikasyon na gusto mong tawagin.
Only one command can be specified. If you want to run multiple commands in one go, or set up environmental variables, then you should create a script or batch file and then invoke this from ZAP.

Mga parameter

Ang mga parameter na dapat mapasa ng ZAP patungo sa utos.
Ang sumusunod na mga tag ay maaring magamit upang mapasa ang mga parameter:
     %url% sa kabuuang url, e.g 'http://localhost/test?a=b'
     %site% ang sayt, e.g. 'http://localhost:8080/'
     %host% ang hostname, e.g. 'localhost'
     %port% ang port, e.g. '80'
     %cookie% ang unang cookie na field mula sa rekwest na header (kung mayroon)
     %postdata% ang POST na datos na napadala, kung mayroon, na mayroong kahit anong mga newline na ipinagpalit ng "\n"
     %msgid% the HTTP message id to fetch data from the API (/JSON/core/view/message/?id=$msgid), does not alway exist, will return -1 if no message id
     %header-{{ header }}% the request header by name (if any), e.g. %header-user-agent% would return the value of the User-Agent header
Kaya kung ang utos ay tumanggap ng isang URL gamit ang '-u' ikaw ay magtutukoy ng mga parameter bilang: '-u%url%'

Pagkuha ng Output

Kung susuriin ay mapapatakbo ang utos at ang kahit anong mga output na magagawa nito ay makikita sa Tab ng pag-output.
You should always use this option when running scripts or commands that do not have a UI.
You can also use this option for troubleshooting - if an application does not run as expected then copy the command run and try it from a command line prompt.

Mag-output patungo sa Note

Kung nasuri ay ang kahit anong na-output ay madudugtong sa Note na nakakabit sa may-katuturang Pagkasaysayan ng tab na talaan.
Tandaan na ang ibang mga noda sa loob ng Mga Sayt na tab ay walang kaugnay Na Tab ng Kasaysayan na mga talaan - para sa mga noda ang pagpipiliang ito ay magkakaroon ng walang epekto.
