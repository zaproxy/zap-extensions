Pantalla de opciones de aplicaciones

Esta pantalla te permite configurar las aplicaciones that can be invoked.
By default there are no applications available, you need to add all of the applications that you want to use.

Mostrar Nombre

El nombre que será usado para ésta aplicación ZAP.

Comando completo

La ruta de acceso completa de la aplicación que usted desea invocar.
Only one command can be specified. If you want to run multiple commands in one go, or set up environmental variables, then you should create a script or batch file and then invoke this from ZAP.

Parámetros

Los parámetros que ZAP debería pasar al comando.
Las siguientes etiquetas pueden ser usadas para el paso parámetros:
     %url% la dirección url completa, por ejempo: 'http://localhost/test?a=b'
     %site% el sitio, como por ejemplo: 'http://localhost:8080/'
     %host% el nombre del host, por ejemplo: 'localhost'
     %port% el puerto, por ejemplo: '80'
     %cookie% el primer campo de cookies del encabezado de la solicitud (de haber alguno)
     %postdata% la data POST enviada, si existe, junto con cualquier línea nueva reemplazada con "\n"
     %msgid% the HTTP message id to fetch data from the API (/JSON/core/view/message/?id=$msgid), does not alway exist, will return -1 if no message id
     %header-{{ header }}% the request header by name (if any), e.g. %header-user-agent% would return the value of the User-Agent header
Así que si un comando aceptó una URL utilizando "-u" el parámetro se especificaría como: "-u %url%"

Capturar Salida

De estar marcado y ser ejecutado el comando, cualquier salida que genere será mostrada en la Pestaña de salida.
You should always use this option when running scripts or commands that do not have a UI.
You can also use this option for troubleshooting - if an application does not run as expected then copy the command run and try it from a command line prompt.

Salida a Nota

De estar marcada, cualquier salida será anexada a una Nota adjunto al correspondiente registro de la Pestaña de Historia .
Considere que en algunos nodos las Pestañas de sitios no tienen asociados registros de la Pestaña de Historia - Para estos nodos, ésta opción no tendrá efecto alguno.
