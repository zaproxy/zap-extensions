Invocar Aplicaciones

Otras aplicaciones pueden invocarse si se pasan como información contextual, tal como la URL del mensaje seleccionado.
Por ejemplo, Nmap puede ser invocada pasando el sitio que desea escanear.

Las aplicaciones se configuran utilizando la pantalla de Opciones de Aplicaciones.

Acceso vía

     Sitios tab el elemento del menú de clic derecho llamado "Ejecutar aplicación"
     Pestaña Historia el elemento del menú de clic derecho llamado "Ejecutar aplicación"
