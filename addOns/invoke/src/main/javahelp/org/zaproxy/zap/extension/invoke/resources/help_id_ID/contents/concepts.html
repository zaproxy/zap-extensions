Memanggil Aplikasi

Aplikasi lain bisa dipanggil melewati konteks informasi, seperti URL pesan yang terpilih.
Jadi, contohnya Nmap dapat dipanggil untuk melewati situs yang ingin anda pindai.

Aplikasi dikonfigurasi menggunakan Pilihan Aplikasi layar.

Diakses melalui

     Tab situs 'Jalankan aplikasi' klik kanan pada menu item
     Sejarah tab 'Jalankan aplikasi' klik kanan pada menu item
