选项应用程序屏幕

此屏幕允许您配置 应用程序 that can be invoked.
默认情况下没有可用的应用程序，您需要添加所有要使用的应用程序。

显示名称

将在ZAP中用于此应用程序的名称。

完整的命令

要调用的应用程序的完整路径。
只能指定一个命令。如果您想要一次运行多个命令，或者设置环境变量，则应该创建一个脚本或批处理文件，然后从ZAP中调用它。

参数

ZAP应传递给命令的参数。
下列标记可用于传递参数：
     %url% 完整的url，例如"http://localhost/test?a=b"
     %site% 站点，例如："http://localhost:8080/"
     %host% 主机名，例如"本地主机"
     %port% the port, e.g. '80'
     %cookie% 请求标头中的第一个cookie字段（如果有）
     %postdata% the POST data sent, if any, with any newlines replaced with "\n"
So if a command accepted a URL using '-u' you would specify the parameters as: '-u %url%'

捕获输出

如果选中，则命令运行，并且它生成的任何输出都将显示在 输出选项卡.
You should always use this option when running scripts or commands that do not have a UI.
You can also use this option for troubleshooting - if an application does not run as expected then copy the command run and try it from a command line prompt.

Output to Note

If checked then any output will be appended to a 备注 attached to the relevant 历史记录选项卡 记录。
Note that some nodes in the 站点选项卡 没有关联 历史记录选项卡 records - for those nodes this option will have no effect.
