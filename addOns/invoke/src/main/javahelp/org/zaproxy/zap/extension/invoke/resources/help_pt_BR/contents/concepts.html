Invocar aplicações

Other applications can be invoked passing in context information, such as the URL of the message selected.
So, for example, Nmap could be invoked passing the site which you want it to scan.

Essas aplicações são configuradas usando-se a Tela de opções de aplicações.

Acessado via

     Aba de Sites Item de menu de clique direito "Executar aplicação"
     Aba de histórico Item de menu de clique direito "Executar aplicação"
