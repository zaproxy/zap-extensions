Uygulamalar ekranı seçenekleri

Bu ekran sizin çağırılabilir uygulamaları that can be invoked.
By default there are no applications available, you need to add all of the applications that you want to use.

Ekran Adı

Bu uygulamanın ZAP'ta kullanılacak adı.

Tam Komut

Çağırmak istediğiniz uygulamanın tam yolu.
Only one command can be specified. If you want to run multiple commands in one go, or set up environmental variables, then you should create a script or batch file and then invoke this from ZAP.

Parametreler

ZAP'ın komuta geçmesi gereken parametreler.
Aşağıda bulunan etiketler parametreleri iletmek amacıyla kullanılabilir:
     %url% tam url, 'http://localhost/test?a=b'
     %site% site, 'http://localhost:8080/'
     %host% ana makine adı, örn. 'localhost'
     %port% port, örn. '80'
     %cookie% (eğer varsa) istek başlığında bulunan ilk çerez alanı
     %postdata% Gönderilen POST verileri, eğer varsa, yeni satırlarla birlikte "\n" ile değiştirilir
     %msgid% the HTTP message id to fetch data from the API (/JSON/core/view/message/?id=$msgid), does not alway exist, will return -1 if no message id
     %header-{{ header }}% the request header by name (if any), e.g. %header-user-agent% would return the value of the User-Agent header
Yani bir komut bir URL'yi '-u' kullanarak kabul ettiyse, parametreleri şu şekilde belirtirsiniz: '-u %url%'

Çıkış Yakalama

Eğer seçili ise o halde komutu çalıştırır ve onun ürettiği çıktılar gösterilir Çıkış sekmesi.
You should always use this option when running scripts or commands that do not have a UI.
You can also use this option for troubleshooting - if an application does not run as expected then copy the command run and try it from a command line prompt.

Not çıktısı

Eğer herhangi bir çıktı eklenirse Not ilgiliye bağlı Geçmiş sekmesi kaydet.
Not olarak bazı düğümler Siteler sekmesi ilişkili değil Geçmiş sekmesi kayıtlar - bu düğümler için bu seçenek bir etkiye sahip olmayacaktır.
