Code Dx

Code Dx eklentisi uyumlu raporlar oluşturmanıza izin verir ve onları bir Code Dx sunucusuna yükler. Rapor menüsü altında iki seçenek vardır.

Code Dx: Yükleme Raporu

Bu seçenek sizin için bir rapor oluşturacak ve onu bir Code Dx sunucusuna yükleyecek. Onu seçmek sizi Sunucu URL'si ve API Anahtarı için yönlendirecek. Alanları doldurduktan sonra, API Anahtarı için hazır projelerin bir listesini almak için yenile butonuna tıkla. Sonra sonuçları Code Dx sunucusuna göndermek için Yükle butonuna tıkla.

Code Dx: XML Raporu Oluştur

Bu seçenek, HTTP istek ve yanıt verilerini içeren standart bir XML raporu oluşturacaktır.
