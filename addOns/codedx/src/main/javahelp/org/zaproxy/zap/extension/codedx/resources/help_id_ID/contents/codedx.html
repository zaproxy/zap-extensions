Kode Dx

Kode Dx add-on memungkinkan anda untuk menghasilkan laporan yang kompatibel dan mengunggah mereka untuk server Kode Dx. Ada dua pilihah di bawah menu laporan.

Kode Dx: Laporan

Pilihan ini akan menghasilkan laporan untuk anda dan mengunggah ke server Kode Dx. Memilihnya akan meminta anda untuk URL Server dan kunci API. Setelah mengisi kolom, klik tombol menyegarkan untuk mendapatkan daftar proyek yang tersedia ke kunci API. Kemudian klik tombol Unggah untuk mengirim hasil ke server Kode Dx.

Kode Dx: Menghasilkan Laporan XML

Pilihan ini akan menghasilkan laporan standar XML yang mencakup permintaan HTTP dan data tanggapan.
