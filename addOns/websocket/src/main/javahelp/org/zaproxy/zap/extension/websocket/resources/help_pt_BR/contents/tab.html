Aba de WebSocket

A aba de WebSockets exibe todas as mensagens de conexões de WebSockets. Com o ZAP ativo, visite por exemplo BrowserQuest da Mozilla para ver WebSockets em ação.

Filtros

Seletor de canais

You can restrict the display of messages to one specific WebSocket channel or all.
If you select a specific channel, then the Show handshake button () becomes enabled. When you click on it, the corresponding HTTP handshake is shown in the Request/Response tab.

Diálogo de filtro

In the filter dialog () you can further control which messages are displayed.

Pontos de interrupção

You can set custom breakpoints using the Add Custom Breakpoints button ().
Alternatively you can use the Right click menu.

Menu do botão direito

Clicar com o botão direito sobre uma mensagem de WebSockets faz aparecer um menu que lhe permitirá:

Excluir da WebSocket

WebSocket connections can be excluded from the WebSockets tab by adding an excluded URL to the Session Properties. Clicar neste item exibe as propriedades da sessão com a URL preenchida. Excluir uma conexão WebSocket não resultará no encerramento da conexão, mas em encaminhá-la sem processamento adicional.

Interrupção...

This will bring up the Add Break Point dialog which allows you to set up a custom break point.
