Pestaña WebSocket

La pestaña WebSockets muestra todos los mensajes de las conexiones WebSocket. Mientras que ZAP está activa, visita por ejemplo: Mozilla's BrowserQuest para ver WebSockets en acción.

Filtros

Selector de canal

You can restrict the display of messages to one specific WebSocket channel or all.
If you select a specific channel, then the Show handshake button () becomes enabled. When you click on it, the corresponding HTTP handshake is shown in the Request/Response tab.

Cuadro de diálogo de filtro

In the filter dialog () you can further control which messages are displayed.

Puntos de interrupción

You can set custom breakpoints using the Add Custom Breakpoints button ().
Alternatively you can use the Right click menu.

Excluir de WebSocket

WebSocket connections can be excluded from the WebSockets tab by adding an excluded URL to the Session Properties. Al hacer clic en este elemento, abre las propiedades de la sesión con el URL ya precargado. Excluir una conexión WebSocket no resultará en el cierre de la conexión, sino será reenviada sin que se siga procesando.

Interrupción...

This will bring up the Add Break Point dialog which allows you to set up a custom break point.
