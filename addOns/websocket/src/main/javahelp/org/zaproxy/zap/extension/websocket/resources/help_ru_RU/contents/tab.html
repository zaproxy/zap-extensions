Вкладка WebSocket

На вкладке WebSockets отображаются все сообщения от соединений WebSocket. Пока ZAP активен, посетите, например, Mozilla's BrowserQuest. чтобы увидеть WebSockets в действии.

Фильтры

Селектор каналов

Если вы выберете конкретный канал, Показать рукопожатие кнопка ( ) становится включенным. Когда вы щелкаете по нему, соответствующее рукопожатие HTTP отображается на вкладке «Запрос/ответ».
Если вы выберете конкретный канал, Show handshake (Показать рукопожатие ) кнопка ( ) становится включенным. Когда вы нажимаете на нее, соответствующее рукопожатие HTTP отображается на вкладке Запрос/Ответ.

Диалоговое окно фильтра

В диалоговом окне фильтра ( ) вы можете дополнительно контролировать, какие сообщения отображаются.

Исключить из WebSocket

Соединения WebSocket можно исключить из вкладки WebSockets, добавив исключенный URL-адрес Свойства сессии. Нажав на этот элемент вызывает свойства сеанса с предварительно заполненным URL-адресом. Исключение соединения WebSocket не приведет к закрытию соединения, а в пересылке их без дальнейшей обработки.
