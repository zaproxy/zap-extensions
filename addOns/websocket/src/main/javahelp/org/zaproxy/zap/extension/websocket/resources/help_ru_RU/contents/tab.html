Вкладка WebSocket

На вкладке WebSockets отображаются все сообщения от соединений WebSocket. Пока ZAP активен, посетите, например, Mozilla's BrowserQuest. чтобы увидеть WebSockets в действии.

Фильтры

Селектор каналов

You can restrict the display of messages to one specific WebSocket channel or all.
If you select a specific channel, then the Show handshake button () becomes enabled. When you click on it, the corresponding HTTP handshake is shown in the Request/Response tab.

Диалоговое окно фильтра

In the filter dialog () you can further control which messages are displayed.

Прерывания

You can set custom breakpoints using the Add Custom Breakpoints button ().
Alternatively you can use the Right click menu.

Исключить из WebSocket

WebSocket connections can be excluded from the WebSockets tab by adding an excluded URL to the Session Properties. Нажав на этот элемент вызывает свойства сеанса с предварительно заполненным URL-адресом. Исключение соединения WebSocket не приведет к закрытию соединения, а в пересылке их без дальнейшей обработки.
