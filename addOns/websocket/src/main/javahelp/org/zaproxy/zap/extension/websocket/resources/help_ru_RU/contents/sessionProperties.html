Специфические свойства сеанса WebSocket

Исключить из WebSocket

Это позволяет вам управлять URL-адресами, где связь WebSocket не обрабатывается в ZAP.
Хотя все сообщения с исключенных URL-адресов пересылаются, ничего не сохраняется, и вы не можете просмотреть их в пользовательском интерфейсе. Эта функция полезна для больших объемов, критически важных для производительности соединений WebSocket.
Вы можете различать порт для одного домена, используя, например: example.com:443, чтобы исключить соединения WebSocket с example.com только через порт 443.
