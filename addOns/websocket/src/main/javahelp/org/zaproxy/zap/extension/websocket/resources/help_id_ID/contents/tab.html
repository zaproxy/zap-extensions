WebSocket tab

WebSocket tab memperlihatkan semua pesan dari WebSocket koneksi. Sementara ZAP adalah aktif, kunjungi misalnya: Mozilla PencarianBrowser untuk melihat WebSocket pada aktifitas.

Filter

Pemilih Saluran

Anda bisa membatasi tampilan dari pesan untuk suatu tertentu WebSocket saluran atau semuanya.
Jika anda memilih sebuah saluran tertentu, maka Menunjukkan jabat tangan tombol () diaktifkan. Saat anda mengkliknya, HTTP handshake yang sesuai ditampilkan dalam tab Permintaan/Respon.

Dialog Penyaring

Dalam dialog penyaring () anda dapat mengontrol lebih jauh dimana pesan-pesan ditampilkan.

Titik-selaan

Anda dapat mengatur kustom titik istirahat menggunakan Tambahkan Titik-Selaan Pilihan tombol ().
Alternatifnya anda bisa menggunakan Klik kanan menu.

Tidak termasuk dari WebSocket

Koneksi WebSocket dapat dikecualikan dari tab WebSocket dengan menambahkan sebuah URL pengecualian ke Properti Sesi. Klik pada item ini, akan menampilkan sesi properti dengan URL yang sudah terisi. Pengecualian koneksi WebSocket tidak akan berakibat pada penutupan koneksi, melainkan dalam meneruskan tanpa memproses lebih lanjut.

Break...

Ini akan memunculkan Tambahkan dialog Titik Selaan yang memungkinkan anda untuk mengatur break point Khusus.
