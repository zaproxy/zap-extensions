WebSocket tab

WebSocket tab memperlihatkan semua pesan dari WebSocket koneksi. Sementara ZAP adalah aktif, kunjungi misalnya: Mozilla PencarianBrowser untuk melihat WebSocket pada aktifitas.

Filter

Pemilih Saluran

You can restrict the display of messages to one specific WebSocket channel or all.
If you select a specific channel, then the Show handshake button () becomes enabled. When you click on it, the corresponding HTTP handshake is shown in the Request/Response tab.

Dialog Penyaring

In the filter dialog () you can further control which messages are displayed.

Titik-selaan

You can set custom breakpoints using the Add Custom Breakpoints button ().
Alternatively you can use the Right click menu.

Tidak termasuk dari WebSocket

WebSocket connections can be excluded from the WebSockets tab by adding an excluded URL to the Session Properties. Klik pada item ini, akan menampilkan sesi properti dengan URL yang sudah terisi. Pengecualian koneksi WebSocket tidak akan berakibat pada penutupan koneksi, melainkan dalam meneruskan tanpa memproses lebih lanjut.

Break...

This will bring up the Add Break Point dialog which allows you to set up a custom break point.
