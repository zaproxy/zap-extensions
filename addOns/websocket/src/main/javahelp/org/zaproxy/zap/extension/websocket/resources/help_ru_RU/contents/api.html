Каналы

Возвращает все зарегистрированные каналы веб-сокетов.

сообщение

Обязательные параметры:
  • channelId: идентификатор канала
  • messageId: идентификатор сообщения
Возвращает полную информацию об указанном сообщении.
«channelId» и «messageId» могут быть обнаружены через представление «сообщения».

сообщения

Дополнительные параметры:
  • channelId: идентификатор канала
  • start: смещение, с которого начинают возвращаться сообщения, начиная с 0.
  • count: количество возвращаемых сообщений.
  • payloadPreviewLength: максимальное количество байтов, возвращаемых для содержимого полезной нагрузки.
Возвращает список всех сообщений, соответствующих заданным критериям. По умолчанию возвращаются все сообщения
. ChannelId можно найти в представлении «каналы».
Параметры start и count можно использовать для постраничного просмотра сообщений
. 'payloadPreviewLength' можно использовать для ограничения размера возвращаемых данных.

Действия

sendTextMessage

Обязательные параметры:
  • channelId: идентификатор канала
  • исходящий: «Истина», если сообщение должно быть отправлено на сервер, и «Ложь», если оно должно быть отправлено клиенту
  • сообщение: сообщение для отправки
Отправляет указанное сообщение по указанному каналу.
ChannelId можно найти в представлении «каналы».
