TLS Debug

Este complemento exibe o rastreamento TLS em uma guia dedicada.

Digite a URL para se conectar no campo de entrada e clique em Check.

Clique direito no mouse o console de saída para limpar o rastreamento.

Observe que as configurações ZAP não são consideradas ainda. Estas são características para a próxima versão:

  • Selecione o nível de detalhes do rastreamento ("all", "ssl")
  • Use um certificado de cliente
  • Use um proxy de saída
