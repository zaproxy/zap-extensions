TLS Debug

Tambahan ini menampilkan jejak TLS pada tab khusus.

Ketik URL yang akan disambung ke kolom masukan dan tekan Periksa.

Klik kanan pada konsol output untuk menghapus jejaknya.

Perhatikan bahwa konfigurasi ZAP belum dipertimbangkan. Ini adalah fitur untuk versi berikutnya:

  • Pilih tingkat rincian jejak ("all", "ssl")
  • Gunakan sertifikat klien
  • Menggunakan sebuah proxy yang keluar
