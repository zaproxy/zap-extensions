Postman – 自動化フレームワーク対応

このアドオンは自動化フレームワークに対応しています。

このアドオンでは、URLまたはローカルファイルから Postman コレクションをインポートできます。

変数パラメーターの動作は、こちらで定義されている方法と同様です。

ジョブ: postman

postman ジョブでは、URLまたはファイルから Postman コレクションをインポートできます。 
  - type: postman         # Postman コレクションのインポート
    parameters:
      collectionFile:     # 文字列: Postman コレクションを含むローカルファイル。デフォルト: null、コレクションはインポートされません
      collectionUrl:      # 文字列: Postman コレクションを含むURL。デフォルト: null、コレクションはインポートされません
      variables:          # 文字列: key1=value1,key2=value2,... の形式でキーと値のペアとしてコンマ区切りで指定された変数リスト。これらの変数は、コレクション内の変数よりも優先されます
