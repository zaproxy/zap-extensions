Guided configuration

As we know, setting up authentication in ZAP is a multi-step process. A checklist of required configurations with their current configuration status(configured or not) is provided to assist manual configuration. The checklist understands the current configuration status and suggests you the next preferred step. Clicking the icon will open up relevant properties dialog where the necessary configuration can be done.

For each checklist item, its current configuration status is indicated using an Icon.


Configuration Status Icon Description
Good Necessary conditions are met
Bad Necessary conditions not met
Minimal Minimum required conditions are met
Not validated yet The checks need to be rerun
