Authentication Helper

In ZAP you can set up authentication for the web application being tested. This allows ZAP to run tests as an authenticated user. It is an excellent feature to have as with the ability of automatic (re)authentication ZAP can check for vulnerabilities that are hidden in authorized resources. This is also a good feature to test broken authentication and session management. However, setting up authentication in ZAP is a multi-step process and need to be done with care.

This add-on is your companion to set up and test authentication in ZAP. This add-on provides three main features.

  • Automated configuration
  • Guided configuration
  • Authentication status scanning
