Authentication status tab

Authentication status tab is used to show authentication status scan results.


  • The 'Check status' button launches the authentication helper dialog from which authentication status scan can be started.
  • The scan can be run on multiple Sites in parallel and the results for each scan are shown by selecting the scan via the 'Progress' pull-down.
  • The toolbar shows information about a scan and allows to control it. It provides a set of buttons which allow to:
    • Pause (and resume) selected scan
    • Stop the selected scan
    • Clean completed scans
    • Open the authentication helper options screen.
  • The progress bar shows how far the selected scan has progressed. It is also shown the number of active scans and the number of successful, failed, conflicting and unknown authentication statuses found in the selected scan.
