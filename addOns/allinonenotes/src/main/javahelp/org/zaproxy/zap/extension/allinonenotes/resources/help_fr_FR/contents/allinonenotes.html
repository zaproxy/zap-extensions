All In One Notes

Il s’agit d’une extension simple qui vous permet de voir toutes vos notes dans un seul panneau, ainsi que de les exporter. L’onglet "Historique" permet d’ajouter ou de modifier des notes pour une requête particulière. Cette extension rassemble toutes les notes dans une vue unique et pratique.

Cette extension fournit :

Panneau d’état

Un panneau d’état basique affichant un tableau des notes et leurs identifiants de requête correspondants. En cliquant sur une ligne, vous accédez automatiquement à la requête correspondante dans l’onglet Historique. Ce panneau inclut également un bouton "Export" permettant d’exporter toutes les notes vers un fichier CSV, ainsi qu’un bouton "Reload" permettant de recharger la vue des notes plus facilement.
