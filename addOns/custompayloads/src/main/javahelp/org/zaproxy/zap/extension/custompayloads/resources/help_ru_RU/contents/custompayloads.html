Пользовательские полезные нагрузки

Этот аддон добавляет панель параметров, из которой пользователи могут добавлять, обновлять, удалять полезные данные, созданные ими/выбранные для использования активными или пассивными правилами сканирования. которые поддерживают пользовательские полезные нагрузки (доступны через пункт меню «Параметры» в меню «Инструменты»).

Интерфейс панели опций также облегчает добавление нескольких полезных данных из файла.

You can see the Scan Rules which support Custom Payloads here.
