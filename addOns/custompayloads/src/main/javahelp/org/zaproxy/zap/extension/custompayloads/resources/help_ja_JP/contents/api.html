カスタムペイロードAPI

以下のビューはAPI経由で利用可能です：

ビュー

  • customPayloads(category): 現在読み込まれているすべてのペイロードを一覧表示します (カテゴリ、ペイロード、有効状態)。 カテゴリでフィルタすることもできます (任意)。
  • customPayloadsCategories: 利用可能なカテゴリをすべて一覧表示します。

アクション

  • addCustomPayload (category*, payload): カスタムペイロードを追加します (追加時に有効化されます)。
  • disableCustomPayloads(category): カスタムペイロードを無効化します。 カテゴリで絞り込むこともできます (任意)。 （カテゴリを指定しない場合はすべてが対象になります）
  • enableCustomPayloads(category): カスタムペイロードを有効化します。 カテゴリで絞り込むこともできます (任意)。 （カテゴリを指定しない場合はすべてが対象になります）
  • enableCustomPayload (category*, payload): カスタムペイロードを有効化します。
  • disableCustomPayload (category*, payload): カスタムペイロードを無効化します。
  • removeCustomPayload (category*, payload): カスタムペイロードを削除します。
注: * は必須パラメーターです。
