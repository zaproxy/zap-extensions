API Selenium

La API "selenium" permite fijar y ver las rutas a los WebDrivers requeridos y binary.

Vistas o acciones API

Nombre Tipo Parámetros Descripción
optionChromeDriverPath vista Devuelve la ruta actual a ChromeDriver
optionFirefoxBinaryPath vista Devuelve la ruta actual a los archivos binarios Firefox
optionFirefoxDriverPath vista Devuelve la ruta actual a la ruta Firefox driver (geckodriver)
optionIeDriverPath vista Devuelve la ruta actual a IEDriverServer
optionPhantomJsBinaryPath vista Devuelve la ruta actual a los archivos binarios PhantomJS
setOptionChromeDriverPath acción String Establece la ruta actual a ChromeDriver
setOptionFirefoxBinaryPath acción String* Establece la ruta actual a los archivos binarios Firefox
setOptionFirefoxDriverPath acción String* Sets the current path to Firefox driver (geckodriver)
setOptionIeDriverPath action String* Sets the current path to IEDriverServer
setOptionPhantomJsBinaryPath action String* Sets the current path to PhantomJS binary
Los parámetros señalados son obligatorios
