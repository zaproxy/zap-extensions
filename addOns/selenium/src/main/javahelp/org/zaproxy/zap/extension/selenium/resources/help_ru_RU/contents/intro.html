Selenium

Надстройка Selenium предоставляет Веб-Драйверы (WebDrivers), вызвать и удаленно управлять веб-браузерами. Он также включает в себя HtmlUnit веб-браузер , headless Java зависимости веб-браузера.

Он добавляет тип сценария «Selenium» — эти сценарии вызываются всякий раз, когда селен (selenium) запускает браузер, например, для Ajax Spider или для ручного просмотра.

Надстройка также добавляет возможность щелчка правой кнопкой мыши на панели ZAP, в которых перечислены URL-адреса (например, в дереве «История» и «Сайты»). который позволяет открыть выбранный URL в любом из поддерживаемых браузеров. Скрипты Selenium имеют доступ к запущенный браузер и может взаимодействовать с ним, например, запускать сценарии JavaScript, получать доступ к URL-адресам, заполнять формы, нажимать кнопки...
Примечание. Сценарии Selenium выполняются синхронно для AJAX Spider, чтобы предотвратить помехи. с помощью сканера позже, скрипты должны получить доступ к требуемым URL-адресам.

Чтобы использовать Firefox, Chrome, Opera, PhantomJS и Safari, вы должны установить их в вашей системе. ID браузера можно использовать для выбора браузере при настройке ZAP через командную строку или с помощью ZAP API (например, настроить AJAX Spider на использование того или иного браузера).

Некоторые требования (например, веб-драйверы) для браузеров можно настроить на экране Параметры Selenium.

Связанные веб-драйверы

ZAP предоставляет надстройки с WebDrivers, когда эти надстройки установлены, ZAP попытается использовать эти связанные WebDrivers по умолчанию. Некоторые ОС могут не иметь WebDriver для некоторых браузеров в этих случаях ZAP сообщит на панели параметров, что нет доступного связанного WebDriver. Входящие в комплект WebDrivers также можно (пере)установить с помощью кнопка «В комплекте» (например, если вручную был установлен другой WebDriver). Не все версии браузера поддерживаются встроенными веб-драйверами, рекомендуется по возможности использовать более новые/последние версии браузеров.

Примечание. Надстройки ZAP могут добавлять дополнительные браузеры.

Примечание Firefox/geckodriver. Существует проблема (Ошибка 1103196), которая предотвращает использование HTTPS-сайтов в версиях ≥ 48 и < 52.

PhantomJS Note: Есть проблема( ошибка #11342 ), что предотвращает сайты на локальном хосте, 127.0.0.1 и ::1 от проксирования через ZAP. Пока не будет исправления доступными, рекомендуется не использовать его в этих случаях. Некоторые надстройки могут отображать предупреждающее сообщение, когда это произойдет. В качестве обходного пути можно определить, в хосты файл, сопоставление доменного имени с локальным адресом и использование этого доменного имени.
