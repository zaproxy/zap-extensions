Selenium API

'selenium' API'ı gerekli WebDriver'larına ve binary'sine giden yolları belirlemeyi ve görüntülemeyi sağlar.

API eylemler / görüntüleme

Isim Tür Parametreler Açıklama
optionChromeDriverPath görüntüle ChromeDriver' a geçerli yoluna döner
optionFirefoxBinaryPath görüntüle Mevcut yolu Firefox binary'isine döndürür
optionFirefoxDriverPath görüntüle Mevcut yolu Firefox sürücüsüne (geckodriver) döndürür
optionIeDriverPath görüntüle Mevcut yolu IEDriverServer'a döndürür
optionPhantomJsBinaryPath görüntüle Mevcut yolu PhantomJS binary'sine döndürür
setOptionChromeDriverPath eylem String* Mevcut yolu ChromeDriver'a belirler
setOptionFirefoxBinaryPath eylem String* Mevcut yolu Firefox binary'sine belirler
setOptionFirefoxDriverPath eylem String* Mevcut yolu Firefox sürücüsüne (geckodriver'a) ayarlar
setOptionIeDriverPath eylem String* Mevcut yolu IEDriverServer'a belirler
setOptionPhantomJsBinaryPath eylem String* PhantomJS ikilisine dosya yolu belirle
Yıldızlı parametreler zorunludur
