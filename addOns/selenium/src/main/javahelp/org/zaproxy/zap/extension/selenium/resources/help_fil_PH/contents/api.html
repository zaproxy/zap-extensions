Ang Siliniyum ng API

Ang 'selenium' API ay pinapayagan na isaayos at makita ang mga daan ng mga kailngan ng WebDrivers at ng binary.

Ang actions/views ng API

Mga pangalan Mga klase Mga parametro Ang paglalaraawan
optionChromeDriverPath view Nag babalik ang kasalukuyang daan para sa Chromedrive
optionFirefoxBinaryPath view Nagbabalik ang kasalukuyang daan para sa binary ng Firefox
optionFirefoxDriverPath view Nagbabalik ang kasalukuyang daan para sa driver ng Firefox (geckodriver)
optionIeDriverPath view Nagbabalik ang kasalukuyang daan para sa IEDriverServer
optionPhantomJsBinaryPath view Nagbabalik ang kasalukuyang daan para sa binary ng PhantomJS
setOptionChromeDriverPath action String* Isaayos ang kasalukuyang daan para sa ChromeDriver
setOptionFirefoxBinaryPath action String* Isaayos ang kasalukuyang daan para sa binary ng Firefox
setOptionFirefoxDriverPath action String* Isaayos ang kasalukuyang daan para sa driver ng Firefox (geckodriver)
setOptionIeDriverPath action String* Isaayos ang kasalukuyang daan para sa IEDriverServer
setOptionPhantomJsBinaryPath action String* Isaayos ang kasalukuyang daan para sa binary ng PhantomJS
Ang maliliit na mga parametro ay iniuutos

     Ang Siliniyum para sa pangkalahatang ideya ng add-on na siliniyum
     Mga opsyon ng siliniyum sa iskrin para sa pangkalahatang ideya ng siliniyum sa mga opsyon
