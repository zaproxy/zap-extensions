Selenium API

API selenium позволяет задавать и просматривать пути к необходимым веб-драйверам ( WebDrivers ) и двоичный ( бинарным).

Действия/представления API

Имя: Тип Параметры Описание
optionChromeDriverPath просмотр Возвращает текущий путь к ChromeDriver
optionFirefoxBinaryPath просмотр Возвращает текущий путь к двоичному файлу Firefox
optionFirefoxDriverPath просмотр Возвращает текущий путь к драйверу Firefox (geckodriver)
optionPhantomJsBinaryPath просмотр Возвращает текущий путь к двоичному файлу PhantomJS
setOptionChromeDriverPath действие String* Устанавливает текущий путь к ChromeDriver
setOptionFirefoxBinaryPath действие String* Устанавливает текущий путь к двоичному файлу Firefox
setOptionFirefoxDriverPath действие String* Устанавливает текущий путь к драйверу Firefox (geckodriver)
setOptionPhantomJsBinaryPath действие String* Устанавливает текущий путь к двоичному файлу PhantomJS
Параметры, отмеченные звездочкой, являются обязательными.
