Pantalla de opciones de Selenium

Esta pantalla permite configurar los requisitos de alguno de los WebDrivers.

Explicación de las opciones de configuración

Opciones de configuración
Campo Detalles Por defecto
WebDrivers
ChromeDriver Permite seleccionar la ubicación de ChromeDriver. La ruta al conjunto WebDriver, si está disponible.
geckodriver Permite seleccionar la ubicación de geckodriver (driver de Firefox). La ruta al conjunto WebDriver, si está disponible.
IEDriverServer Permite seleccionar la ubicación de IEDriverServer. La ruta al conjunto WebDriver, si está disponible.
Archivos Binarios
Firefox Permite seleccionar la ubicación de los archivos binarios de Firefox (por ejemplo, utilizar una versión distinta al sistema predeterminado). (Ninguno)
PhantomJS Permite seleccionar la ubicación de los archivos binarios PhantomJS. (Ninguno)
Nota: También es posible configurar las ubicaciones antes mencionadas (archivos binarios y WebDrivers) utilizando las propiedades de sistema Java, en cuyo caso las opciones de arriba serán anuladas y en su lugar se muestran esos valores.

Se pueden encontrar mas detalles sobre el conjunto WebDrivers en la página de resumen.
