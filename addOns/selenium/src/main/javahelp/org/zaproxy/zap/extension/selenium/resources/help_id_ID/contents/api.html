Selenium API

API 'selenium' mengizinkan untuk mengatur dan melihat path ke WebDrivers dan biner yang dibutuhkan.

Tindakan/penampil API

Nama Tipe Parameter Deskripsi
optionChromeDriverPath lihat Kembalikan path saat ini ke ChromeDriver
opsi Firefox Biner Jalan lihat Kembalikan path saat ini ke biner Firefox
optionFirefoxDriverPath lihat Kembalikan path saat ini ke driver Firefox (geckodriver)
optionIeDriverPath lihat Kembalikan path saat ini ke IEDriverServer
optionPhantomJsBinaryPath lihat Kembalikan path saat ini ke biner PhantomJS
setOptionChromeDriverPath tindakan String* Tetapkan path saat ini ke ChromeDriver
setOptionFirefoxBinaryPath tindakan String* Menetapkan jalur saat ini ke biner Firefox
setOptionFirefoxDriverPath tindakan String* Menetapkan jalur saat ini untuk driver Firefox (geckodriver)
setOptionIeDriverPath tindakan String* Menetapkan jalur saat ini ke IEDriverServer
setOptionPhantomJsBinaryPath tindakan String* Menetapkan jalur saat ini ke biner PhantomJS
Parameter berbintang adalah wajib
