Selenium API

API 'selenium' mengizinkan untuk mengatur dan melihat path ke WebDrivers dan biner yang dibutuhkan.

Tindakan/penampil API

Nama Tipe Parameter Deskripsi
optionChromeDriverPath lihat Kembalikan path saat ini ke ChromeDriver
opsi Firefox Biner Jalan lihat Kembalikan path saat ini ke biner Firefox
optionFirefoxDriverPath lihat Kembalikan path saat ini ke driver Firefox (geckodriver)
optionPhantomJsBinaryPath lihat Returns the current path to PhantomJS binary
setOptionChromeDriverPath action String* Sets the current path to ChromeDriver
setOptionFirefoxBinaryPath tindakan String* Sets the current path to Firefox binary
setOptionFirefoxDriverPath tindakan String* Sets the current path to Firefox driver (geckodriver)
setOptionPhantomJsBinaryPath tindakan String* Sets the current path to PhantomJS binary
Parameter berbintang adalah wajib
