Selenium API

API 'selenium' mengizinkan untuk mengatur dan melihat path ke WebDrivers dan biner yang dibutuhkan.

Tindakan/penampil API

Nama Tipe Parameter Deskripsi
optionChromeBinaryPath lihat Returns the current path to Chrome binary
optionChromeDriverPath lihat Returns the current path to ChromeDriver
optionFirefoxBinaryPath lihat Returns the current path to Firefox binary
optionFirefoxDriverPath lihat Returns the current path to Firefox driver (geckodriver)
optionPhantomJsBinaryPath view Returns the current path to PhantomJS binary
setOptionChromeBinaryPath tindakan String* Sets the current path to Chrome binary
setOptionChromeDriverPath tindakan String* Sets the current path to ChromeDriver
setOptionFirefoxBinaryPath tindakan String* Sets the current path to Firefox binary
setOptionFirefoxDriverPath action String* Sets the current path to Firefox driver (geckodriver)
setOptionPhantomJsBinaryPath action String* Sets the current path to PhantomJS binary
Parameter berbintang adalah wajib
