Параметры Selenium экрана

Этот экран позволяет настроить требования некоторых веб-драйверов.

Параметры конфигурации

WebDrivers

Поле Детали По умолчанию Файл конфигурации
ChromeDriver Это позволяет вам выбрать расположение ChromeDriver. Путь к связанному WebDriver, если он доступен. Ключ: selenium.chromeDriver
Значение: путь в файловой системе к ChromeDriver.
geckodriver Это позволяет вам выбрать местоположение geckodriver (драйвера Firefox). Путь (path) к связанному WebDriver, если он доступен. Ключ: selenium.firefoxDriver
Значение: путь в файловой системе к geckodriver

Двоичные файлы

Поле Детали По умолчанию Файл конфигурации
Chrome This allows you to select the location of Chrome binary (for example, to use a version other than the system default). (Нет) Key: selenium.chromeBinary
Value: file system path to the Chrome binary
Firefox This allows you to select the location of Firefox binary (for example, to use a version other than the system default). (Нет) Key: selenium.firefoxBinary
Value: file system path to the Firefox binary
PhantomJS This allows you to select the location of PhantomJS binary. (None) Key: selenium.phantomJsBinary
Value: file system path to the PhantomJS binary
Примечание: Также возможно установить вышеуказанные местоположения (двоичные файлы и WebDrivers) с использованием системных свойств Java, и в этом случае указанные выше параметры будут переопределены, а вместо них показаны эти значения.

Более подробную информацию о связанных WebDrivers можно найти на странице обзора.

Расширение для браузера

Вы можете добавлять, удалять, включать или отключать расширения браузера с помощью соответствующих кнопок на этом экране. Поддерживаемые браузеры и расширения файлов:
  • Firefox: .xpi
  • Chrome: .crx
Надстройки также могут добавлять новые расширения для браузера. Если вы удалите расширение браузера через этот экран, оно будет добавлено снова дополнением (add-on), поэтому либо просто отключите ее, либо удалите надстройку, чтобы удалить расширение браузера.
