Pilihan layar Selenium

Layar ini memungkinkan anda untuk men-setup persyaratan dari beberapa Web Driver.

Pilihan konfigurasi penjelasan

Opsi konfigurasi
Bidang Rincian Default
WebDrivers
ChromeDriver Ini memungkinkan anda untuk memilih lokasi dari ChromeDriver. Jalan ke dibundel WebDriver, jika tersedia.
geckodriver Ini memungkinkan anda untuk memilih lokasi dari geckodriver (driver Firefox). Jalan ke dibundel WebDriver, jika tersedia.
IEDriverServer Ini memungkinkan anda untuk memilih lokasi dari IEDriverServer. Jalan ke dibundel WebDriver, jika tersedia.
Binari
Firefox Ini memungkinkan anda untuk memilih lokasi dari biner Firefox (contohnya, untuk menggunakan versi selain dari sistem default). (tidak ada)
PhantomJS Hal ini memungkinkan anda untuk memilih lokasi PhantomJS biner. (tidak ada)
Catatan: Ini juga memungkinkan untuk mengatur lokasi-lokasi di atas (binari dan WebDrivers) menggunakan properti sistem Java, sehingga opsi di atas akan ditimpa dan nilai tersebut ditampilkan.

Rincian selengkapnya tentang bundel WebDrivers bisa ditemukan pada halaman ikhtisar.
