Selenium

El complemento de Selenium proporciona WebDrivers, para permitir que otros complementos pueden invocar y controlar navegadores web de forma remota. También incluye HtmlUnit, un navegador web sin cabeza construido en Java.

Este complemento también agrega una opción de click derecho a los paneles ZAP que enumera URLs (como el historial y el mapa del sitio) lo que permite abrir el URL seleccionado con cualquiera de los navegadores compatibles.

Navegadores compatibles

Los siguientes navegadores web son compatibles:
Name ID Requerimientos/Notas
Chrome chrome Se sabe que funcionan las siguientes versiones: 59 y 60 (las versiones anteriores podrían funcionar también). Requiere ChromeDriver, si no está en el PATH del sistema, pude ser configurado en las opciones. Para más información sobre ChromeDriver y cómo obtenerlo, consulte el sitio web de ChromeDriver.
Firefox firefox Se sabe que funcionan las siguientes versiones: 45 (ESR), 46, 47.0.1, 54, y 55 (las versiones anteriores podrían funcionar también). Se sabe que algunas versiones no funcionan, por ejemplo. 47.0. Versiones más recientes (≥ 48) requieren de geckodriver, el cual puede ser configurado en las opciones. Para más información sobre geckodriver y cómo obtenerlo, consulte el sitio web de geckodriver (consulta la advertencia sobre el uso de geckodriver al pie de página).
HtmlUnit htmlunit Navegador incluido, no tiene ningún requerimiento.
Internet Explorer ie Se sabe que funciona la siguiente version: 11 (las versiones anteriores podrían funcionar también). Requiere IEDriverServer, si no está en el PATH del sistema, pude ser configurado en las opciones. Para más información sobre IEDriverServer y cómo obtenerlo, consulte el sitio web de IEDriverServer (consulta la advertencia al usar Internet Explorer al pie de página).
Opera opera Temporalmente sin funcionamiento.
PhantomJS phantomjs The following version is known to work: 2.1.1 (older versions might work too). Requires PhantomJS binary, if not on the system's PATH, it can be set in the options. For more information on PhantomJS and how to obtain it refer to the PhantomJS website (see footer note for caveat when using PhantomJS).
Safari safari  

Para utilizar Firefox, Chrome, Internet Explorer, Opera, PhantomJS y Safari, debe tenerlos instalados en su sistema. El ID del navegador puede ser utilizado para elegir el navegador al configurar el ZAP a través de la línea de comando, o utilizando el API de ZAP (por ejemplo, para fijar el AJAX Spider para que funcione con algún navegador en específico).

Algunos de los requisitos (por ejemplo, WebDrivers) de los navegadores pueden configurarse en la Pantalla de Opciones de Selenium.

WebDrivers incluidos

ZAP proporciona complementos con los WebDrivers, cuando se instalan estos complementos, ZAP intentará utilizarlos de forma predeterminada. Algunos sistemas operativos pueden no tener WebDriver para algunos navegadores, en esos casos ZAP informará en el panel de opciones que no se encuentra disponible un WebDriver incorporado. Los WebDrivers incorporados también pueden ser fijados (nuevamente) con el botón "Incorporados" (Por ejemplo, si otro WebDriver ya fue fijado manualmente). No todas las versiones de navegadores son compatibles con los WebDrivers incorporados, se recomienda usar las versiones más recientes de los navegadores cuando sea posible.

Note: ZAP add-ons can add additional browsers.

Firefox/geckodriver Note: There's an issue (Bug 1103196) that prevents HTTPS sites from being used in versions ≥ 48 and < 52.

PhantomJS Note: There's an issue (Issue #11342) that prevents sites at localhost, 127.0.0.1 and ::1 from being proxied through ZAP. Until a fix is available is advised to not use it in those cases. Some add-ons might choose to show warning message when that happens. As workaround one could define, in the hosts file, a domain name mapping to the local address and use that domain name instead.

Internet Explorer Note: Not all versions of Internet Explorer work out of the box, refer to the IEDriverServer website for more details on how to configure them.
