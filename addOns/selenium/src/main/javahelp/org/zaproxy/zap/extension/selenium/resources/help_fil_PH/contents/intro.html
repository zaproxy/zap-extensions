Silinyum

Ang Silinyum na add-on ay mga nagbibigay ng mga WebDriver, para sa ibang mga karagdagan, upang manawagan at makontrol ng malayo sa mga web browser. Ito rin ay nagbubuklod ng HtmlUnit ng web browser, isang headless na Java na nakabase sa web browser.

Ang add-on ay nagdagdag din ng isang pagpindot sa kanan na pagpipilian sa mga panel na listahan ng mga URL (tulad sa kasaysayan at puno ng mga site) kung saan pinapayagan ka na buksan ang piniling URL sa kahit na anong suportadong browser.

Para sa pag-famit ng Firefox, Kromo, Pagsisiyasat sa Internet, Opera, PhantomJS at Safari, ay kailngan mayroon ka nitong isinalin sa iyong sistema. Ang ID ng browser ay maaaring ginamit sa pagpili ng browser habang kinukumpigura ang ZAP sa pamamagitan ng linya na naatasan o sa pag gamit ng ZAP API (Halimbawa, sa pagtakda ng AJAX Ispayder sa pag gamit ng isa o ibang browser).

Ang ibang mga kailangan (hal. mga WebDriver) sa mga browser na maaaring makumpigura sa mga pagpipilian ng silinyum iskrin.

Magkasamang mga WebDriver

Ang ZAP ay naglaan ng mga add-on kasama ang mga WebDriver, sa pamamagitan ng mga add-on na naisaling ZAP ay maaaring magtangka na gumamit ng magkasamang mga WebDriver bilang default. Ilan sa mga OS ay maaring walang WebDriver para sa ilang mga browser, doon sa mga kaso ng ZAP ay ipapaalam, sa mga pagpipilian ng panel, ay merong walang magkasamang webDriver na magagamit. Ang magkasamang mga WebDriver ay maaari ring i- (re) set sa pindutan na 'Magkasama' (hal. kung sa ibang WebDriver ay minamanwal ang pagtakda). Hindi lahat ng mga bersyon ng mga browser ay suportado kasama ang mga webDrivers na magkasama, ito ay nirerekomenda sa mas bago/pinakabagong mga browser na gagamitin kahit saan man ay posible.

Talaan: ang mga add-on ng ZAP ay maaaring magdagdag ng mga browser.

Firefox/geckodriver na Talaan: Merong mga isyu (Bug 1103196) ay maaaring mapigilan ang mga site ng HTTPS mula sa pag gamit sa mga bersyon ≥ 48 at < 52.

PhantomJS na talaan: Merong isang isyu (Isyu #11342 ay maaaring mapigilan ang mga site at localhost, 127.0.0.1 at ::1 mula sa pagiging kinatawan sa pamamagitan ng ZAP. Hanggang sa maaari ng gamitin ang pag aayos sa inabisong hindi ito gagamitin sa ganitong mga kaso. Ang ilan sa mga add-on ay maaaring pagpilian para makita ang mga mensaheng nagbababala habang ito ay mga nangyayari. Bilang workaround ang isa ay maaaring tumukoy, sa may mga host ng payl, ang isang dominyon na pangalan ay nagma-mapping sa adres na lokal at sa halip gagamitin ang pangalan ng dominyon.

Ang talaan ng pagsisiyasat sa internet: Hindi lahat ng mga bersyon ng Pagsisiyasat sa Internet ay gumagana sa kahon, sumangguni sa IEDriverServer website para sa mga detalye kung paano makumpigura ang mga ito.
