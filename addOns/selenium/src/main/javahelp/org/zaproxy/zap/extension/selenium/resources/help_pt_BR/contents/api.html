API Selenium

A API Selenium permite definir e exibir os caminhos para os WebDrivers e binários necessários.

Ações/Visualizações de API

Nome Tipo Parâmetros Descrição
optionChromeBinaryPath visualização Returns the current path to Chrome binary
optionChromeDriverPath visualização Returns the current path to ChromeDriver
optionFirefoxBinaryPath visualização Returns the current path to Firefox binary
optionFirefoxDriverPath visualização Returns the current path to Firefox driver (geckodriver)
optionPhantomJsBinaryPath view Returns the current path to PhantomJS binary
setOptionChromeBinaryPath ação String* Sets the current path to Chrome binary
setOptionChromeDriverPath ação String* Sets the current path to ChromeDriver
setOptionFirefoxBinaryPath ação String* Sets the current path to Firefox binary
setOptionFirefoxDriverPath action String* Sets the current path to Firefox driver (geckodriver)
setOptionPhantomJsBinaryPath action String* Sets the current path to PhantomJS binary
* Parâmetros com asterisco são obrigatórios
