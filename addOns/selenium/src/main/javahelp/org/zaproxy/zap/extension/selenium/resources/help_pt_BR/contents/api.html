API Selenium

A API Selenium permite definir e exibir os caminhos para os WebDrivers e binários necessários.

Ações/Visualizações de API

Nome Tipo Parâmetros Descrição
optionChromeDriverPath visualização Retorna o caminho atual para ChromeDriver
optionFirefoxBinaryPath visualização Retorna o caminho atual para o Firefox binário
optionFirefoxDriverPath visualização Retorna o caminho atual para o Firefox driver (geckodriver)
optionIeDriverPath visualização Retorna o caminho atual para IEDriverServer
optionPhantomJsBinaryPath visualização Retorna o caminho atual para binário PhantomJS
setOptionChromeDriverPath ação String* Define o caminho atual para ChromeDriver
setOptionFirefoxBinaryPath ação String* Retorna o caminho atual para o Firefox binário
setOptionFirefoxDriverPath ação String* Retorna o caminho atual para o Firefox driver (geckodriver)
setOptionIeDriverPath ação String* Retorna o caminho atual para IEDriverServer
setOptionPhantomJsBinaryPath ação String* Retorna o caminho atual para binário PhantomJS
* Parâmetros com asterisco são obrigatórios
