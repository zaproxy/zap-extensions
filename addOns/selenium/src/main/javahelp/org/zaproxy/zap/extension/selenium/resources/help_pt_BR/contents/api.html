API Selenium

A API Selenium permite definir e exibir os caminhos para os WebDrivers e binários necessários.

Ações/Visualizações de API

Nome Tipo Parâmetros Descrição
optionChromeDriverPath visualização Retorna o caminho atual para ChromeDriver
optionFirefoxBinaryPath visualização Retorna o caminho atual para o Firefox binário
optionFirefoxDriverPath visualização Retorna o caminho atual para o Firefox driver (geckodriver)
optionPhantomJsBinaryPath visualização Returns the current path to PhantomJS binary
setOptionChromeDriverPath action String* Sets the current path to ChromeDriver
setOptionFirefoxBinaryPath ação String* Sets the current path to Firefox binary
setOptionFirefoxDriverPath ação String* Sets the current path to Firefox driver (geckodriver)
setOptionPhantomJsBinaryPath ação String* Sets the current path to PhantomJS binary
* Parâmetros com asterisco são obrigatórios
