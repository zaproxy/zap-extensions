Веб-драйверы Windows

Надстройка Windows WebDrivers предоставляет веб-драйверы для следующих браузеров:

  • Chrome - ChromeDriver 97.0.4692.71
  • Firefox - geckodriver 0.30.0

Смотрите также

     geckodriver » Поддерживаемые платформы узнать, какие версии Firefox поддерживаются.
     Страница загрузки ChromeDriver узнать, какие версии Chrome поддерживаются.
