Python Scriptleme

The Python Scripting add-on allows you to integrate Python scripts in ZAP.
It's bundled Jython 2.7.2.

Yeni bir script oluşturduğunuzda, çeşitli Python şablonlarından seçmek için seçeneğin yanı sıra Python kullanmak için seçenek belirlemiş olacaksınız.

Python Scripting is configured using the Options Jython screen.
