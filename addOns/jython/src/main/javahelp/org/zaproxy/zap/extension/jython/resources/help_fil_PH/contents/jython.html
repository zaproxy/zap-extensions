Pagscript gamit ang Python

The Python Scripting add-on allows you to integrate Python scripts in ZAP.
It's bundled Jython 2.7.2.

Kapag gagawa ka ng bagong script ikaw ay mabibigyan ng opsyon na gumamit ng Python, pati na rin ang opsyon na pumili sa iba't ibang Python na mga template.

Python Scripting is configured using the Options Jython screen.
