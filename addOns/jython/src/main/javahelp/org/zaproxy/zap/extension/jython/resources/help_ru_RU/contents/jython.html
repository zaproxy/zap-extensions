Python-скрипты

Надстройка Python Scripting позволяет интегрировать сценарии Python в ZAP.
В комплекте Jython 2.7.2.

Когда вы создаете новый скрипт, вам предоставляется возможность использовать Python, а также возможность выбора из различных шаблонов Python.

Сценарии Python настраиваются с помощью экрана параметров Jython..
