Llamadas a Casa

Este complemento se utilizará para todas las "llamadas a casa".

Actualmente admite:

Verificar Actualizaciones

La llamada "Verificar actualizaciones" se utiliza para obtener la información más reciente sobre ZAP y los complementos publicados. Es un complemento obligatorio desde ZAP 2.12.0, ya que esta función es utilizada por el núcleo de ZAP.

Novedades

La llamada de "Novedades" se utiliza para obtener la última información sobre ZAP, la cual se muestra en el panel de inicio rápido de ZAP en el escritorio.

Telemetría

La llamada de telemetría informa sobre los complementos que se están utilizando y algunas estadísticas internas de ZAP. Nunca recopilará detalles de los sitios en los que se utiliza ZAP, detalles de las vulnerabilidades o información personal identificable (PII).

Para obtener más información, consulta la publicación del blog sobre Planes de telemetría de ZAP.

Opción de línea de comandos: -notel

Si se especifica -notel en la línea de comandos, no se realizarán llamadas de telemetría. Las otras llamadas no se verán afectadas. Tampoco se realizarán llamadas de telemetría si se utiliza la opción de línea de comandos estándar -silent ZAP.

Panel de opciones de llamada a casa

Este panel te permite activar y desactivar el soporte de telemetría.

También muestra el último conjunto de datos de telemetría transmitidos. Los datos de telemetría también se registran en el archivo de registro de ZAP con el nivel DEBUG.
