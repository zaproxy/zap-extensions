Генерация отчета

This add-on allows you to generate a variety of reports in a flexible and extensible way.

Он предоставляет диалоговое окно, доступ к которому можно получить через меню «Отчет / Создать отчет..." или через панель инструментов "Создать отчет..." кнопка.

Он также поддерживает Автоматизацию Фреймворк.

Диалоговое окно создания отчета

Поля области действия

Вкладка Scope содержит следующие поля:

Название отчета

Заголовок, который будет использоваться в отчете.

Название отчета

Имя файла, которое будет использоваться для отчета. Это имя может включать следующие шаблоны (patterns):
  • {{шаблон даты/времени }} Любой Дата Java /Шаблон времени, например. гггг-мм-дд
  • [[site]] Название первого выбранного сайта
Имя будет автоматически обновлено в соответствии с выбранным сайтом и расширение для типа отчета.

Чтобы изменить используемое по умолчанию имя, измените «Шаблон имени отчета» ( 'Report Name Pattern' ).

Каталог отчетов

Каталог, в который будет записан отчет. Этот каталог будет выбран при следующем открытии диалогового окна.

Описание

Описание, которое будет включено в отчет.

Контексты

Список всех контекстов, доступных в настоящее время. Выберите один или несколько включать только оповещения и другую информацию из этих контекстов в отчет. Вы можете использовать клавиши Shift или Control для выбора и отмены выбора более одного контекста.

По умолчанию будут включены все контексты.

Сайты

Список всех сайтов, доступных в настоящее время. Выберите один или несколько, чтобы включать только оповещения и другую информацию с этих сайтов в отчет. Вы можете использовать клавиши Shift или Control для выбора и отмены выбора более одного сайта.

По умолчанию будут включены все сайты.

Если вы выбираете сайты, которые не входят в те контексты, которые у вас есть также выбран, то данные не будут включены в отчет.

Генерировать, если нет предупреждений

По умолчанию диалоговое окно не позволит вам создать отчет без оповещения. Установите этот флажок, если вы хотите создать отчет с никаких предупреждений.

Отобразить отчет

Если выбрано, то ZAP попытается открыть отчет, используя настройки по умолчанию. программа, используемая вашей операционной системой для этого типа отчета.

Поля шаблона

Вкладка «Шаблон» ( Template tab ) содержит следующие поля:

Шаблон

Выпадающий список (pulldown containing), содержащий все доступные шаблоны. Шаблон имена включают формат, в котором они будут созданы, например. HTML, MD, PDF и др.

Темы

Шаблоны могут опционально определять «темы» ('themes' ) — обычно разных цветов. и стилей, которые применяются к одному и тому же содержимому. Если какие-либо темы определены для выбранного шаблона, вы можете выбрать их здесь.

Разделы

Шаблоны могут опционально определять «разделы» ('sections')— части отчета, которые могут быть включены или исключены. Если выбранный шаблон определяет разделы то здесь будет отображаться флажок для каждого раздела. По умолчанию все флажки будут установлены. Если вы отмените выбор любого из разделов, то они не будут включены в отчет.

Поля фильтра

Вкладка «Фильтр» содержит следующие поля:

Включить риски

Включить оповещения с выбранными рисками.

Включить конфиденциальность

Включить оповещения с помощью избранные доверительные отношения.

Поля опций

Вкладка «Параметры» содержит следующие поля:

Шаблон имени отчета

Шаблон, который будет использоваться для создания имени отчета.

Каталог шаблонов

Каталог, из которого будут загружены шаблоны. Тебе не следует обычно необходимо изменить, если только вы не разрабатываете новые отчеты, в которых случае может быть полезно изменить, чтобы указать на ваш локальный исходный код каталог отчетов. Папка «отчеты» в вашем домашнем каталоге ZAP используется по умолчанию.
