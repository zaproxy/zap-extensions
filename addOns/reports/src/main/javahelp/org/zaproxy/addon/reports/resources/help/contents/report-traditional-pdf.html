Script Diagnostics Section

When the scriptdiagnostics section is enabled and the Script Console add-on has recorded runs in the session database, the report can include a Script Diagnostics section listing script diagnostic records in run order (for example failed automation runs), with created time, outcome, summary, scripts, steps, and outputs.
