Поддержка инфраструктуры автоматизации генерации отчетов

Это дополнение поддерживает Automation Framework.

Задание: отчет

Задание отчета позволяет создавать отчеты с использованием любого установленного шаблона отчетов. 
  - type: report                       # Генерация отчета 
    parameters:
      template:                        # Строка: идентификатор шаблона, по умолчанию: традиционный-html 
      reportDir:                       # Строка: каталог, в который будет записан отчет. 
      reportFile:                      # Строка: Шаблон имени файла отчета, по умолчанию: {{гггг-ММ-дд}}-ZAP-Report-[[сайт]] 
      reportTitle:                     # Строка: название отчета 
      reportDescription:               # Строка: описание отчета 
      displayReport:                   # Boolean: отображать отчет при создании, по умолчанию: false 
    risks:                             # Список: риски для включения в этот отчет, по умолчанию все 
      - high
      - medium
      - low
      - info
    confidences:                       # Список: конфиденциальные данные для включения в этот отчет, по умолчанию все 
      - high
      - medium
      - low
      - falsepositive
    sections:                          # Список: разделы шаблона для включения в этот отчет — см. соответствующий шаблон, по умолчанию все
