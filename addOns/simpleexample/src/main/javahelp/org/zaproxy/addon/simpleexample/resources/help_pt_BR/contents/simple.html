Exemplo simples

This is a simple example add-on that you can copy and use as the basis of a new add-on that actually does something useful.

Ele contém:

Menu de nível superior

Um item no menu de nível superior 'Ferramentas'.

Menu de botão direito

Um item de Menu para o botão direito do mouse, que está habilitado para exibir mensagens.

Painel de status

Um painel básico de status.

ZAP API Component

An API component that adds an action endpoint.
