## Unreleased

### Added
- Initial version of the WSTG Mapper add-on.
- Bundled WSTG 4.2 checklist and alert-to-WSTG mappings.
- Interactive compliance dashboard with automatic alert-driven coverage tracking.
- Session-persistent statuses and notes for each WSTG test.
- Markdown and CSV export for WSTG coverage reporting.
- Dynamic loading: If the JSON file with test cases is updated (e.g., new test cases are added), the add-on automatically loads and reflects these changes without requiring code modifications.
