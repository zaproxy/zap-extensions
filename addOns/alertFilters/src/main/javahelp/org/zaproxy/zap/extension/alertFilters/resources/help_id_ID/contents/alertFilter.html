Filter Peringatan Sinyal

Filter Peringatan Konteks memungkinkan Anda untuk secara otomatis mengganti tingkat risiko dari setiap peringatan yang diajukan oleh aturan pemindaian aktif dan pasif dalam konteks tertentu.

Add-on ini menambahkan panel 'Alert Filters' ke dialog konteks.
Panel menampilkan daftar semua filter Lansiran beserta tombol untuknya menambahkan, menghapus, dan menghapusnya.

Untuk menggunakan add-on ini Anda perlu:

  • Tambahkan aplikasi Anda ke suatu konteks
  • Membuka Sesi dialog Properti (misalnya dengan mengklik ganda pada konteks)
  • Pilih panel 'Alert panel' konteks
  • Klik 'Add' dan isi rincian untuk setiap peringatan yang ingin Anda timpa
Catatan: Filter peringatan hanya akan diterapkan ke lansiran baru, bukan untuk lansiran apa pun sudah hadir saat filter dibuat.

Dialog 'menambahkan' dan 'modifikasi' memiliki bidang berikut:

Lansiran

Tarik ke bawah yang berisi semua peraturan peringatan aktif dan pasif yang saat ini terpasang.

Tingkat Risiko Baru

Tingkat risiko baru yang akan diberikan pada peringatan yang diajukan sesuai dengan kriteria yang ditentukan oleh peraturan tersebut.

URL

URL opsional.
Jika ditentukan maka aturan ini akan diterapkan jika URL sesuai dengan URL peringatan yang dinaikkan.

URL adalah regex?

Jika ditetapkan dan URL ditentukan maka URL akan diperlakukan sebagai ekspresi regresi bila dibandingkan dengan URL lansiran.
Jika tidak disetel, maka URL tertentu harus sama persis dengan URL lansiran.

Parameter

Parameter opsional.
Jika ditentukan maka aturan ini akan diterapkan jika parameter sama persis dengan parameter peringatan yang dinaikkan.

Diaktifkan

Jika ditetapkan maka aturan ini akan diterapkan pada semua peringatan yang muncul terhadap konteks yang diberikan.
