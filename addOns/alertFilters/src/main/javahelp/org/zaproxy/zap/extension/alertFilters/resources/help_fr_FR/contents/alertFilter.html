Filtres d'alertes

Les filtres d’alertes permettent de modifier automatiquement les niveaux de risque des alertes générées par les analyses actives et passives.

Il existe 2 types de filtres d’alertes :

  • Filtres d’alertes du contexte
  • Filtres d'alertes globaux
Les filtres d’alertes peuvent également être définis dans le Framework d'automatisation.

La manière la plus simple de créer un filtre d’alerte consiste à faire un clic droit sur une alerte, puis à sélectionner l’option 'Créer un filtre d’alerte… '. Cela affichera la boîte de dialogue Filtre d’alerte.
Vous pouvez également créer un filtre d’alerte manuellement.

Par défaut, les filtres d’alertes ne s’appliquent qu’aux nouvelles alertes, mais vous pouvez à la fois tester et appliquer ces filtres aux alertes existantes.
