Filtros de alerta de contexto

Filtros de alerta de contexto permitem que você contorne automaticamente os níveis de risco de quaisquer alertas gerados pelas regras varreduras ativa e passiva dentro de um contexto.

Este complemento adiciona um painel de 'Filtros de Alerta' para a caixa de diálogo de contextos.
O painel mostra uma lista de todos os filtros de alerta junto com botões para adicionar, remover e excluir.

Para usar este complemento que você precisará:

  • Adicionar a seu aplicativo para um contexto
  • Abra a caixa de diálogo Propriedades da sessão (por exemplo, clicando duas vezes sobre o contexto)
  • Selecione o painel de 'Filtros de Alerta' dos contextos
  • Clique em 'Adicionar' e preencha os detalhes para cada alerta que você deseja substituir
Nota: Os filtros de alerta serão aplicados somente para novos alertas, não para qualquer alertas que já estavam presentes quando o filtro foi criado.

As caixas de diálogo 'Adicionar' e 'Modificar' têm os seguintes campos:

Alerta

Menu pull down que contém todas as regras de alerta ativas e passivas atualmente instaladas.

Novo nível de risco

O novo nível a ser atribuído a qualquer alerta gerado que corresponde aos critérios definidos pela regra.

URL

Um URL opcional.
Se especificado, então esta regra será aplicada se a URL coincide com a URL de um alerta.

URL é regex?

Se ligado, e uma URL for especificada, em seguida, a URL será tratada como uma expressão regex quando comparada com a URL do alerta.
Se não estiver ligado então qualquer URL especificada deve corresponder exatamente à URL do alerta.

Parâmetro

Um parâmetro opcional.
Se especificado, então esta regra será aplicada se o parâmetro corresponde exatamente ao parâmetro de alerta.

Habilitado

Se ligado/setado, então esta regra será aplicada a todos os alertas em um determinado contexto.
