командная строка

Надстройка Quick Start поддерживает следующие параметры командной строки:
     -quickurl Указывает URL-адрес целевого приложения, которое будет атаковано.
     -quickout Указывает файл для записи отчета. Формат отчета будет зависеть от расширения файла. Поддерживаемые расширения файлов: 
.html
, 
.json
, 
.md
, 
.xml
. Если ни одно из этих расширений не используется, формат будет XML. Если не установлено в режимах «встроенный» и демона, отчет записывается в выходной поток по умолчанию.
     -quickprogress Показывать индикаторы выполнения ascii, если запущен с флагом -cmd.

Примеры:

  • Запустите ZAP во встроенном режиме с индикаторами выполнения ascii, атакуйте целевое приложение http://example.com/ и напишите отчет в выходной поток по умолчанию: 
    cmd -quickurl http://example.com/ -quickprogress
  • Запустите ZAP с пользовательским интерфейсом, атакуйте целевое приложение http://example.com/ и сохраните отчет в файл: 
    -quickurl http://example.com/ -quickout /path/to/report.xml
