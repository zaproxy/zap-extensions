Linha de comando

Complemento para início rápido, suporta as seguintes opções de linha de comando:
     -quickurl Especifica a URL do aplicativo de destino que será atacado.
     -quickout Especifica o arquivo para gravar o relatório XML. Se não for definido em modos 'inline' e daemon, o relatório é gravado no fluxo de saída default.
     -quickprogress Mostra barras de progresso em ascii, quando se inicia com o sinalizador '-cmd'.

Exemplos:

  • Iniciar o ZAP no modo 'inline' com barras de progresso do ascii, atacar o aplicativo alvo http://example.com/ e escrever o relatório de fluxo de saída padrão: 
    -cmd -quickurl http://example.com/ -quickprogress
  • Iniciar o ZAP com UI, atacar o aplicativo alvo http://example.com/ e salvar o relatório em um arquivo escolhido pelo usuário: 
    -quickurl http://example.com/ -quickout /path/to/report.xml
