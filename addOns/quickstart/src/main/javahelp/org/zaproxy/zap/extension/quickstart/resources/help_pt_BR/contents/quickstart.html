Início Rápido

A aba de início rápido fornece uma maneira fácil de testar rapidamente um aplicativo da web.

Digite a URL do seu aplicativo de destino e pressione o botão de 'Ataque'.

O ZAP irá vasculhar o alvo e então atacá-lo com a varredura ativa.

Para um teste de mais profundidade você deve explorar seu aplicativo usando o seu navegador ou testes de regressão automatizada, fazendo proxy através do ZAP.

Launch Browser

As long as you have the Selenium add-on installed then you will see an option to launch browsers.
Select the browser you want from the pull-down and press the 'Launch Browser' button.
Browsers launched this way will be configured to proxy via ZAP and will ignore certificate errors, so you will not need to import the ZAP Root CA Certificate.
ZAP will remember the last browser you chose when you restart it.

Note that launched browsers will stop working if you change the address or port that ZAP is listening on.

Plug-n-Hack

If you have the Plug-n-Hack add-on installed then you will see a button for configuring Firefox with Plug-n-Hack.
This automatically configures Firefox to proxy via ZAP and will import and trust the ZAP Root CA.
