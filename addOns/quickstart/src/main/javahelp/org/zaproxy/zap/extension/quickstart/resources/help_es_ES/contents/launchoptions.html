Opciones de Inicio Rápido

Página de Inicio

Esto controla la página que se muestra cuando se inicia un navegador para ser proxy a través de ZAP.
Existen 3 Opciones:
  • Default ZAP Page
  • Blank Page
  • URL (especifique debajo)
La página ZAP predeterminada se lee desde una página en línea al inicio.
Si no desea que se solicite la página en línea, puede elegir una de las otras opciones.

URL

La URL de la página que desea que muestren los navegadores de ZAP.
