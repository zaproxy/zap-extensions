Línea de comando

El complemento Quick Start admite las siguientes opciones de línea de comando:
     -quickurl Especifica la URL de la aplicación destino que será atacada.
     -quickout Especifica el archivo para escribir el reporte XML. Si no está configurado en 'en línea', los modos daemon y el informe se escriben en la secuencia de salida predeterminada.
     -quickprogress Mostrar barras de progreso ascii, si se inició con el indicador -cmd.

Ejemplos:

  • Inicie ZAP en modo 'en línea' con barras de progreso ascii, ataque la aplicación de destinohttp://example.com/ y escribir el informe en la secuencia de salida predeterminada: 
    -cmd -quickurl http://example.com/ -quickprogress
  • Comience ZAP con UI, ataque la aplicación de destino http://example.com/y guarda el informe a un archivo: 
    -quickurl http://example.com/ -quickout /path/to/report.xml
