Параметры Быстрый Старт Запуск

Начальная страница

Это управляет страницей, которая отображается при запуске браузера для проксирования через ZAP.
Есть 3 варианта:
  • Страница ZAP по умолчанию
  • Пустая страница
  • URL-адрес (укажите ниже)
ZAP-страница по умолчанию считывается с онлайн-страницы при запуске.
Если вы не хотите, чтобы запрашивалась онлайн-страница, вы можете выбрать один из других вариантов.

URL-адрес

URL-адрес страницы, которую вы хотите, чтобы браузеры, запущенные из ZAP, отображали.
