Pilihan Cepat Mulai Meluncurkan

Halaman awal

Ini mengontrol halaman yang ditampilkan saat anda memulai browser untuk proxy melalui ZAP.
Ada 3 pilihan:
  • Halaman default ZAP
  • Halaman kosong
  • URL (sebutkan di bawah)
Default ZAP halaman baca dari halaman pada start up.
Jika anda tidak ingin secara online halaman yang diminta maka anda dapat memilih salah satu dari pilihan lain.

URL

URL dari halaman yang anda ingin browser yang diluncurkan dari ZAP untuk menampilkan.
