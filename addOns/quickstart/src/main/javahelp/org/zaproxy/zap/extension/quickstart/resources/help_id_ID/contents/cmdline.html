Baris Perintah

Zap mendukung pilihan garis perintah berikut ini:
     -quickurl Menentukan URL dari aplikasi target yang akan diserang.
     -quickout Menentukan berkas untuk menulis laporan XML. Jika tidak diatur didalam mode 'inline' dan daemon, laporan ditulis ke aliran keluaran default.
     -quickprogress Tampilkan bar perkembangan ascii, jika dimulai dengan bendera -cmd.

Contoh:

  • Mulai ZAP dalam modus 'inline' dengan bar perkembangan ascii, serang aplikasi sasaran http://example.com/ dan tulis laporan ke keluaran stream bawaan: 
    -cmd -quickurl http://example.com/ -quickprogress
  • Mulai ZAP dengan UI, serang aplikasi sasaran http://example.com/ dan simpan laporan ke berkas: 
    -quickurl http://example.com/ -quickout /path/to/report.xml
