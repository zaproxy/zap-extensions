Mulai Cepat

Sebuah label Mulai Cepat menyediakan kamu jalan yang mudah untuk menguji aplikasi web secepatnya.

Masukkan URL aplikasi yang kamu target dan tekan tombol 'Attack'.

ZAP akan menjebak sasaran kemudian menyerangnya dengan pemindai aktif.

Untuk percobaan yang lebih mendalam kamu harus jelajahi aplikasi kamu menggunakan browser atau uji pemulihan otomatis saat proxy melalui ZAP.

Luncurkan Peramban

Selama kamu memasang add-on Selenium maka kamu akan melihat pilihan untuk meluncurkan browser.
Pilihlah browser yang kamu inginkan dari pull-down dan tekan tombol 'Launch Browser'.
Browser yang diluncurkan dengan cara ini akan disusun untuk proxy melalui ZAP dan akan mengabaikan kesalahan sertifikat, jadi kamu tidak perlu mengimpor Sertifikat CA Root ZAP.
ZAP akan mengingatkan browser terakhir yang kamu pilih saat kamu memulainya kembali.

Perhatikan bahwa browser yang diluncurkan akan berhenti bekerja jika Anda mengubah alamat atau tindakan yang sedang didengarkan ZAP.

Plug-n-Hack

Jika kamu memasang Plug-n-Hack add-on kemudian kamu akan melihat tombol untuk mengkonfigurasi Firefox dengan Plug-n-Hack.
Ini secara otomatis mengkonfigurasi Firefox ke proxy melalui ZAP dan akan mengimpor dan mempercayai ZAP Root CA.
