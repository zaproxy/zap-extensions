Complemento HTTPS Info

El complemento HTTPS Info es accedido mediante el menú de contexto dentro del Árbol de Sitios o en la tabla de Historial. Muestra un cuadro de diálogo en el cual se muestra información variada resumida sobre el certificado HTTPS del servidor destino, y el conjunto SSL/TLS cipher proporcionado.

Este complemento utiliza otro proyecto OWASP: Deep Violet para realizar el certificado y la recopilación de información del conjunto cipher.

General

La parte superior del cuadro de diálogo está dedicada a los detalles generales del certificado SSL/TLS presentado por el servidor. Como:
  • Subject DN
  • Signing Algorithm
  • Certificate Fingerprint
  • Issuer DN
  • Validity Dates
  • Self-signed Status
  • etc.

Conjuntos de Cipher

La parte inferior del cuadro de diálogo está dedicada a la enumeración o la creación de una lista de los conjuntos Cipher específicos que ofrece el servidor destino, agrupado por el protocolo de enlace (SSLv2, SSLv3, TLSv1, TLSv1.1 etc.).
