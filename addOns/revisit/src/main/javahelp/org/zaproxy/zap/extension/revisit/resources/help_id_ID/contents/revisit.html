Kembali

Pernahkah anda menginginkan untuk bisa menelusuri sebuah aplikasi anda tes dengan ZAP pada sebuah waktu tertentu, mungkin ketika anda tidak memiliki akses ke sana, atau setelah dirubah dalam beberapa cara?
Jika demikian maka add-on ini adalah untuk anda!

Add-on mengizinkan anda untuk memberi instruksi ZAP untuk mengembalikan konten dari ZAP antara riwayat tertentu tanggal daripada meneruskan permintaan ke situs.
Hal ini mengizinkan anda untuk melihat browser dan dan menggunakan sebuah situs web anda yang tidak lagi memiliki akses ke.
Hal ini juga mengizinkan anda untuk melihat apapun yang tampak seperti di antara waktu-waktu tertentu, yang berguna jika aplikasi telah berubah sejak.

Ketika mengembalikan halaman dari ZAP riwayat pengembangan footer yang disuntikkan ke dalam halaman HTML untuk memberikan sebuah petunjuk visual yang halaman ini dihasilkan dari ZAP.
ZAP akan kembali ke halaman-halaman yang merupakan hasil dari investasi GET dan permintaan POST asalkan menemukan sesuatu yang sesuai.
Halaman ini hanya cocok jika semua URL dan parameter POST yang sama, mengabaikan perintah dan apa saja yang dikenal CSRF token.
Jika tidak ada halaman yang cocok bisa ditemukan dalam riwayat kemudian kustom 404 halaman yang akan dikembalikan memasukkan link ke halaman yang sama jika ia telah dipanggil dengan parameter yang berbeda atau di luar periode waktu yang ditentukan.

Add-on ini ditambahkan klik kanan menu untuk Situs pohon:

Aktifkan kembali

Ketika memilih sebuah popup akan muncul yang meminta untuk mulai dan akhir untuk digunakan.
Waktu mulai default saat pertama permintaan di sesi ini.
Akhir waktu default untuk saat permintaan terakhir dalam sesi ini.
Jika salah satu elemen waktu (misalnya 'bulan') tidak berubah maka elemen tersebut akan dinonaktifkan.
Jika dialog tombol OK ditekan maka kembali fungsi akan diaktifkan untuk situs yang dipilih, dan ikon 'kalender' ditambahkan ke situs node dalam Sejarah pohon.

Nonaktifkan Kembali

Ketika dipilih maka kembali fungsi akan dinonaktifkan untuk situs yang dipilih.

API

Menambahkan pada mendukung API titik akhir berikut:

kembali / melihat / kembali lis maka daftar situs dapat diaktifkan
mengunjungi kembali / tindakan / mengunjungikembalisitusOn Berbalik mengunjungi kembali pada untuk situs tertentu. Tanggal harus menggunakan format: "yyyy/MM/dd HH:mm:ss"
kembali / tindakan / kunjungi kembali Situs Off Ternyata kembali off untuk situs tertentu
