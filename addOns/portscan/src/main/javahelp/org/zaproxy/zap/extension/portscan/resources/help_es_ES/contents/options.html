Pantalla de opciones de escaneo de puerto

Esta pantalla permite configurar las opciones de escaneo de puerto:

El mayor número de puerto para escanear

ZAP comenzará desde 1 y el trabajará hasta este número de puerto.
Seleccionar un número alto incrementará significativamente el tiempo de escaneo de puerto.

Subproceso de escaneos concurrentes por host

El número de subprocesos que utilizará el escáner por host.
Aumentar el número de subprocesos acelerará el análisis pero puede poner presión extra en la computadora donde ZAP está funcionando y en el host destino.
