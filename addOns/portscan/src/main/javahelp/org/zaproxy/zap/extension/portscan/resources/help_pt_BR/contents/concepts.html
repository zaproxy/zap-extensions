Scanner de Portas

Proporciona um escaneador de portas básico, que mostra quais as portas abertas nos sites de destino.

O escaneamento de portas é configurado usando-se a Tela de opções de varredura de portas.


Item de menu em pop up

Host para a Varredura de Portas

Isto irá iniciar uma varredura de portas do host para o nó selecionado.
A Aba de varredura de portas vai aparecer e irá mostrar o progresso da varredura.

Acessado via

     Aba de varredura de portas
     Aba de Sites Item de menu de clique direito ' Ataque/Host de Varredura de portas'
