Tela de opções de Varredura de Portas

This screen allows you to configure the port scan options:

Highest port number to scan

ZAP will start from 1 and work up to this port number.
Selecionar um número elevado aumentará significativamente o tempo que leva de uma varredura de portas.

Fluxos de varredura simultâneos por host

O número de fluxos (threads) que o scanner irá usar por host. Aumentar o número de fluxos (threads) vai acelerar o escaneamento mas pode sobrecarregar tanto o computador onde o ZAP está rodando quando o host alvo.
