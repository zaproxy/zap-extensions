Ang-port Scan

Ay nagbibigay ng isang pangunahing port scanner na nagpapakita kung aling mga port ay bukas sa mga site ng target.

Na-configure ang pag-scan ng port gamit ang Mga Pagpipilian sa Port Scan screen .


Pop up ang mga item sa menu

Port Scan host

Ito ay magsisimula ng port scan ng host para sa napiling node.
Ang Port Scan tab magiging display at ipapakita ang progreso ng mga scan.

Na-access sa pamamagitan ng

     Port Scan tab
     Mga site na tab I-click ang menu item ng 'Pag-atake / Port Scan host'
