Вкладка «Сканирование портов»

Вкладка «Сканирование портов» позволяет выполнить базовое сканирование портов на любом из сайтов, на которых был получен доступ.

Сайты можно выбрать через панель инструментов или вкладку Сайты.
Любые сайты, которые были сканированы в настоящее время, помечены жирным шрифтом на панели инструментов.
Панель инструментов содержит набор кнопок, позволяющих запускать, останавливать, приостанавливать и возобновить сканирование.
Индикатор выполнения показывает, как далеко продвинулось сканирование выбранного сайта.
Значение «Текущие сканы» показывает, сколько сканирований в данный момент активно — зависание над этим значением во всплывающем окне будет отображаться список сканируемых сайтов.

Меню правой кнопки мыши

Щелкнув правой кнопкой мыши по узлу, вы откроете меню, которое позволит вам:

Копировать

Это скопирует данные выбранного порта в буфер обмена.
