Mga Pagpipilian sa Port Scan screen

Screen na ito ay nagpapahintulot sa iyo na magkumpigura ng mga opsiyon para sa mga port iskan:

pinakamataas na numero ng port na i-scan

ZAP ay magsisimula mula 1 at gawain sa numero ng port na ito.
Pagpili ng isang mataas na bilang ay makabuluhang dagdagan ang oras na kailangan ng port magsuri.

Kasabay ng pag-scan thread kada host

Ang bilang ng mga thread na ang scanner ay gagamit ng bawat host.
Ang pagtaas ng bilang ng mga thread ay bumibilis ang scan ngunit maaaring maglagay ng ekstrang pinagmanahan sa computer tumatakbo sa ZAP at hukbo ng target.
