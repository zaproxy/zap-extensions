Layar Pilihan Pindai Port

Layar ini memungkinkan anda untuk mengkonfigurasi pilihan pindai port:

Highest port number to scan

ZAP akan mulai dari 1 dan bekerja sampai ke nomor port ini.
Memilih nomor yang tinggi akan secara signifikan meningkatkan waktu pengambilan pemindaian port.

Bersamaan memindai thread per host

Jumlah thread yang pemindai akan gunakan per host.
Tingkatkan jumlah thread yang akan mempercepat pemindaian tetapi akan memberikan tegangan ekstra pada komputer ZAP yang sedang berjalan dan pada host sasaran.
