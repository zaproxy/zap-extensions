Port Scan

Menyediakan pemindai port dasar yang menampilkan port mana yang terbuka pada situs target.

Pemindaian port dikonfigurasi menggunakan Option Port Scan screen.


Item menu pop up

Port Scan host

Ini akan menginisiasikan pemindai port dari host untuk node yang terpilih.
The Port Scan tab akan menampilkan dan akan menunjukkan kemajuan dari pemindaian.

Diakses melaui

     Port Scan tab
     Tab situs 'Attack/Port Scan host' klik kanan menu item
