Сканирование портов

Предоставляет базовый сканер портов, который показывает, какие порты открыты на целевых сайтах.

Сканирование портов настраивается с помощью Параметры сканирования портов. экран.


Пункт всплывающего меню

Хост сканирования портов

Это инициирует сканирование портов хоста для выбранного узла.
The Вкладка «Сканирование портов» будет отображаться и будет показывать ход выполнения сканирование.

Доступ через

     Вкладка «Сканирование портов»
     Вкладка "Сайты" Пункт контекстного меню «Атака/сканирование портов»
