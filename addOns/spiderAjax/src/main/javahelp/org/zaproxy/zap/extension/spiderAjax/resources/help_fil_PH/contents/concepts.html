AJAX Spider

Ang AJAX Spider add-on ay sumasama sa ZAP bilang krawler ng AJAX at ang tawag sa site nato ay crawljax. Pwede mong gamitin ito para malaman ang mga pahina ng targeted site. Pwede mong isama ito sa (normal) spider para masmapagandang resolta.

Ang spider ay maisasaayos gamit ang Options AJAX Spider iskren.

This add-on supports the Automation Framework.

Sapamamagitan ng pag akses

     AJAX Spider tab
     Tab ng kasaysayan 'Attack/AJAX Spider...' right click menu item
     Mga site tab 'Attack/AJAX Spider...' right click menu item

Tignan din

     Mga pagpipilian sa AJAX spider iskren para sa kabuoan ng pag pipilian sa AJAX spider
     AJAX Spider tab para sa kabuoan ng AJAX Spider Tab
     AJAX Spider diyalogo para sa kabuoang diyalogo ng AJAX Spider
