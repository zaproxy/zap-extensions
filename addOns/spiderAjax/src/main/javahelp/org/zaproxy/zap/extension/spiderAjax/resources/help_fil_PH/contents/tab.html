AJAX Spider tab

Ang AJAX Spider tab ay nagpapakita sayo ng kakaibang URls natagpuan ng AJAX Spider.


Para sa bawat hiling na maaari mong makikita:
     Ang hiling ng index-bawat hiling ay may numero, nagsisimula sa 1
     Ang hiling ng timestamp
     Ang HTML na pamamaraan, e.g. GET o POST
     Ang kahilingan ng URL
     Ang tugon ng estado code sa HTTP
     Ang maikling buod ng kung ano ang ibig sabihin ng HTTP response code
     Ang haba ng oras na kinuha ng buong request
     Ang sukat sa tugon ng header
     Ang sukat sa tugon ng body
     AnumangAlerts sa kahilingan
     Anuman Notes ay kaylangan idagdag sakahilingan
     Anuman Tags ay kaylangan idagdag sa kahiligan
Ang mga napiling kahilingan ay ipapakita sa Request tab at Response tab above.

Right klick menu

Ang pag right klick ng node ay magdadala sayo papunta sa menu kung saan papahintulutan ka nito:

Atake

Ang atake menu ay sumusunod sa mga submenu:

Ang aktibo eskan site

Ito ay pasimula sa Aktibong iskan ang lahat ng site ay naglalaman ng napiling node.
Ang Aktibong iskan tab ay magiging display at ipapakita ang pag unlad ng iskan.

Aktibong Iskan Node

Ito ay pasimula ng isang Aktibong iskan na laman ng node na napili.
Ang Aktibong Iskan tab ay magiging display at ipapakita ang pag unlad ng iskan.

Spider Site

Ito ay pasimula ng spider ang buong site ay naglalaman ng napiling node.
Ang Spider tab ay magiging display at ipapakita ang pag unlad ng iskan.

Forced Browse Site

Ito ay pasimula ng forced browse ang kabuoan ng site ay naglalaman ng napiling node.
Ang Forced Browse tab ay magiging display at ipapakita ang pag unlad ng iskan.

Forced Browse Directory

Ito at pasimula ng forced browse na napili ng derektoryo.
Ang Forced Browse tab ay dinisplay at nagpapakita sa pag unlad ng iskan.

Forced Browse Directory (at mga bata)

Ito ay pasimula ng forced browse sa napiling derektoryo at ang lahat ng mga bata ay natagpuan.
Ang Forced Browse tab ito ay dinisplay at pinagpakita ang pag unlad ng iskan.

Ibukod mula

Itong menu ay sumosunod sa mga submenu:

Proxy

Ito ay magbubukod ng mga napiling mga nodes galing sa proxy. Ito ay mananatiling proxied galing sa ZAP ngunit hindi ipinakita sa kahit anong tabs.
Ito ay pweding gamitin upang hindi mapansin ang mga URL na alam mong walang kaugnayan sa kasalukuyan mong sinusuri.
Ang nodes ay magiging kasama ulit sa pamamagitan ng Mga katangian ng sesyon diyalogo

Iskaner

Ito ang nagpipigil sa pagpili ng mga node upang maging aktibong iskan.
Ang nodes ay maaaring maisama muli sa pamamagitan ng Mga katangian ng sesyon diyalogo

Spider

Ito ay pumipigil sa napiling nodes mula sa pagiging spidered.
Ang mga nodes ay maaaring maisama ulit sa pamamagitan ng Mga katangian ng sesyon diyalogo

Pagpapatakbo ng aplikasyon

Ang talaan ay nagbibigay pahintulot sayo na tumawag sa aplikasyon na maaari mong isayos sa pamamagitan ng Pagpili sa aplikasyon ng iskren na maaari din mapuntahan sa pamamagitan ng pagkumpigura ng mga aplikasyon... 'submenu.

Namamahala ng mga Tags...

Dadalhin nito paitaas ang Namamahala sa diyalogo ng mga tag na nagbibigay sayo ng pahintuloy upang palitan ang mga tag na nauugnay sa kahilingan.

Tandaan...

Dadalhin nito ang Nadagdag na diyalogo sa talaan na magbibigay ng pahintulot sayo na itala ang mga paalala na may kaugnayan sa kahilingan.

Break...

Dadalhin nito paitaas ang Nadagdag na dayalogo sa Break Point na magpapahintulot sa iyo na magtakda ng break point sa URL.

Alerto sa mga node na ito

Kung ang pinili ng URL ay may mga alerto kaugnay nito at magkakaroon sila ng listahan sa ilalim ng menu na ito.
Pinipili ang isa sa mga alerto na magiging dahilan upang maipakita ito.

Ipadala Muli...

Dadalhin nito paitaas ang Diyalogong pinadala muli na nagpapahitulot sa iyo upang muling ipadala ang kahilingan pagkatapos gumawa ng anumang mga pagbabago sa mga ito na gusto mo.

Bagong Alerto...

Dadalin nito paitaas ang Nadagdag na diyalogo aa alert na nagpapahintulot sayo na manumanung i-rekord ang isang bagong Alert laban sa hiling na ito.

Buksan ang URL sa Browser

Bubuksan nito ang URL sa napiling node sa iyong default browser.
