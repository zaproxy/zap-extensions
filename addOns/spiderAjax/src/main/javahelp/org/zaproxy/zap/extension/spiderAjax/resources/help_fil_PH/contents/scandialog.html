Diyalogo ng AJAX Spider

Itong diyalogo ay naglulunsad ng AJAX Spider.

Saklaw

Ang unang tab na ito ay nagpapahintulot sa iyo upang baguhin ang mga pangunahin tampok tulad ng:

Starting Point

Ang URL na kung saan ang AJAX spider ay magsisimula na gumapang mula, o ang isang konteksto (sa kung saan ang caso ay ginagamit ang isa sa URL na nasa konteksto bilang panimulang punto).

Konteksto

Nagpapahintulot upang piliin ang konteksto para maging spidered.

Gumagamit

Nagpapahintulot upang pumili ng isa sa mga gumagamit na bakante mula sa napiling konteksto, para magsagawa sa spider scan bilang gumagamit (ZAP will (re) ay nagpapatunay bilang isang gumagamit kung kinakailangan).

Just In Scope

Kung itakda pagkatapos ng anumang URL kung saan ang nasa labas ng saklaw ay hindi papansinin.

Tandaan: Ang pagpipilian Just In Scope ay pare-pareho eksklusibo sa Konteksto sa pagpipilian, kung ang isa ay nagamit ang iba ay hindi papansinin.

Spider Subtree lamang

Kung itakda at pagkatapos ang spider ay papasok lamang sa mapagkunan na nasa ilalim ng panimulang punto (URI). Kapag sinusuri kung ang mapagkunan ay matagpuan sa loob ng tinukoy na subtree, ang spider ay itinuturing lamang ang pamamaraan, host, port, at path na mga bahagi sa URI.

Browser

Ang uri ng browser para gamitin.
Ang mga browser lang ang ipapakita kung ang ZAP ay sapat sa pagsasaayos ng takbo sa kanila.
Maaari ito ay huminto sa pagtakbo kung ang mga browser ay hindi matagpuan o kung ang pagsasaayos ng impormasyon ay hindi tama.

Show Advanced Options

Kung napili ito ang mga pagpipilian tab ay dapat ipakita.

Mga opsyon

Ang tab na ito ay napapahitulot sa iyo na baguhin ang mga kasama sa pagpipilian:

Number of Browser Windows to Open

Pwede mong ma-configure ang bilang ng mga window na gagamitin sa AJAX Spider.
Masmaraming window, masmagiging mabilis ang proseso.

Maximum Crawl Depth

Ang pinakamataas na lalim ng crawler na pweding maabot. Zero ibig sabihin ay walang hanggan ang lalim.

Maximum Crawl States

Ang pinakamataas na bilang ng estado sa crawler ay crawl. Zero ibig sabihin ay walang katapusan ang estado ng crawl.

Maximum Duration

Ang pinakamahabang oras na pinahintulutan sa pagtakbo ng crawler. Zero ibig sabihin ay walang hanggan na pagtakbo ng oras.

Event Wait Time

Matapos maghintay ang panig ng kliyente ang kaganapan ay inalis.

Reload Wait Time

Ang paghihintay ng oras matapos maloadan ang URL.

Allowed Resources

The list of allowed resources. The allowed resources are always fetched even if out of scope, allowing to include necessary resources (e.g. scripts) from 3rd-parties.
