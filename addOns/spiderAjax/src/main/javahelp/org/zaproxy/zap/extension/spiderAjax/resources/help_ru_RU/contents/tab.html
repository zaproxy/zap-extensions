Вкладка «Паук AJAX»

На вкладке AJAX Spider отображается набор уникальных URI, найденных AJAX Паук .


Для каждого запроса вы можете увидеть:
     Индекс запроса - каждый запрос нумеруется, начиная с 1
     Отметка времени запроса
     Метод HTML, например. ПОЛУЧИТЬ ( GET ) или ОТПРАВИТЬ (POST)
     Запрошенный URL
     Код состояния ответа HTTP
     Краткое описание того, что означает код ответа HTTP
     Время выполнения всего запроса
     Размер заголовка ответа
     Размер тела ответа
     Любые оповещения по запросу
     Любые примечания, которые вы добавили к запросу
     Любые теги, которые вы добавили в запрос
Выбор запроса отобразит его в Вкладка «Запрос» и Вкладка "Ответ" выше

Меню правой кнопки мыши

Щелкнув правой кнопкой мыши по узлу, вы откроете меню, которое позволит вам:

Атака

В меню «Атака» есть следующие подменю:

Активное сканирование сайта

Это инициирует Активное Сканирование всего сайта, содержащего выбранный узел.
Вкладка "Активное сканирование" будет отображаться и показывать ход сканирования.

Узел активного сканирования

Это инициирует Активное Сканирование только выбранного узла.
Вкладка «Активное сканирование» будет отображаться и показывать ход сканирования.

Сайт паука

Это инициирует паука всего сайта, содержащего выбранный узел.
Вкладка паук будет отображаться и показывать ход сканирования.

Принудительный просмотр сайта

Это инициирует принудительный просмотр всего сайта, содержащего выбранный узел.
Вкладка «Принудительный просмотр» будет отображаться и показывать ход сканирования.

Принудительный просмотр каталога

Это инициирует принудительный просмотр в выбранном каталоге.
Вкладка «Принудительный просмотр» будет отображаться и показывать ход сканирования.

Принудительный просмотр каталога (и дочерние элементы)

Это инициирует принудительный просмотр в выбранном каталоге и всех найденных дочерних элементах.
Вкладка «Принудительный просмотр» будет отображаться и показывать ход сканирования.

Исключить из

Это меню имеет следующие подменю:

Прокси-сервер

Это исключит выбранные узлы из прокси. Они по-прежнему будут проксироваться через ZAP но не будет отображаться ни на одной из вкладок.
Это можно использовать для игнорирования URL-адресов, которые, как вы знаете, не имеют отношения к вашей системе. в данный момент проходят испытания.
Узлы могут быть снова включены через Свойства сессии диалог

Сканер

Это предотвратит активное сканирование выбранных узлов.
Узлы могут быть снова включены через Свойства сессии dialog

Паук

Это предотвратит сканирование выбранных узлов.
Узлы могут быть снова включены через Свойства сессии диалог

Запустить приложение

Это меню позволяет вам вызывать приложения который вы настроили через Параметры Экран приложений который также доступен через «Настроить приложения...».

Управление тегами...

Это вызовет Диалоговое окно «Управление тегами» который позволяет изменить теги связанные с запросом.

Примечание...

Это вызовет Диалоговое окно добавления заметки который позволяет записывать заметки связанных с запросом.

Прерывание...

Это вызовет Диалоговое окно «Добавить точку останова» который позволяет вам установить точку останова на этом URL-адресе.

Оповещения для этого узла

Если выбранный URL имеет оповещения связанные с ним, они будут перечислены в этом меню.
Выбор одного из предупреждений приведет к его отображению.

Переслать...

Это вызовет Повторно отправить диалог который позволяет повторно отправить запрос после внесения каких-либо изменений в это то, что вы хотите.

Новое оповещение...

Это вызовет Диалоговое окно «Добавить предупреждение» который позволяет вручную записывать новый Оповещение против этого запроса.

Открыть URL в браузере

Это откроет URL-адрес выбранного узла в браузере по умолчанию.
