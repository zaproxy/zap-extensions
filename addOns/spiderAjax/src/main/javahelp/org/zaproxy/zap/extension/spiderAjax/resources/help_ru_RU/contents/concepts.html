AJAX-паук

Надстройка AJAX Spider интегрирует в ZAP сканер сайтов с поддержкой AJAX, который называется Crawljax. Вы можете использовать его для идентификации страниц целевого сайта. Вы можете комбинировать это с (обычным) пауком для лучших результатов.

Паук настраивается с помощью экрана Параметры AJAX Spider. .

Это дополнение поддерживает Automation Framework. .

Доступ через

     Вкладка «Паук AJAX»
     Вкладка История 'Атака/AJAX Spider...' пункт контекстного меню
     Вкладка Сайты 'Attack/AJAX Spider...' right click menu item
