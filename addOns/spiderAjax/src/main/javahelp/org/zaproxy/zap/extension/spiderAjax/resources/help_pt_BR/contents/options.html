Tela de opções do AJAX Spider

Esta tela permite que você configure opções do AJAX Spider . O Spider AJAX é um complemento para um rastreador chamado Crawljax. O complemento configura um proxy local no ZAP falar com o Crawljax. O spider AJAX permite a você percorrer aplicações web escritas em AJAX em muito mais profundidade do que o spider nativo. Use o spider AJAX se você estiver testando aplicativos web escritos em AJAX. Você também deve usar o spider nativo para uma cobertura completa de um aplicativo da web (por exemplo, para abranger comentários do HTML).

Explicação das opções de configuração



Opções de configuração
Campo Detalhes Padrão
Navegador O spider AJAX se baseia em um navegador externo para rastrear o site alvo. Você pode especificar qual dos dois você deseja usar. Para mais detalhes sobre navegadores suportados consulte páginas de ajuda da extensão "Selenium". Firefox
Número de janelas do navegador para abrir Você pode configurar o número de janelas para ser usado pelo spider AJAX. Quanto mais janelas, mais rápido o processo será. 1
Profundidade máxima de rastreamento A profundidade máxima que o rastreador pode chegar. Zero significa profundidade ilimitada. 10
Máximo de estados de navegação O número máximo de Estados que o rastreador deve percorrer. Zero significa ilimitado. 0 (ilimitado)
Duração máxima O tempo máximo que o rastreador pode ser executado. Zero significa ilimitado tempo de execução. 60 minutos
Tempo de espera do evento O tempo de espera após um evento de lado do cliente é disparado. 1000 ms
Tempo de espera de recarga O tempo de espera depois que URL é carregada. 1000 ms
Clicar em elementos uma vez Quando habilitado, o rastreador tenta interagir com cada elemento (por exemplo, clicando) apenas uma vez. Se isso não for definido, o rastreador tentará clicar várias vezes. Desativar esta opção representa mais rigor, mas a varredura pode levar consideravelmente mais tempo. Verdadeiro
Usar valores aleatórios em campos de formulário Quando habilitado, insere valores aleatórios para campos de formulário. Caso contrário, ele deixa o valor vazio. Verdadeiro
Clicar somente em elementos padrão Quando habilitado, só os elementos "a", "botão" e "entrada" serão clicados durante a varredura. Caso contrário, ele usa a tabela abaixo para determinar quais elementos irão ser clicados. Para obter análise aprofundada, desative isso e configure os elementos clicáveis na tabela. Verdadeiro
Selecionar elementos a clicar durante o rastreamento (tabela) A lista de elementos a percorrer. Esta tabela aplica-se somente quando "clicar somente em elementos padrão" não está habilitado. Use "habilitar tudo" para análise mais aprofundada, que pode demorar um pouco mais. Tudo habilitado
