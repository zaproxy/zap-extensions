Mga pagpipilian sa AJAX Spider iskren

Ang iskren na ito ay nagpapahintulot sayo para maisaayos ang AJAX Spider mga pagpipilian. Ang AJAX Spider ay isang add-on para sa crawler ang tawag dito ay crawljax. Ang add-on sets up ay isang lokal proxy sa ZAP para makapagsalita sa crawljax. Ang AJAX Spider ay nag papahintulot sa iyo na isulat ang crawl web aplikasyon sa AJAX para masmapalalim pa kesa sa native spider. Gamitin ang AJAX Spider Kung meron kang mga web aplikasyon isulat sa AJAX. Maaari mo lamang gamitin Ang native spider para makumpleto ang coverage ng aplikasyon web (e.g., to cover HTML comments).

Field Mga detalye Default
Browser Ang AJAX spider ay umaasa sa panglabas na browser para ma krawl ang targeted site. Pwede mong matukoy ang anumang gusto mung gamiten. para sa ibapang detalye sa mga supported browser pumunta sa "Selenium" add-on pangtulong na pahina. Firefox
Bilang ng mga windows sa browser upang buksan Maaari mong I-configure ang bilang ng mga window para magamit sa AJAX Spider. Pag masmaraming windows, masmabilis ang magiging proseso. 1
Pinakamataas na lalim ng crawl Ang pinakamataas na lalim ng crawler ay hindi maabot. Zero ibig sabihin walang hangan ang lalim nito. 10
Ang pinakamataas na estado ng crawl Ang pinakamataas na estado ng crawler ay crawl. Zero ibigsabihin walang kataposan ang estado ng crawl. 0 (walangkatapusan)
Pinakamataas na durasyon Ang pinakamataas na oras ng crawler ay pinapahintulutan tumakbo. Zero ibigsabihin walanghangan ang pagtakbo ng oras. 60 minuto
Paghihintay ng oras sa kaganapan Matapos maghintay ng oras ang panig ng kliyente Ang kaganapan ay inalis na. 1000 ms
Naghihintay ng oras sa Reload Naghihintay ng oras matapos maloadan ang URL. 1000 ms
I-klick ang mga elemento minsan Kapag pinagana, ang crawler susubukan nitong makipag-ugnayan sa bawat elemento (e.g., by clicking) isang beses lamang. Kung hindi ito naiset, ang crawler ay susubukan ito maikliek ng maraming beses. Unsetting itong opsiyon ay magiging masmahigpit ngunit magiging mas matagal. True
Gumamit ng mga random na halaga sa mga form field Kung pinagana, isinget ang halaga ng random sa mga form ng field. Kung hindi, ito ay gagamit ng walang laman na halaga. True
Pindutin lamang ang mga default na elemento Kapag pinagana, lamang ang mga elemento "a", "button" at "input" pweding mag-klik sa crawl. Kung Sana, ang mga ito ay gamit sa ilalim ng table para matukoy Kung anung mga elemento ang gagamitin para mapindot. Para sa mas malalim na pagsusuri, huwag paganahin ito at isaayos ang mga napipindot na elemento sa table. True
Pumili ng mga elemento na pipindutin sa crawl (table) Ang listahan ng mga elemento sa crawl. Itong table lamang ang nag aaplay sa "napindot na mga elementong na default lamang" ito ay Hindi pinagana. Gamitin "ang lahat ng hindi pinagana" para sa higit pang pag papalalim ng pagsusiri, bagaman maaari itong bumagal ng matagal. Lahat ng pinagana
