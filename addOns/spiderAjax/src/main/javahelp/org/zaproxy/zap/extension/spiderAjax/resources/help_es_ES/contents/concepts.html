AJAX Spider

El complemento AJAX Spider integra en ZAP un rastreador de los sitios AJAX rich llamado Crawljax. Se puede utilizar para identificar las páginas del sitio destino. Se puede combinar con el rastreador normal para mejores resultados.

El rastreador se configura utilizando la pantalla Opciones de AJAX Spider.

This add-on supports the Automation Framework.

Acceso vía

     Pestaña AJAX Spider
     History tab 'Attack/AJAX Spider...' right click menu item
     Sitios tab 'Attack/AJAX Spider...' right click menu item
