Spider AJAX

O complemento Spider AJAX integra-se ao ZAP um rastreador/crawler de sites AJAX chamado Crawljax. Você pode usá-lo para identificar as páginas do site alvo. Você pode combiná-lo com o spider normal para melhores resultados.

O spider é configurado usando-se a tela de opções Spider AJAX.

This add-on supports the Automation Framework.
