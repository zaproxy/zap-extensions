Поддержка инфраструктуры автоматизации Ajax Spider

Это дополнение поддерживает Automation Framework.

задание: SpiderAjax

Задание SpiderAjax позволяет вам запускать Ajax Spider — он медленнее, чем традиционный паук, но хорошо справляется с современными веб-приложениями. 
  - type: spiderAjax                   # Паук ajax — медленнее, чем паук, но хорошо справляется с современными приложениями. 
    parameters:
      context:                         # String: Имя контекста для паука, по умолчанию: первый контекст 
      url:                             # String: URL-адрес для начала сканирования, по умолчанию: URL-адрес первого контекста. 
      maxDuration:                     # Int: Максимальное время в минутах, в течение которого пауку ajax будет разрешено работать, по умолчанию: 0 неограниченно 
      maxCrawlDepth:                   # Int: Максимальная глубина, которую может достичь краулер, по умолчанию: 10, 0 не ограничено 
      numberOfBrowsers:                # Int: Количество браузеров, которые будет использовать паук, больше будет быстрее, но будет использовать больше памяти, по умолчанию: 1 
      browserId:                       # String: Используемый идентификатор браузера, по умолчанию: firefox-headless 
      clickDefaultElems:               # Bool: Когда включено, щелкните только элемент по умолчанию: «a», «кнопка» и введите, по умолчанию: true 
      clickElemsOnce:                  # Bool: Когда включено, щелкните каждый элемент только один раз, по умолчанию: true 
      eventWait:                       # Int: Время ожидания в миллисекундах после запуска события на стороне клиента, по умолчанию: 1000. 
      maxCrawlStates:                  # Int: Максимальное количество обходов указывает, что сканер должен сканировать, по умолчанию: 0 неограниченно 
      randomInputs:                    # Bool: Когда включено, в элемент ввода будут вводиться случайные значения, по умолчанию: true 
      reloadWait:                      # Int: Время ожидания в миллисекундах после загрузки URL-адреса, по умолчанию: 1000. 
    tests:
      - name: 'At least 100 URLs found'      # String: Название теста, по умолчанию: статистика + оператор + значение 
        type: 'stats'                        # String: Тип теста, пока поддерживается только "статистика" 
        statistic: 'spiderAjax.urls.added'   # String: 'Имя целочисленной/длинной статистики, поддерживаемой в настоящее время: 'spiderAjax.urls.added' 
        operator: '>='                       # String ['==', '!=', '>=', '>', '<', '<=']: Operator used for testing
        value: 100                           # Int: Change this to the number of URLs you expect to find
        onFail: 'info'                       # String [warn, error, info]: Change this to 'warn' or 'error' for the test to take effect
