Laba-laba AJAX

Spider AJAX add-on yang terintegrasi dalam ZAP crawler dari situs AJAX kaya yang disebut Crawljax. Anda bisa menggunakannya untuk mengidentifikasi halaman situs yang ditargetkan. Anda dapat menggabungkannya dengan spider (normal) untuk hasil yang lebih baik.

Laba-laba dikonfigurasi dengan menggunakan Pilihan layar Spider AJAX.

This add-on supports the Automation Framework.

Diakses melalui

     Tab Spider AJAX
     Riwayat tab 'Menyerang/AJAX Spider...' klik kanan item menu
     Tab situs 'Menyerang/AJAX Spider...' klik kanan item menu

Lihat juga

     Pilihan layar Spider AJAX untuk sebuah ikhtisar dari Pilihan Spider AJAX
     Tab Spider AJAX untuk sebuah ikhtisar dari Tab Spider AJAX
     Dialog Spider AJAX untuk sebuah ikhtisar dari AJAX Spider Dialog
