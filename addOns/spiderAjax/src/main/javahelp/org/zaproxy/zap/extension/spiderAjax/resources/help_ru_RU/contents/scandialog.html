Диалоговое окно AJAX Spider

Этот диалог запускает Паука "Аякс".

Область

Первая вкладка позволяет изменять ключевые функции, такие как:

Отправная точка

URL-адрес, с которого AJAX spider начнет сканирование, или контекст (в этом случае будет использоваться один из URL, которые находятся в контексте в качестве отправной точки).

Контекст

Позволяет выбрать контекст для паука.

Пользователь

Позволяет выбрать один из существующих пользователей из выбранного контекста, чтобы выполнить сканирование паук как пользователь (зап будем (пере)авторизуйтесь как пользователь в случае необходимости).

Только в рамках

если установлено, то любые URL-адреса, выходящие за рамки, будут игнорироваться.

Примечание: Опция Just In Scope (Только в рамках ) является взаимоисключающим с Контекст вариант, если используется один, другой игнорируется.

Только поддерево паука

Если установлено, паук будет получать доступ только к ресурсам, которые находятся под начальной точкой (URI). При оценке того, найден ли ресурс в указанном поддереве, паук рассматривает только компоненты схемы, хоста, порта и пути URI.

Браузер

Тип используемого браузера.
Браузеры будут отображаться только в том случае, если ZAP имеет достаточную конфигурацию для их запуска.
Они по-прежнему могут не запускаться, если не удается найти браузеры или если информация о конфигурации неверна.

Показать дополнительные параметры

Если выбрано, будет показана вкладка «Параметры».

Параметры

Эта вкладка позволяет изменить параметры, включая:

Количество открытых окон браузера

Вы можете настроить количество окон, которые будет использовать AJAX Spider.
Чем больше окон, тем быстрее будет процесс.

Максимальная глубина обхода

Максимальная глубина, которую может достичь краулер. Ноль означает неограниченную глубину.

Максимальное количество состояний сканирования

Максимальное количество состояний, которое должен просканировать сканер. Ноль означает неограниченное количество состояний сканирования.

Максимальная продолжительность

Максимальное время работы сканера. Ноль означает неограниченное время работы.

Время ожидания события

Время ожидания после запуска события на стороне клиента.

Время ожидания перезагрузки

Время ожидания после загрузки URL.

Разрешенные ресурсы

Список разрешенных ресурсов. Разрешенные ресурсы всегда извлекаются, даже если они выходят за рамки, что позволяет включать необходимые ресурсы (например, сценарии) от третьих сторон.
