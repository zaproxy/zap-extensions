Параметры экрана AJAX Spider

Этот экран позволяет настроить AJAX-паук опции. AJAX Spider — это надстройка для краулер под названием Crawljax. Надстройка устанавливает локальный прокси в ZAP для связи с Crawljax. AJAX Spider позволяет сканировать веб-приложения, написанные на AJAX, гораздо глубже, чем родной Паук. Используйте AJAX Spider, если у вас есть веб-приложения, написанные на AJAX. Вы также должны использовать родной Spider, а также для полного покрытия веб-приложения (например, для покрытия комментариев HTML).

Параметры конфигурации


Поле Детали По умолчанию
браузер AJAX Spider использует внешний браузер для сканирования целевого сайта. Ты сможешь укажать, какой из них вы хотите использовать. Дополнительные сведения о поддерживаемых браузерах см. Страницы справки надстройки "Selenium". Firefox Headless
Количество открытых окон браузера Вы можете настроить количество окон, которые будет использовать AJAX Spider. Чем больше окон, тем быстрее будет процесс. 1
Максимальная глубина обхода Максимальная глубина, которую может достичь краулер. Ноль означает неограниченную глубину. 10
Максимальное количество состояний сканирования Максимальное количество состояний, которое должен просканировать сканер. Ноль означает неограниченное количество состояний сканирования. 0 (неограниченно)
Максимальная продолжительность Максимальное время работы сканера. Ноль означает неограниченное время работы. 60 минут
Время ожидания события Время ожидания после запуска события на стороне клиента. 1000 мс
Время ожидания перезагрузки Время ожидания после загрузки URL. 1000 мс
Нажмите Элементы один раз Если этот параметр включен, сканер пытается взаимодействовать с каждым элементом (например, щелкнуть мышью) только один раз. Если это не установлено, сканер попытается щелкнуть несколько раз. Отключение этого параметра является более строгим, но может занять значительно больше времени. Истинный
Используйте случайные значения в полях формы Когда включено, вставляет случайные значения в поля формы. В противном случае используются пустые значения. Истинный
Нажмите «Только элементы по умолчанию». Если этот параметр включен, во время сканирования будут нажиматься только элементы «a», «button» и «input». В противном случае он использует приведенную ниже таблицу, чтобы определить, какие элементы будут нажаты. Для более глубокого анализа отключите это и настройте интерактивные элементы в таблице. Истинный
Выберите элементы для нажатия во время сканирования (таблица) Список элементов для обхода. Эта таблица применяется только в том случае, если не включен параметр «щелкать только элементы по умолчанию». Используйте «включить все» для более глубокого анализа, хотя это может занять несколько больше времени. Все включено
Разрешенные ресурсы (таблица) Список разрешенных ресурсов. Разрешенные ресурсы всегда извлекаются, даже если они выходят за рамки, что позволяет включать необходимые ресурсы (например, сценарии) от третьих сторон.
