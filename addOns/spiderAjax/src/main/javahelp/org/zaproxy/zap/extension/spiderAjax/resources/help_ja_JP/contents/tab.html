AJAX スパイダー タブ

The AJAX Spider tab shows you the set of unique URIs found by AJAX Spider.


閲覧可能なリクエスト:
     リクエストインデックス - 各リクエストは１から番号付けされます。
     The request timestamp
     HTMLメソッド、例えば、GETまたはPOST
     リクエストされたURL
     The HTTP response status code
     HTTPレスポンスコードの短い要約
     リクエスト全体にかかった時間の長さ
     The size of the response header
     The size of the response body
     Any Alerts on the request
     Any Notes you have added to request
     Any Tags you have added to request
Selecting a requests will display it in the リクエストタブ 及び レスポンスタブ above.

右クリック メニュー

Right clicking on a node will bring up a menu which will allow you to:

攻撃

The Attack menu has the following submenus:

サイトの動的スキャン

This will initiate an 動的スキャン of the whole of the site containing the selected node.
　 動的スキャンタブ will be display and will show the progress of the scan.

動的スキャンノード

This will initiate an 動的スキャン of just the node selected.
　 動的スキャンタブ will be display and will show the progress of the scan.

Spider Site

This will initiate a スパイダー of the whole of the site containing the selected node.
　 スパイダータブ will be display and will show the progress of the scan.

Forced Browse Site

This will initiate a forced browse of the whole of the site containing the selected node.
　 Forced Browse tab will be display and will show the progress of the scan.

Forced Browse Directory

This will initiate a forced browse on the selected directory.
　 Forced Browse tab will be displayed and will show the progress of the scan.

Forced Browse Directory (and children)

This will initiate a forced browse on the selected directory and all children found.
　 Forced Browse tab will be displayed and will show the progress of the scan.

以下の処理から除外する:

このメニューには、次のサブメニューがあります。

プロキシ

This will exclude the selected nodes from the proxy. They will still be proxied via ZAP but will not be shown in any of the tabs.
This can be used to ignore URLs that you know are not relevant to the system you are currently testing.
The nodes can be included again via the セッション・プロパティ dialog

スキャン

This will prevent the selected nodes from being actively scanned.
The nodes can be included again via the セッション・プロパティ dialog

スパイダー

This will prevent the selected nodes from being spidered.
The nodes can be included again via the セッション・プロパティ dialog

アプリケーションの実行

This menu allows you to invoke applications that you have configured via the Options Applications screen which is also accessible via the 'Configure applications...' submenu.

タグ管理...

This will bring up the Manage Tags dialog which allows you to change the タグ associated with the request.

ノート...

This will bring up the ノートダイアログを追加 which allows you to record ノート related to the request.

ブレーク...

This will bring up the ブレークポイントダイアログを追加 which allows you to set a break point on that URL.

Alerts for this node

If the URL selected has アラート associated with it then they will be listed under this menu.
Selecting one of the alerts will cause it to be displayed.

再送信...

This will bring up the 再送信ダイアログ which allows you to resend the request after making any changes to it that you want to.

新しいアラート...

This will bring up the アラートダイアログを追加 which allows you to manually record a new Alert against this request.

ブラウザで表示

既定のブラウザーで選択されたノードのURLを開きます。
