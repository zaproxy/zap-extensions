AJAX スパイダー タブ

The AJAX Spider tab shows you the set of unique URIs found by AJAX Spider.


閲覧可能なリクエスト:
     リクエストインデックス - 各リクエストは１から番号付けされます。
     The request timestamp
     HTMLメソッド、例えば、GETまたはPOST
     リクエストされたURL
     The HTTP response status code
     HTTPレスポンスコードの短い要約
     リクエスト全体にかかった時間の長さ
     The size of the response header
     The size of the response body
     Any Alerts on the request
     Any Notes you have added to request
     Any Tags you have added to request
Selecting a requests will display it in the リクエストタブ 及び レスポンスタブ above.

右クリック メニュー

ノードを右クリックすると、以下の操作を可能にするメニューが表示されます：

攻撃

攻撃メニューには以下のサブメニューがあります：

アクティブスキャンサイト

This will initiate an アクティブスキャン of the whole of the site containing the selected node.
　 アクティブスキャンタブ will be display and will show the progress of the scan.

アクティブスキャンノード

This will initiate an アクティブスキャン of just the node selected.
　 アクティブスキャンタブ will be display and will show the progress of the scan.

Spider Site

This will initiate a スパイダー of the whole of the site containing the selected node.
　 スパイダータブ will be display and will show the progress of the scan.

Forced Browse Site

This will initiate a forced browse of the whole of the site containing the selected node.
　 強制ブラウズタブ will be display and will show the progress of the scan.

Forced Browse Directory

This will initiate a forced browse on the selected directory.
　 強制ブラウズタブ will be displayed and will show the progress of the scan.

Forced Browse Directory (and children)

This will initiate a forced browse on the selected directory and all children found.
　 強制ブラウズタブ will be displayed and will show the progress of the scan.

以下の処理から除外する:

このメニューには以下のサブメニューがあります：

プロキシ

選択したノードをプロキシから除外します。ZAP経由でプロキシされますが、どのタブにも表示されません。
現在テストしているシステムに関係のないURLを無視するために使用できます。
The nodes can be included again via the セッションプロパティ dialog

スキャナー

選択したノードがアクティブスキャンされることを防ぎます。
The nodes can be included again via the セッションプロパティ dialog

スパイダー

選択したノードがスパイダーされることを防ぎます。
The nodes can be included again via the セッション・プロパティ dialog

アプリケーションの実行

This menu allows you to アプリケーションの呼び出し that you have configured via the オプション - アプリケーション画面 which is also accessible via the 'Configure applications...' submenu.

タグ管理...

This will bring up the Manage Tags dialog which allows you to change the タグ associated with the request.

ノート...

This will bring up the ノートダイアログを追加 which allows you to record ノート related to the request.

ブレーク...

This will bring up the ブレークポイントダイアログを追加 which allows you to set a break point on that URL.

このノードのアラート

If the URL selected has アラート がある場合、このメニューの下に一覧表示されます。
アラートの1つを選択すると、そのアラートが表示されます。

再送信...

This will bring up the 再送信ダイアログ which allows you to resend the request after making any changes to it that you want to.

新しいアラート...

This will bring up the アラートの追加ダイアログ が表示され、このリクエストに対して新しい アラート against this request.

URLをブラウザで開く

選択したノードのURLを既定のブラウザーで開きます。
