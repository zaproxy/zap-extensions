Dialogo de araña AJAX

Éste dialogo lanza la araña AJAX.

Ámbito

La primera pestaña te permite cambiar características principales como:

Starting Point

El URL al que la araña AJAX se empezará a gatear desde, (en calquier manera va a ser usada por uno de los URL que se encuentran en contexto al inicio de la calle como punto de salida).

contexto

Permite seleccionar el contexto para ver alguna araña.

Usuario

Permite seleccionar uno de los usuarios disponibles del contexto seleccinado, para llevar a cabo escaneos de araña como un usuario (ZAP reautenticará al uuario tantas veces sea necesario).

Just In Scope

De configurarse, ninguna URLs, las cuales se encuentran fuera de alcance, serán ignoradas.

Nota: La opción Just In Scope es mutuamente exclusiva con Contexto opción, si uno está usado el otro debería estar libre.

Solo sub-arboles de Araña

Si se configura después entonces la araña solo accederá a los recursos que hay bajo el punto de incio (URI). Cuándo se evalúa si un recurso es encontrado entre el sub-árbol especificado, ñla araña solo considera el componente de esquema, el host y la ruta de componentes de URI.

Navegador

El tipo de navegador para usar.
Los navegadores sólo aparecerán si ZAP tiene suficiente configuración para correrlos.
Estos todavía podrían fallar para correr, si los navegadores se encuentran colapsados o si la configuración es incorrecta.

Show Advanced Options

De ser seleccionada, la pestaña de opciones será mostrada.

Opciones

Esta pestaña te permite cambiar opciones, incluyendo ninguna update:

Number of Browser Windows to Open

Pudes configurar el numero de ventanas a ser usadas por la araña AJAX.
Mientras mas ventanas, más rápido será el proceso.

Maximum Crawl Depth

La máxima profundidad que el crawler puede alcanzar, cero representa una vista ilimitada.

Maximum Crawl States

The maximum number of states that the crawler should crawl. Zero means unlimited crawl states.

Maximum Duration

The maximum time that the crawler is allowed to run. Zero means unlimited running time.

Event Wait Time

The time to wait after a client side event is fired.

Reload Wait Time

The time to wait after URL is loaded.

Allowed Resources

The list of allowed resources. The allowed resources are always fetched even if out of scope, allowing to include necessary resources (e.g. scripts) from 3rd-parties.
