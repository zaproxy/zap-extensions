Aba do Spider Ajax

A aba do spider AJAX aranha mostra o conjunto de URIs encontrado pelo Spider AJAX.


Para cada request, você pode ver:
     O índice do request - cada solicitação é numerada, começando em 1
     O timestamp do request
     O método HTML, por exemplo, GET ou POST
     A URL solicitada
     O código de status da resposta HTTP
     Um breve resumo do que significa o código de resposta HTTP
     O tempo que levou toda a solicitação
     O tamanho do cabeçalho de resposta
     O tamanho do corpo da resposta
     Alertas sobre o request
     Quaisquer notas que você tiver adicionado ao pedido
     Qualquer Tags que você adicionou ao pedido
Um request selecionado será exibido na Aba de Request e Na Aba de Respostas acima.

Menu do botão direito

Clicar com o botão direito em um nó faz aparecer um menu que lhe permitirá:

Atacar

O menu de ataque tem os seguintes submenus:

Varredura Ativa de Site

Isto irá iniciar uma Varredura Ativa de todo o site contido no nó selecionado.
A aba da varredura ativa vai aparecer e irá mostrar o progresso da varredura.

Nó da Varredura ativa

Isto irá iniciar uma Varredura Ativa apenas no nó selecionado.
A aba da varredura ativa vai aparecer e irá mostrar o progresso da varredura.

Usar o Spider no site

Isto irá iniciar uma varredura do Spider de todo o site contido no nó selecionado.
A Aba do Spider vai aparecer e irá mostrar o progresso da varredura.

Site com Navegação Forçada

Isto irá iniciar uma navegação forçada de todo o site contendo o nó selecionado.
A aba navegação forçada vai aparecer e irá mostrar o progresso da varredura.

Diretório com navegação forçada

Isto irá iniciar uma navegação forçada no diretório selecionado.
A aba navegação forçada vai aparecer e irá mostrar o progresso da varredura.

Diretório (e subdirs) com navegação forçada

Isto irá iniciar uma navegação forçada no diretório selecionado e todos os subdiretórios encontrados.
A aba navegação forçada vai aparecer e irá mostrar o progresso da varredura.

Exclusões

Este menu possui os seguintes submenus:

Proxy

Isto irá excluir do proxy os nós selecionados. Eles ainda passarão pelo ZAP, mas não serão exibidos em qualquer uma das guias.
Isto pode ser usado para ignorar URLs que você sabe que não são relevantes para o sistema que você está testando.
Os nós podem ser incluídos novamente através do diálogo de Propriedades da Sessão

Scanner

Isto irá impedir que os nós selecionados entrem na varredura ativa.
Os nós podem ser incluídos novamente através do diálogo de Propriedades da Sessão

Spider

Isto irá impedir que os nós selecionados sejam percorridos pelo spider.
Os nós podem ser incluídos novamente através do diálogo de Propriedades da Sessão

Executar a aplicação

Este menu permite a você Invocar aplicações que você tenha configurado através da Tela de opções de aplicações que também é acessível via o submenu 'Configurar aplicações...'.

Gerenciar Tags...

Isso irá abrir o diálogo Gerenciar Tags que permite a você alterar as tags associadas com a solicitação.

Notas...

Isso irá abrir o diálogo Acrescentar nota que permite a você gravar notas relacionada com a solicitação.

Interrupção...

Isso irá abrir o diálogo Adicionar ponto de interrupção que permite a você definir um ponto de interrupção naquela URL.

Alertas para este nó

Se a URL selecionada tem alertas associados a ela, então eles serão listados aqui neste menu.
Selecionar um dos alertas fará com que ele seja exibido.

Reenviar...

Isso irá abrir o Diálogo de reenvio que lhe permite reenviar o request depois de fazer quaisquer alterações que você deseje.

Novo Alerta...

Isso irá abrir o Diálogo Acrescentar Alerta que permite a você registrar manualmente um novo Alerta para um request.

Ver no Navegador

Isto irá abrir a URL do nó selecionado em seu navegador padrão.
