Pestaña AJAX Spider

The AJAX Spider tab shows you the set of unique URIs found by AJAX Spider.


Por cada petición podrá observar:
     El index de la petición - cada petición está numerada, empezando por el número 1
     The request timestamp
     El método HTML, por ejemplo GET o POST
     La URL solicitada
     The HTTP response status code
     Un breve resumen de lo que significa el código de respuesta HTTP
     La duración total de la solicitud
     El tamaño del encabezado de respuesta
     El tamaño del cuerpo de respuesta
     Any Alerts on the request
     Any Notes you have added to request
     Any Tags you have added to request
Al seleccionar una petición, ésta será mostrada en la pestaña Petición y en la pestaña Respuesta ubicadas mas arriba.

Menú de botón derecho

Al hacer click derecho sobre un nodo se abrirá un menú que le permitirá:

Atacar

El menú de ataque tiene los siguientes submenús:

Escaneo de sitio Activo

Esto iniciará un Escaneo Activo en todo el sitio contenido en el nodo seleccionado.
Se mostrará la Pestaña de Escaneo Activo indicando el progreso del escaneo.

Nodo de escaneo Activo

Esto iniciará un Escaneo Activo solo en el nodo seleccionado.
Se mostrará la Pestaña de Escaneo Activo indicando el progreso del escaneo.

Sitio Spider

Esto iniciará un spider en todo el sitio contenido en el nodo seleccionado.
Se mostrará la Spider tab indicando el progreso del escaneo.

Sitio de navegación predefinida

Esto iniciará un navegación forzada en todo el sitio contenido en el nodo seleccionado.
Se mostrará la pestaña de Navegación Predefinida indicando el progreso del escaneo.

Directorio de navegación definido

Esto iniciará un navegación forzada en el directorio seleccionado.
Se mostrará la pestaña de Navegación Predefinida indicando el progreso del escaneo.

Directorio de navegación definido (e hijos)

Esto iniciará un navegación forzada en el directorio seleccionado y en cualquier directorio hijo encontrado.
Se mostrará la pestaña de Navegación Predefinida indicando el progreso del escaneo.

Excluir de

Este menú tiene los siguientes submenús:

Proxy

This will exclude the selected nodes from the proxy. They will still be proxied via ZAP but will not be shown in any of the tabs.
Esto puede ser utilizado para ignorar URLs que sabe que no son relevantes para el sistema que está probando.
Los nodos pueden ser incluidos a través del cuadro de dialogo Propiedades de la Sesión .

Escáner

Esto evitará que los nodos seleccionados sean escaneados activamente.
Los nodos pueden ser incluidos a través del cuadro de dialogo Propiedades de la Sesión .

Spider(Araña)

Esto evitará que los nodos seleccionados sean rastreados.
Los nodos pueden ser incluidos a través del cuadro de dialogo Propiedades de la Sesión .

Ejecutar aplicación

Este menú le permite invocar aplicaciones that you have configured via the Pantalla de opciones de aplicaciones which is also accessible via the 'Configure applications...' submenu.

Administrar Tags...

Esto hará que aparezca el cuadro de dialogo Gestionar Etiquetas que le permite cambiar las etiquetas asociadas a la solicitud.

Nota...

Esto hará que aparezca el cuadro de dialogo Agregar Nota que le permitirá guardar notas related to the request.

Parada...

Esto hará que aparezca el cuadro de dialogo Add Break Point dialog which allows you to set a break point on that URL.

Alertas para este nodo

Si la URL seleccionada tiene alertas asociadas a ella entonces serán listadas en este menú.
Seleccionar una alerta hará que se muestre en pantalla.

Reenviar...

Esto hará que aparezca el cuadro de dialogo Reenviar que le permite reenviar la solicitud después de hacer cualquier cambio que desee.

Nueva alerta...

Esto hará que aparezca el cuadro de dialogo Añadir Alerta, el que le permitirá agregar una nueva Alert de forma automática en la petición.

Ver en el navegador

La dirección URL del nodo seleccionado se abrirá en el navegador por defecto.
