令牌生成和分析

此附件允许您生成和分析伪随机令牌，例如像用于会话处理或者CSRF保护的那些一样。

为了生成和分析令牌：

  • 查找一个包含您想要生成的令牌请求
  • 右键单击站点选项卡或者其中一个底部选项卡，同时选择"生成令牌..."
  • 选择您想要生成的令牌
  • 按生成按钮
  • 在令牌生成器（Token Gen）选项卡中列出响应和令牌
  • 当所有令牌生成后，将显示"分析令牌(Analyse Tokens)"对话框
Note: Token generations are not allowed in Safe mode nor Protected if the message is not in scope.
