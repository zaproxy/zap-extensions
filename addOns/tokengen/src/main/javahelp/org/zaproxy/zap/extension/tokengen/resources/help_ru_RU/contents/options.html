Экран генератора токенов опций

На этом экране можно настроить генератор токенов.

Параметры конфигурации

Поле Детали По умолчанию Файл конфигурации
Количество потоков Количество потоков, используемых во время генерации токена. 5 Ключ: tokengen.threadsPerScan
Значение: неотрицательное целое число.
Задержка запроса (в миллисекундах) Время ожидания перед отправкой каждого запроса, чтобы избежать перегрузки целевого сервера.

Примечание. Учитывая большое количество запросов, отправленных во время токена Генерация, увеличивающая задержку, может сильно повлиять на время, в течение которого поколение займет, чтобы завершить.

0 Ключ: tokengen.requestDelayInMs
Значение: неотрицательное целое число.
