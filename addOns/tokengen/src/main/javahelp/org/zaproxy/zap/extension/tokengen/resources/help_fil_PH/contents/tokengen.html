Ang token na tagabuo at pagsusuri

Ang add-on na ito ay nagbibigay-daan sa iyo upang bumuo at umitindi ng pseudo random na mga token, tulad ng mga ginagamit para sa sesyon sa paghawak o CSRF proteksyon.

Upang makabuo at umitindi ng mga token:

  • Maghanap ng isang kahilingan na naglalaman ng token na gusto mong buohin
  • Kanan na klik sa tab ng site at isa sa mga ibabang mga tab at piliin ang 'Makabuo ng mga token...'
  • Piliin ang token na gusto mong mabuo
  • Pindutin ang Generate na button
  • Ang mga sagot at mga token ay pagkatapos malista sa 'Token Gen' na tab
  • Kapag lahat ng mga token ay nabuo sa 'Analyse Tokens' na diyalogo ay ipapakita
Note: Token generations are not allowed in Safe mode nor Protected if the message is not in scope.
