Geração e análise de tokens

Este complemento permite a você gerar e analisar tokens pseudo aleatórios, tais como aqueles usados para manipulação de sessão ou proteção CSRF.

Para gerar e analisar tokens:

  • Encontre uma solicitação que contém o token que você deseja gerar
  • Clique com o botão direito na aba Sites ou em uma das abas da parte inferior e selecione 'Gerar Tokens...'
  • Escolha o token que você deseja gerar
  • Pressione o botão gerar
  • As respostas e tokens de então serão listados na guia 'Token Gen'
  • Quando todos os tokens foram gerados será exibida a caixa de diálogo 'Analisar Tokens'
Note: Token generations are not allowed in Safe mode nor Protected if the message is not in scope.
