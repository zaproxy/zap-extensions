Generasi dan Analisis Token

Tambahan ini memungkinkan anda untuk menghasilkan dan menganalisis token acak palsu, seperti yang digunakan untuk penanganan sesi atau perlindungan CSRF.

Untuk menghasilkan dan menganalisis token:

  • Temukan permintaan yang berisi token yang ingin anda hasilkan
  • Klik kanan pada situs tab atau salah satu tab bawah dan pilih 'Generate Token...'
  • Pilih token yang ingin anda hasilkan
  • Tekan tombol Generate
  • Respon dan token akan tercantum dalam tab 'Token Gen'
  • Ketika semua token telah dihasilkan dialog 'Analyse Token' akan ditampilkan
Note: Token generations are not allowed in Safe mode nor Protected if the message is not in scope.
