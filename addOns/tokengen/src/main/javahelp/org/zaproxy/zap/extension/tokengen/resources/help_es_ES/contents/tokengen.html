Generación de Token y Análisis

Este complemento permite generar y analizar tokens seudo aleatorios, como esos utilizados por el manejo de la sesión y protección CSRF.

Para generar y analizar tokens:

  • Encuentra una solicitud que contenga el token que se quiere generar
  • Haga clic derecho en la pestaña Sitios o uno de las pestañas del fondo y seleccione "Generar Tokens..."
  • Seleccione el token que se quiere generar
  • Presione el botón Generar
  • Las respuestas y los token se mostrarán en una lista en la pestaña "Token Gen"
  • Cuando todos los tokens hayan sido generados se mostrará el cuadro de diálogo "Analizar Tokens"
Note: Token generations are not allowed in Safe mode nor Protected if the message is not in scope.
