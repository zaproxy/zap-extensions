Escáner de secuencia

Este complemento facilita las secuencias de registro y escaneo de las solicitudes. En algunas aplicaciones web, para eventos o páginas particulares es necesario ser accedidas o completadas en determinado orden. Al registrar una secuencia y luego escanearla, es posible asegurar que dicho flujo ocurra en una manera esperada.

Crear secuencias

Un script de secuencia debe ser creado por cada operación multi-step en la aplicación o sitio a prueba.

Hay dos opciones para crear secuencias:

  1. Ya sea en el árbol de Sitios o la pestaña Historial seleccione las solicitudes que se deseen incluir, haga clic derecho y utilice "Agregar al script Zest" (ya sea pare crear un nuevo script o agregar a un script de Secuencia existente).
  2. Desde la barra de herramienta principal, utilice el botón "Registrar un nuevo script Zest...", seleccionando como tipo "Secuencia".

Escanear
