Платформа автоматизации — задержка задания

Это задание ожидает определенное время, если не выполняется одно из определенных условий.
Его можно использовать для ожидания прохождения регрессионных тестов через ZAP, чтобы более тщательно изучить ваше приложение.

Параметр времени поддерживает форматы чч:мм:сс, мм:сс и сс, поэтому «5» — это 5 секунд, «1:30» — это одна минута и 30 секунд, а «2:20:30» — это 2 часа, 20 минут и 30 секунд.

Поддерживаемые условия:

  • Создание файла, заданного необязательным параметром fileName.
  • Вызов статического метода: org.zaproxy.addon.automation.jobs.DelayJob.setEndJob(true);
  • Calling the API end point: automation / action / endDelayJob

YAML

  - type: delay                        # Приостановить выполнение плана на определенный период времени или событие (создание файла, вызов статического метода, вызов конечной точки API) 
    parameters:
      time:                            # String: Время ожидания, формат любой из ['чч:мм:сс', 'мм:сс', 'сс'], по умолчанию: 0 
      fileName:                        # String: Имя файла, создание которого приведет к досрочному завершению задания, по умолчанию: пусто
