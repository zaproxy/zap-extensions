Pantalla de opciones Fuzzer

Esta pantalla permite configurar las fuzzing opciones:

Categoría predeterminada

La categoría que se seleccionará inicialmente se muestra cuando el Diálogo Fuzz se muestre.

Subprocesos simultáneos de escaneo por host

La cantidad de subprocesos que usará el escáner por host.
Aumentar el número de subprocesos acelerará el escaneo, pero puede ejercer más presión sobre la computadora en la que se está ejecutando ZAP y el host de destino.

Agregue un archivo Fuzz personalizado

Te permite agregar tus propios archivos para usarlos cuando fuzzing.
Estos deberían ser archivos de texto con un ataque por línea.
Los archivos se agregan al directorio 'fuzzers' debajo del directorio inicial de ZAP.
