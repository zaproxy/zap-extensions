Diálogo Fuzzer

Esto le permite seleccionar los fuzzers para usar cuando fuzzing haga una solicitud.

Pestaña de ubicaciones de Fuzz

Para configurar el fuzzing:
  • Resalte una cadena que desee difuminar en la pestaña de Ubicaciones de Fuzz
  • Haga clic en el botón 'Agregar...' para abrir el cuadro de diálogo
  • Agregue las cargas útiles que desea usar
  • Haga clic en el botón 'Procesadores...' para abrir el diálogo de Procesadores de Ubicación (si es necesario)
  • Haga clic en el botón 'Comenzar Fuzzer' para iniciar el fuzzing
  • Los resultados se mostrarán en la pestaña Fuzzer y los seleccionarán para ver las solicitudes y respuestas completas.
También puede buscar cadenas en los resultados de fuzz usando la 'pestaña Buscar'.

Pestaña de opciones

Esta pestaña le permite configurar las opciones que se utilizarán cuando fuzzing.

Pestaña de Procesadores de mensajes

Los procesadores de mensajes pueden acceder y cambiar los mensajes que se están enviando, controlar el proceso de fuzzing e interactuar con la interfaz de usuario de ZAP.
Los procesadores de mensajes disponibles dependen del tipo de mensaje que se está difuminando, este complemento incluye los procesadores de mensajes HTTP predeterminados.

Los complementos también pueden definir generadores de carga adicionales.
