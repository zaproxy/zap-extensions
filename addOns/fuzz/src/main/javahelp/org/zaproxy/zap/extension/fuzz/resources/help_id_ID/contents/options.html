Layar pilihan Fuzzer

Layar ini memungkinkan anda untuk mengkonfigurasi fuzzing pilihan:

Kategori default

Kategori yang akan dipilih mulanya ditampilkan saat Fuzz dialog is displayed.

Bersamaan pemindaian benang per host

Jumlah benang yang akan digunakan pemindai per tuan rumah.
Meningkatkan jumlah benang akan mempercepat pemindaian namun dapat memberi tekanan ekstra pada komputer ZAP yang sedang berjalan dan target tuan rumah.

Add custom Fuzz file

Memungkinkan anda untuk menambah berkas anda sendiri untuk digunakan saat fuzzing.
Ini harus menjadi berkas teks dengan satu kali serangan per baris.
Berkas ditambahkan ke direktori 'fuzzer' dibawah direktori beranda ZAP.
