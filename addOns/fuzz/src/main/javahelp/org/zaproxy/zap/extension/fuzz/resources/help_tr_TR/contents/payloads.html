Payload'lar diyaloğu

Bu size ekstra güç üreticisini kullanmanıza izin verir belirsizleştirmek bir sorgu.

Ekstra güç üreticiler belirsizleştiricinin gönderdiği hedef uygulamalara hafif saldırı üretir.

Aşağıdaki üretici tipleri varsayılan olarak sağlanır:
  • Boş/Sıfır - seçili ek gücü birçok kez mesaji değiştirmeden üretir. Bu yük üreticisi sonradan işlenen çoklu mesaj gönderimi için kullanışlıdır, örneğin, bir Fuzzer HTTP İşlemci (Script) ile.
  • Dosya - yerel dosyalarınızdan birini bir defaya mahsus bir saldırı için seçiniz
  • File Fuzzers - select any combination of the fuzzing files registered with ZAP, e.g. via add-ons like fuzzdb
  • Numberzz - özel artışla bir dizi sayı üretmesine olanak tanır
  • Regex - regex desenlerine dayalı saldırılar üretir
  • Dizeler - el ile girilebilen veya yapıştırılan işlenmemiş dizeler
  • Komut dosyası - gerekli yük üretebilen özel komut dosyaları
  • Json - generate attacks by fuzzing the provided json
Özel yük üretici scriptleri yazabilirsiniz - bunlar ihtiyacınız olan her yükü sağlayabilir.

Eklentiler ek yük üreticileri olarak da tanımlanır.

'İşlemciler...' düğmesi Payload İşlemciler iletişim kutusu seçtiğiniz palyload jeneratörüne uygulanan yük taşıma işlemcilerini yapılandırmanıza olanak tanır.
