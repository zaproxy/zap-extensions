Pestaña Fuzzer

La pestaña Fuzzer muestra las solicitudes y respuestas realizadas cuando se le hace fuzzing a una cadena.
Seleccione una fila para ver las solicitudes y respuestas realizadas. También puede buscar por cadenas en los resultados del fuzzing utilizando la pestaña de búsqueda.

Resultados del Fuzzer en HTTP

Los resultados deben ser evaluados manualmente para saber si se encontró alguna vulnerabilidad.

Menú de clic derecho

Right clicking on a row will bring up a menu which has the same options as the History tab. Plus providing an option to add messages from the fuzz results to the Sites Tree and History tab (the messages will be tagged FromFuzzer and initially show a fuzzer icon in the Sites Tree).
