ペイロードプロセッサダイアログ

これは、特定のペイロードジェネレーターで使用するペイロードプロセッサを選択できるようにします。

組み込みペイロードプロセッサには、以下が含まれます：
  • Base64 デコード
  • Base64 エンコード
  • 拡張（指定された最小長まで）
  • JavaScript エスケープ
  • JavaScript アンエスケープ
  • MD5 ハッシュ
  • 接尾辞文字列
  • 接頭辞文字列
  • SHA-1 ハッシュ
  • SHA-256 ハッシュ
  • SHA-512 ハッシュ
  • トリム
  • URL デコード
  • URL エンコード
また、カスタムのペイロードプロセッサスクリプトを作成することもできます。これらは、必要とするあらゆるペイロードの操作を実行できます。

アドオンによって、追加のペイロードプロセッサが定義されることもあります。
