Fuzzer tab

The Fuzzer tab shows you the requests and responses performed when you fuzz sebuah string.
Memilih turut melihat penuh permintaan dan tanggapan. Anda juga dapat mencari string dalam fuzz hasil menggunakan tab Pencarian.

Hasil HTTP Fuzzer

Hasil harus secara manual dinilai untuk mengetahui jika ada kerentanan yang ditemukan.
