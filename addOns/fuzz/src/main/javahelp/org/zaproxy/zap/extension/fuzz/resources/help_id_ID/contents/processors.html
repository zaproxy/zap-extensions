Dialog Prosesor Payload

Hal ini memungkinkan anda untuk memilih payload prosesor untuk penggunaan tertentu dengan muatan generator.

Dibangun pada muatan prosesor meliputi:
  • Base64 Decode
  • Encode Base 64
  • Mengembangkan (untuk panjang minimal yang ditetapkan)
  • JavaScript melarikan diri
  • JavaScript Unescape
  • Hash MD5
  • String Postfix
  • String Awalan
  • SHA-1 Hash
  • SHA-256 Hash
  • SHA-512 Hash
  • Memangkas
  • URL Decode
  • Menyandi URL
Anda juga dapat menulis kustom muatan prosesor script ini dapat melakukan manipulasi muatan yang anda butuhkan.

Add-on juga dapat mendefinisikan prosesor muatan tambahan.
