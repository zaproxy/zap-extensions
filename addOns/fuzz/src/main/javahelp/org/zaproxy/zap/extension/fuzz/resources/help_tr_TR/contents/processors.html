Yük işlemcileri penceresi

Bu yük işlemcilerini belirli yük üreticilerle birlikte kullanmanıza izin verir.

Yük işlemcilerinde yapı şunu içerir:
  • Base64 Kod Çözme
  • Base64 Kodlama
  • Genişletin (belirtilen minimum uzunluğa kadar)
  • JavaScript Escape
  • JavaScript Unescape
  • MD5 karma değeri
  • Sonek Dize
  • Önek Dize
  • SHA-1 karma Şifreleme
  • SHA-256 Karma Şifreleme
  • SHA-512 Karma Şifreleme
  • Kırp
  • URL Kod Çözme
  • URL Kodlama
Ayrıca özel yük taşıma işlemcisi komut dosyalarına yazdırabilirsiniz; bunlar, size gereken yükün herhangi bir manipülasyonunu gerçekleştirebilir.

Eklentiler ek yük işlemcileri olarak da tanımlanır.
