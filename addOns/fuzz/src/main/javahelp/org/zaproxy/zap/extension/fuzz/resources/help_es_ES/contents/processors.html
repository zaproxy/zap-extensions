Diálogo de procesadores de carga útil

Esto le permite seleccionar los procesadores de carga útil que serán utilizados con generadores de carga útil específicos.

Entre los procesadores de carga útil incorporados se encuentran:
  • Base64 Decode
  • Base64 Encode
  • Expand (hasta un tamaño mínimo especificado)
  • JavaScript Escape
  • JavaScript Unescape
  • MD5 Hash
  • Postfix String
  • Prefix String
  • SHA-1 Hash
  • SHA-256 Hash
  • SHA-512 Hash
  • Trim
  • URL Decode
  • URL Encode
También puede programar scripts de procesadores de carga útil personalizados, estos pueden realizar cualquier manipulación a la carga útil que necesite.

Add-ons can also define additional Payload Processors.
