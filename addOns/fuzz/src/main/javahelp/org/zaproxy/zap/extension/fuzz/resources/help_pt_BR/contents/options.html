Tela de opções do Fuzzer

Esta tela permite que você configure opções de fuzzing :

Categoria Padrão

É a categoria a ser selecionada inicialmente e exibida quando o diálogo de Fuzz for exibido.

Número de varreduras simultâneas em cada host

Número de subdivisões de varreduras (threads) que o ZAP vai fazer em cada host.
Aumentar o número de threads vai acelerar o escaneamento mas pode sobrecarregar tanto o computador onde o ZAP está rodando quando o host alvo.

Inserir arquivo de fuzz customizado

Permite a você acrescentar seus próprios arquivos para serem utilizados no fuzzing.
Eles devem ser arquivos de texto com apenas um ataque por linha.
Os arquivos são gravados no diretório 'fuzzers', abaixo do diretório home do ZAP.
