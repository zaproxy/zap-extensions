Diyalogo ng mga Processor ng Kargamento

Ito ay nagpapahintulot sa iyo na pumili ng mga processor ng kargamento na gamitin sa tiyak na taga-buo ng kargamento.

Ang mga processor ng kargamento ay kinabibilangan ng:
  • Decode na Base64
  • Encode na Base64
  • Palawakin (sa isang pinakamaliit na itinakdang haba)
  • JavaScript Escape
  • JavaScript Unescape
  • MD5 Hash
  • Postfix String
  • Prefix String
  • SHA-1 Hash
  • SHA-256 Hash
  • SHA-512 Hash
  • Pumantay
  • URL Decode
  • URL Encode
Maaari mo ring isulat ang mga script ng processor ng pasadyang kargamento - ang mga ito ay maaaring magsagawa ng anumang pagmamanipula ng kargamento na kailangan mo.

Ang mga add-on ay maaari ring magtukoy ng karagdagang processor ng kargamento.
