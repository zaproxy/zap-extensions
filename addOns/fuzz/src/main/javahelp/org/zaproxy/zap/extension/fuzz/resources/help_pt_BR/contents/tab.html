Aba do fuzzer (difusor)

A aba do difusor mostra as solicitações e respostas realizadas quando você faz difusão de uma string.
Selecionando uma linha você vê todas as solicitações e respostas. Você também pode procurar strings nos resultados do fuzzing usando a guia 'pesquisar'.

Resultados de difusor de HTTP

Os resultados têm de ser avaliados manualmente para saber se alguma vulnerabilidade foi encontrada.

Menu do botão direito

Right clicking on a row will bring up a menu which has the same options as the History tab. Plus providing an option to add messages from the fuzz results to the Sites Tree and History tab (the messages will be tagged FromFuzzer and initially show a fuzzer icon in the Sites Tree).
