командная строка

Примечание. Эта функция доступна только в еженедельных выпусках и версиях после 2.11.
Дополнение Network поддерживает следующие параметры командной строки:
     -certload Загружает корневой сертификат ЦС (Root CA certificate) из указанного файла.
     -certpubdump Сбрасывает сертификат Root CA в указанный файл, это удобно для импорта в браузеры.
     -certfulldump Сбрасывает сертификат Root CA и закрытый ключ в указанный файл, это подходит для импорта в ZAP.
