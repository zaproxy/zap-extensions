Параметры

The Network add-on provides the following options screens:
     Client Certificates Allows to configure the client certificates, PKCS#11 and PKCS#12.
     Connection Allows to configure the connection.
     Global Exclusions Allows to configure the global exclusions.
     Local Servers/Proxies Allows to manage and configure the local servers/proxies.
     Rate Limit Allows to configure rate limiting for domains and hosts.
     Server Certificates Allows to manage and configure the root CA certificate and issued certificates.
