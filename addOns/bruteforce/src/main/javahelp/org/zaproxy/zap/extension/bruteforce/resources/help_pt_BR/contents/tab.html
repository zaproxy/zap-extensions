Aba de navegação forçada

A aba de navegação forçada permite que você execute uma varredura com Navegação Forçada em qualquer dos sites acessados.

Os sites podem ser selecionados através da barra de ferramentas ou pela Aba de Sites.
Quaisquer sites que foram ou estão sendo escaneados são marcados em negrito no menu pulldown de Sites da barra de ferramentas.
A barra de ferramentas fornece um conjunto de botões que permitem iniciar, parar, pausar e continuar a verificação.
Uma barra de progresso mostra o quanto progrediu a verificação do site selecionado.
O indicador 'Varreduras em andamento' mostra quantas varreduras estão ativas - passar o cursor sobre esse valor irá mostrar num pop-up uma lista dos sites que estão sendo examinados.

Menu do botão direito

Clique com o botão direito em uma ou mais linhas, e aparecerá um menu que lhe permitirá:

Excluir de

Este menu possui os seguintes submenus:

Proxy

Isto irá excluir do proxy os nós selecionados. Eles ainda passarão pelo ZAP, mas não serão exibidos em qualquer uma das guias.
Isto pode ser usado para ignorar URLs que você sabe que não são relevantes para o sistema que você está testando.
Os nós podem ser incluídos novamente através do diálogo Propriedades da Sessão .

Scanner

Isto irá impedir que os nós selecionados entrem na varredura ativa.
Os nós podem ser incluídos novamente através do diálogo Propriedades da Sessão .

Spider

Isto irá impedir que os nós selecionados sejam percorridos pelo spider.
Os nós podem ser incluídos novamente através do diálogo Propriedades da Sessão .

Copiar

Isso irá copiar as URLs selecionadas para a área de transferência.

Reenviar...

Isso irá abrir o Diálogo de reenvio que lhe permite reenviar o request depois de fazer quaisquer alterações que você deseja.

Novo Alerta...

Isso irá abrir o diálogo Acrescentar Alerta que permite a você registrar manualmente um novo alerta para este request.

Ver no Navegador

Isto irá abrir a URL do nó selecionado em seu navegador padrão.

Gerar formulário de teste anti CSRF

Isto irá abrir a URL de um formulário destinado à detecção de problemas de CSRF.
Ele só vai ser habilitado para solicitações POST - caso o API esteja habilitado e se o Java suportar a abertura de URLs em um navegador na sua plataforma.
