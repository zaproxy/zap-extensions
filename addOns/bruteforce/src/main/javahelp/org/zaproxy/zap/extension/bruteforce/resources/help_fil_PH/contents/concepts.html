Sapilitang pagbasa

Ang ZAP ay nagpapahintulot sayo na subukan tuklasan ang mga direktoryo at mga file na gumagamit ng sapilitang pag-browse. Ang isang set ng mga file ay naibibigay na naglalaman ng malaking numero ng file at direktoryong mga pangalan.
Ang ZAP ay sumubok na magdirektang pumasok sa lahat ng mga file at mga direktoryo na nakalista sa napiling mga file na direkto sa halip na nagdepende sa mga nakitang mga link sa kanila.

Ang sapilitang browser ay na-configure gamit ang Mga opsyon Sapilitang Browse na screen.

Ang functionality ay batay sa code na mula sa OWASP DirBuster na proyekto.

Ma-access sa pamamagitan ng

     Sapilitang Browse tab
     Ang mga site tab 'Atake/Sapilitang Browse na site' i-right click ang menu item
     Ang mga site tab 'Atake/Sapilitang Browse sa direktoryo' i-right click ang menu item
     Mga site tab 'Atake/Sapilitang Browse sa direktoryo (at children)' i-right click ang menu item

Ang panlabas na mga link
