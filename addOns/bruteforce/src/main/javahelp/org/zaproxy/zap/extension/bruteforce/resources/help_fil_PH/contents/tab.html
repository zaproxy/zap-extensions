Puwersang Pag-browse na tab

Ang Puwersang Pag-browse na tab ay magpapahintulot sa iyo na isagawa ang Puwersang Pag-browse na pagscan sa kahit anong mga sayt na na-akses.

Ang mga sayt ay maaring mapili gamit ang toolbar o ang Mga sayt na tab.
Ang kahit anong mga sayt na kasalukuyang na-scan ay namarkahan ng bold sa toolbar Mga Sayt na kontrol na pag-pulldown.
Ang toolbar ay nagbibigay ng hanay ng mga button na magpapahintulot sa iyo na simulan, itigil, i-pause at ipagpatuloy ang pag-scan.
Ang progresong bar ay magpapakita kung gaano na kalayo ang pagscan sa napiling site.
Ang 'Kasalukuyang mga scan' na balyu ay magpipikita kung gaano karaming mga scan ang kasalukuyang aktibo - ang pag-hover sa ibabaw ng balyung ito ay magpapakita ng listahan ng mga sayt na na-scan sa isang popup.
The toolbar also includes a button to export the displayed content as CSV.

Kanang pagpindot ng menu

Right clicking on one or more rows will bring up a menu which will allow you to:

Magbukod mula

This menu has the following submenus:

Proxy

This will exclude the selected nodes from the proxy. They will still be proxied via ZAP but will not be shown in any of the tabs.
This can be used to ignore URLs that you know are not relevant to the system you are currently testing.
The nodes can be included again via the Mga Katangian ng Sesyon dialog

Taga-scan

This will prevent the selected notes from being actively scanned.
The nodes can be included again via the Mga Katangian ng Sesyon dialog

Ispayder

This will prevent the selected nodes from being spidered.
The nodes can be included again via the Mga Katangian ng Sesyon dialog

Pagkopya

This will copy the selected URLs to the clipboard.

Muling ipadala...

This will bring up the Muling pagpapadala na diyalogo which allows you to resend the request after making any changes to it that you want to.

Bagong Alerto...

This will bring up the Magdagdag ng Alerto na diyalogo which allows you to manually record a new alerto against this request.

Buksan ang URL sa Browser

This will open the URL of the selected node in your default browser.

Gumawa ng anti CSRF na pangtest na form

This will open a URL which will give you a generated form for testing for CSRF issues.
It will only be enabled for POST requests, if the API is enabled and if Java supports the opening of URLs in a browser on your platform.
