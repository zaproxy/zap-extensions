Pilihan Dipaksa melihat Isi layar

Layar ini memungkinkan anda untuk mengkonfigurasi Dipaksa Browse pilihan:

Bersamaan pemindaian benang per host

Jumlah benang yang akan digunakan pemindai per host.
Meningkatkan jumlah benang yang akan mempercepat pemindaian, tetapi dapat menempatkan tekanan ekstra pada komputer ZAP yang berjalan pada host target.

Rekursif

Jika dicentang maka pemindai akan rekursif melalui semua sub-direktori yang ditemukan.
Ini dapat meluangkan waktu yang lama.

Berkas bawaan

Berkas bawaan yang dipilih ketika ZAP dimulai.

Tambahkan custom Dipaksa Browse file

Memungkinkan anda untuk menambahkan berkas anda sendiri untuk digunakan ketika membobol berkas dan direktori.
Ini harus menjadi berkas teks dengan satu berkas atau nama direktori per baris.
Berkas ditambahkan ke direktori 'dirbuster' di bawah direkotori beranda ZAP.
