Navegação Forçada

O ZAP permite que você tente descobrir diretórios e arquivos usando a navegação forçada.
É fornecido um conjunto de arquivos com um grande número de nomes de arquivo e de diretório.
O ZAP tenta acessar diretamente todos os arquivos e diretórios listados no arquivo selecionado diretamente, em vez de depender de encontrar links para eles.

A navegação forçada é configurada usando-se a tela de opções de navegação forçada.

This functionality is based on code from the now retired OWASP DirBuster project.

Acessado via

     Guia de navegação forçada
     Aba de Sites Item de menu de clique direito ' Ataque/Forçar navegação no site'
     Aba de Sites Item de menu de clique direito ' Ataque/Forçar navegação em diretórios'
     Aba de Sites Item de menu de clique direito ' Ataque/Forçar navegação em diretórios (e filhos)'
