Penjalajahan paksa

ZAP mengizinkan anda untuk mencoba menemukan direktori dan berkas dengan menggunakan penjelajahan paksa.
Satu set dari berkas yang disediakan dapat berisi sejumlah besar nama berkas dan direktori.
ZAP mencoba untuk langsung mengakses semua berkas dan direktori yang terdaftar pada berkas yang dipilih secara langsung daripada mengandalkan pencarian tautan ke mereka.

Dipaksa Browse dikonfigurasi dengan menggunakan Pilihan Dipaksa Menelusuri layar.

This functionality is based on code from the now retired OWASP DirBuster project.

Diakses melalui

     Tab Paksa Jelajah
     Tab situs 'Serangan/Dipaksa Telusuri situs' klik kanan menu item
     Tab situs 'Serangan/Dipaksa melihat Isi direktori' klik kanan menu item
     Tab situs 'Serangan/Dipaksa melihat Isi direktori (dan anak-anak)' klik kanan menu item
