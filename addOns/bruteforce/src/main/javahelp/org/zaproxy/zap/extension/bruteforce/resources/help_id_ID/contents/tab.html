Forced Browse tab

Paksa tab Browsing yang memungkinkan anda untuk melakukan Penjalajahan paksa pindai pada setiap situs yang telah diakses.

Situs yang dapat dipilih melalui toolbar atau Situs tab.
Setiap situs yang telah yang saat ini sedang dipindai ditandai dengan bold pada toolbar Situs pulldown kontrol.
Toolbar menyediakan sebuah set tombol-tombol yang memungkinkan anda untuk start, stop, pause dan resume scan.
Sebuah progress bar menunjukkan seberapa jauh scan dari situs yang dipilih telah berkembang.
'Nilai memindai' saat ini menunjukkan berapa banyak pemindaian yang aktif saat ini - melayang di atas nilai ini akan menampilkan daftar situs yang di pindai dalam popup.
The toolbar also includes a button to export the displayed content as CSV.

Klik kanan menu

Right clicking on one or more rows will bring up a menu which will allow you to:

Keluarkan dari

This menu has the following submenus:

Proksi

This will exclude the selected nodes from the proxy. They will still be proxied via ZAP but will not be shown in any of the tabs.
This can be used to ignore URLs that you know are not relevant to the system you are currently testing.
The nodes can be included again via the Properti Sesi dialog

Pemindai

This will prevent the selected notes from being actively scanned.
The nodes can be included again via the Properti Sesi dialog

Laba-laba

This will prevent the selected nodes from being spidered.
The nodes can be included again via the Properti Sesi dialog

Salin

This will copy the selected URLs to the clipboard.

Kirim ulang...

This will bring up the Kirim ulang dialog which allows you to resend the request after making any changes to it that you want to.

Peringatan Baru...

This will bring up the Tambah dialog Peringatan which allows you to manually record a new waspada against this request.

Tampilan dalam Browser

This will open the URL of the selected node in your default browser.

Hasilkan formulir tes anti CSRF

This will open a URL which will give you a generated form for testing for CSRF issues.
It will only be enabled for POST requests, if the API is enabled and if Java supports the opening of URLs in a browser on your platform.
