Forced Browse tab

Paksa tab Browsing yang memungkinkan anda untuk melakukan Penjalajahan paksa pindai pada setiap situs yang telah diakses.

Situs yang dapat dipilih melalui toolbar atau Situs tab.
Setiap situs yang telah yang saat ini sedang dipindai ditandai dengan bold pada toolbar Situs pulldown kontrol.
Toolbar menyediakan sebuah set tombol-tombol yang memungkinkan anda untuk start, stop, pause dan resume scan.
Sebuah progress bar menunjukkan seberapa jauh scan dari situs yang dipilih telah berkembang.
'Nilai memindai' saat ini menunjukkan berapa banyak pemindaian yang aktif saat ini - melayang di atas nilai ini akan menampilkan daftar situs yang di pindai dalam popup.

Klik kanan pada salah satu baris atau lebih akan memunculkan menu yang akan memungkinkan anda untuk:

Ini menu memiliki submenu berikut:

Proksi

Ini akan mengecualikan node yang dipilih dari proxy. Mereka masih akan proksi melalui ZAP tapi tidak akan ditampilkan di setiap tab.
Hal ini dapat digunakan untuk mengabaikan URL yang anda tahu adalah tidak relevan untuk sistem anda yang sedang diuji.
Node dapat dimasukkan lagi melalui Properti Sesi dialog

Pemindai

Hal ini akan mencegah catatan yang dipilih dari aktif dipindai.
Node dapat dimasukkan lagi melalui Properti Sesi dialog

Laba-laba

Hal ini akan mencegah simpul yang dipilih agar tidak dijelajahi.
Node dapat dimasukkan lagi melalui Properti Sesi dialog

Salin

Hal ini akan menyalin URL terpilih ke clipboard.

Kirim ulang...

Hal ini akan memunculkan Kirim ulang dialog yang memungkinkan anda untuk mengirim ulang permintaan tersebut setelah membuat setiap perubahan yang anda inginkan.

Peringatan Baru...

Hal ini akan memunculkan Tambah dialog Peringatan yang memungkinkan anda untuk secara manual rekor baru waspada melawan terhadap permintaan ini.

Tampilan dalam Browser

Ini akan membuka URL yang dipilih node di browser default anda.

Hasilkan formulir tes anti CSRF

Ini akan membuka sebuah URL yang akan memberikan anda formulir yang dihasilkan selama pengujian untuk masalah CSRF.
Hal itu hanya akan diaktifkan untuk POSTING permintaan, jika API diaktifkan dan jika Java mendukung pembukaan URL di browser pada platform anda.
