Pantalla de opciones de navegación forzada

Esta pantalla le permite configurar las opciones de navegación forzada:

Subprocesos de escaneo concurrentes por host

El número de subprocesos que el escáner puede utilizará por host.
Aumentar el número de subprocesos acelerará el análisis pero puede poner presión extra en la computadora en la que ZAP está funcionando y el host objetivo.

Recursiva

Si está marcada el escáner volverá a pasar por todos los sub-directorios encontrados.
Esto puede tomar mucho tiempo.

Archivo por defecto

El archivo por defecto seleccionado cuando ZAP inicia.

Añadir archivo personalizado de Navegación Forzada

Permite añadir sus propios archivos para ser usados al ejecutar fuerza bruta en archivos y directorios.
Deben ser archivos de texto con un nombre de archivo o directorio por línea.
Los archivos se agregan al directorio "dirbuster" bajo el directorio inicial de ZAP.
