Вкладка «Принудительный просмотр»

Вкладка «Принудительный просмотр» позволяет выполнить Принудительный просмотр сканировать на любом из сайтов, которые были доступны

Сайты можно выбирать с помощью панели инструментов или Вкладка "Сайты" .
Любые сайты, которые были сканированы в настоящее время, помечены жирным шрифтом на панели инструментов.
Панель инструментов содержит набор кнопок, позволяющих запускать, останавливать, приостанавливать и возобновить сканирование.
Индикатор выполнения показывает, как далеко продвинулось сканирование выбранного сайта.
Значение «Текущие сканы» показывает, сколько сканирований в данный момент активно — зависание над этим значением во всплывающем окне будет отображаться список сканируемых сайтов.
На панели инструментов также есть кнопка для экспорта отображаемого содержимого в формате CSV.

Меню правой кнопки мыши

Щелкнув правой кнопкой мыши по одной или нескольким строкам, вы откроете меню, которое позволит вам:

Исключить из

Это меню имеет следующие подменю:

Прокси-сервер

Это исключит выбранные узлы из прокси. Они по-прежнему будут проксироваться через ZAP но не будет отображаться ни на одной из вкладок.
Это можно использовать для игнорирования URL-адресов, которые, как вы знаете, не имеют отношения к вашей системе. в данный момент проходят испытания.
Узлы могут быть снова включены через Свойства сессии диалог

Сканер

Это предотвратит активное сканирование выбранных заметок.
Узлы могут быть снова включены через Свойства сессии диалог

Паук

Это предотвратит сканирование выбранных узлов.
Узлы могут быть снова включены через Свойства сессии диалог

Копировать

Это скопирует выбранные URL-адреса в буфер обмена.

Переслать...

Это вызовет Повторно отправить диалог который позволяет повторно отправить запрос после внесения каких-либо изменений в это то, что вы хотите.

Новое оповещение...

Это вызовет Диалоговое окно «Добавить предупреждение» который позволяет вручную записывать новый предупреждение против этого запроса.

Открыть URL в браузере

Это откроет URL-адрес выбранного узла в браузере по умолчанию.

Создать тестовую форму против CSRF

Это откроет URL-адрес, который даст вам сгенерированную форму для тестирования на наличие проблем CSRF.
Он будет включен только для запросов POST, если включен API и если Java поддерживает открытие URL-адресов в браузере на вашей платформе.
