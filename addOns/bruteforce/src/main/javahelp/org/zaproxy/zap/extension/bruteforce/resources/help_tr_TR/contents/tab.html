Forced Browse tab

Forced Browse sekmesi, size, erişilen tüm siteler üzerinden Forced Browse taraması Yapmanızı sağlar.

Siteler araç çubuğu veya Siteler sekmesinden seçilebilir.
Taranmakta olan siteler araç çubuğunda kalın olarak gösterilir.
Araç çubuğu taramayı başlatmaya, durdurmaya, duraklatmaya ve devam ettirmeye yarayan bir kaç buton sağlar.
Bir ilerleme çubuğu seçili sitenin tarama işleminin ne kadar ilerlediğini gösterir.
'Geçerli taramalar' değeri şu anda kaç tane taramanın aktif durumda olduğunu ifade eder - bu değerin üzerine gelindiği zaman, açılır pencerede taranmış olan sitelerin bir listesi gösterilir.
The toolbar also includes a button to export the displayed content as CSV.

Sağ tuş menüsü

Right clicking on one or more rows will bring up a menu which will allow you to:

Şunun dışında bırak

This menu has the following submenus:

Vekil sunucu

This will exclude the selected nodes from the proxy. They will still be proxied via ZAP but will not be shown in any of the tabs.
This can be used to ignore URLs that you know are not relevant to the system you are currently testing.
The nodes can be included again via the Oturum Özellikleri dialog

Tarayıcı

This will prevent the selected notes from being actively scanned.
The nodes can be included again via the Oturum Özellikleri dialog

Ağ

This will prevent the selected nodes from being spidered.
The nodes can be included again via the Oturum Özellikleri dialog

Kopyala

This will copy the selected URLs to the clipboard.

Tekrar gönder...

This will bring up the Değişiklik yaptıktan sonra which allows you to resend the request after making any changes to it that you want to.

Yeni Uyarı...

This will bring up the Buda which allows you to manually record a new uyarı against this request.

URL'yi Tarayıcıda Aç

This will open the URL of the selected node in your default browser.

Antı CSRF test formu oluştur

This will open a URL which will give you a generated form for testing for CSRF issues.
It will only be enabled for POST requests, if the API is enabled and if Java supports the opening of URLs in a browser on your platform.
