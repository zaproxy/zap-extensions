Substituto

O substituto (replacer) é uma maneira fácil de substituir strings em solicitações e respostas.

É acessível através das Opções e, por padrão pode ser rapidamente acessado através da tecla de atalho '< ctrl > R'.

O painel de opções Substituição permite que você defina quantas regras de substituição precisar.
Cada regra é definida pelos seguintes campos:

Descrição

Uma descrição que permite identificar a regra.

Tipo de correspondência

Pode ser:

Header de solicitação (irá adicionar um caso não exista)

Neste caso a 'String' será tratada como um nome de header. Se o cabeçalho está presente então seu valor será substituído pelo texto de substituição. Se o texto de substituição está vazio então o cabeçalho será removido (se houver). Se o cabeçalho não estiver presente e o texto de substituição não está vazio então o cabeçalho será adicionado.

String do cabeçalho de solicitação

Neste caso a String' Match' será tratado como uma expressão string ou regex. Se ele estiver presente no cabeçalho da solicitação, então será substituído pelo texto de substituição.

String do corpo da solicitação

Neste caso, a 'String Match' será tratada como uma expressão string ou regex. Se ela estiver presente no cabeçalho da solicitação, então será substituída pelo texto de substituição.

Cabeçalho de resposta (irá adicionar se não existir)

Neste caso a 'String Match' será tratada como um nome de cabeçalho. Se o cabeçalho estiver presente, então seu valor será substituído pelo texto de substituição. Se o texto de substituição estiver vazio então o cabeçalho será removido (se existir). Se o cabeçalho não estiver presente e o texto de substituição não está vazio então o cabeçalho será adicionado.

String do cabeçalho de solicitação

Neste caso a String' Match' será tratado como uma expressão string ou regex. Se ele estiver presente no cabeçalho da solicitação, então será substituído pelo texto de substituição.

String do corpo da resposta

Neste caso a String' Match' será tratado como uma expressão string ou regex. Se ele estiver presente no corpo da resposta então ele será substituído pelo texto de substituição.

String de comparação/match

A sequência de caracteres que será usada para identificar o que deve ser substituído - ver tipo de correspondência para mais detalhes.

Comparação em Regex

Se definido/set, a string de comparação será tratada como uma expressão regex. Esta opção é desabilitada na comparação de cabeçalhos verdadeiros.

String de substituição

A nova sequência que substituirá a seleção especificada.

Habilitar

Se não for definido, então a regra não será aplicada.

Avançado

Se habilitado, uma aba adicional irá aparecer e permitir que você especifique a quais 'iniciadores' a regra deverá ser aplicável. Cada componente ZAP que pode enviar e receber mensagens é um iniciador separado, então isto dá-lhe um controle muito fino sobre exatamente quando a regra deve ser aplicada.

Se precisar de mais controle sobre quando e como o texto deve ser substituído, você pode usar o Console de Script ZAP que lhe dá total controle sobre as mensagens do ZAP.
