Menggantikan

Cara mudah untuk mengganti string dalam permintaan dan tanggapan.

Hal ini dapat diakses melalui Opsi dan secara default ini dapat diakses secara cepat melalui '<ctrl> R' hotkey.

Yang Replacer Pilihan panel memungkinkan anda untuk menentukan banyak aturan pengganti yang anda butuhkan.
Setiap aturan yang ditentukan oleh bidang-bidang berikut:

Deskripsi

Deskripsi yang memungkinkan anda untuk mengidentifikasi aturan.

Jenis Pencocokan

Mungkin salah satu dari:

Header Permintaan (akan menambahkan jika tidak hadir)

Dalam hal ini 'Mencocokkan String' akan diperlakukan sebagai nama header. Jika header hadir maka nilainya akan diganti dengan teks pengganti. Jika penggantian teks kosong maka header akan dihapus (jika ada). Jika header tidak hadir dan penggantian teks tidak kosong maka header akan ditambahkan.

Request Header String

Dalam hal ini 'Mencocokkan String' akan diperlakukan sebagai string atau regex ekspresi. Jika hadir di header permintaan maka akan diganti dengan teks pengganti.

Permintaan Tubuh String

Dalam hal ini 'Mencocokkan String' akan diperlakukan sebagai string atau regex ekspresi. Jika hal ini hadir dalam permintaan tubuh maka akan diganti dengan teks pengganti.

Response Header (will add if not present)

In this case the 'Match String' will be treated as a header name. Jika header hadir maka nilainya akan diganti dengan teks pengganti. Jika penggantian teks kosong maka header akan dihapus (jika ada). Jika header tidak hadir dan penggantian teks tidak kosong maka header akan ditambahkan.

Respon Header String

Dalam hal ini 'Mencocokkan String' akan diperlakukan sebagai string atau regex ekspresi. Jika hadir di respon header maka akan diganti dengan teks pengganti.

Respon Tubuh String

Dalam hal ini 'Mencocokkan String' akan diperlakukan sebagai string atau regex ekspresi. Jika hal ini hadir dalam respon tubuh maka akan diganti dengan teks pengganti.

Pertandingan String

String yang akan digunakan untuk mengidentifikasi apa yang harus diganti - melihat Jenis Pencocokan untuk rincian.

Pertandingan Regex

Jika diatur kemudian Mencocokkan String akan diperlakukan sebagai regex ekspresi. Opsi ini dinonaktifkan ketika pencocokan header.

String Pengganti

String baru yang akan menggantikan ditentukan pilihan.

Aktifkan

Jika tidak diatur maka aturan tersebut tidak akan berlaku.

Maju

Jika diatur kemudian tab tambahan akan menunjukkan yang memungkinkan anda untuk menentukan yang 'penggagas' aturan harus berlaku untuk. Masing-masing ZAP komponen yang dapat mengirim dan menerima pesan terpisah inisiator, sehingga ini akan memberikan anda sangat baik biji-bijian kontrol atas persis ketika aturan harus diterapkan.

Jika anda membutuhkan lebih mengontrol kapan dan bagaimana teks harus diganti maka anda dapat menggunakan ZAP Script Konsol yang memberi anda kontrol penuh atas ZAP pesan.
