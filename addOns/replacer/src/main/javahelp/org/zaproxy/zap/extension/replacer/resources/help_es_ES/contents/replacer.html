Reemplazador

El reemplazador es una manera fácil de reemplazar cadenas en las solicitudes y respuestas.

Es accesible a través de las Opciones y por defecto puede ser accedida a través de la tecla de acceso rápido"<ctrl>R".

El panel de Opciones del Reemplazador permite definir tantas reglas reemplazadas como sean necesarios.
Cada regla se define por los siguientes campos:

Descripción

Una descripción que permite identificar la regla.

Tipo de coincidencia

Puede ser uno de:

Encabezado de solicitud (se agrega si no está presente)

En este caso la "Cadena de coincidencia" será tratada como un nombre de encabezado. Si el encabezado está presente su valor será reemplazado por el texto de reemplazo. Si el texto de reemplazo está en blanco el encabezado se eliminará (si está presente). Si el encabezado no está presente y el texto reemplazado no está en blanco entonces se agregará el encabezado.

Cadena de encabezado de solicitud

En este caso la "Cadena de coincidencia" se tratará como una cadena o una expresión regex. Si está presente en el encabezado de solicitud, será reemplazado por el texto de reemplazo.

Cadena de cuerpo de solicitud

En este caso la "Cadena de coincidencia" se tratara como una cadena o una expresión regex. Si está presente en el cuerpo de solicitud será reemplazada por el texto de reemplazo.

Encabezado de respuesta (se agrega si no está presente)

En este caso la "Cadena de coincidencia" se considerará como un nombre de encabezado. Si el encabezado está presente su valor será reemplazado por el texto de reemplazo. Si el texto de reemplazo está en blanco el encabezado se eliminará (si está presente). Si el encabezado no está presente y el texto reemplazado no está en blanco entonces se agregará el encabezado.

Cadena de encabezado de respuesta

En este caso la "Cadena de coincidencia" se tratará como una cadena o una expresión regex. Si está presente en el encabezado de respuesta, será reemplazado por el texto de reemplazo.

Cadena de cuerpo de respuesta

En este caso la "Cadena de coincidencia" se tratará como una cadena o una expresión regex. Si está presente en el cuerpo de respuesta, será reemplazado por el texto de reemplazo.

Cadena de coincidencia

La cadena que se utilizará para identificar lo que debe ser reemplazado. Para más detalles vea Tipo de coincidencia.

Coincidencia Regex

Si se configura entonces la cadena de coincidencia se tratará como una expresión regex. Esta opción está deshabilitada al coincidir con encabezados reales.

Cadena de reemplazo

La nueva cadena que reemplazará la selección especificada.

Habilitar

Si no está establecido, entonces la regla no aplicará.

Avanzado

Si se configura aparecerá una pestaña adicional que permite especificar a cuales "iniciadores" se debe aplicar la regla. Cada componente ZAP que puede enviar y recibir mensajes en un iniciador separado, esto da un control detallado sobre cuando exactamente debe aplicarse la regla.

Si se necesita aún mas control sobre cuándo y cómo el texto debe ser reemplazado, se puede utilizar la consola de script ZAP que ofrece un control completo sobre los mensajes ZAP.
