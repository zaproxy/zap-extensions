Ang Palitan

Ang palitan ay isang madaling paraan para mapalitan ang mga string sa mga kahilingan at mga tugon.

Ito ay accessible gamit ang mga opsyon at bilang default ito ay mabilis na ma-access sa pamamagitan ng '<ctrl> R' hotkey.

Ang palitan na mga opsyon panel ay nagpapahintulot sa iyo na tukuyin ang maraming mga pinapalitang mga patakaran kapag kailangan mo.
Bawat patakaran ay tinukoy sa pamamagitan ng sumusunod na mga patlang:

Deskripsyon

Ang deskripsyon na nagbibigay-daan sa iyo upang matukoy ang patakaran.

Tugmang Uri

Maaring isa sa:

Request Header (ay magdagdag kung hindi nagpakita)

sa pagkakataong ito ang 'Tugmang String' ay ituturing bilang isang pangalan ng header. Kung ang header ay meron kung gayon ang halaga nito ay mapapalitan ng replacement na teksto. Kung ang replacement na teksto ay walang laman kung gayon ang header ay tatanggalin (kung mayroon). Kung ang header ay hindi nagpakita at ang pagpapalit ng teksyo ay hindi walang laman na header ay magdagdag.

Humiling ng Header na String

Sa pagkakataon na ito ay 'Tugma na String' ay itinuturing bilang isang string o regex na expression. Kung ito ay nagpapakita sa kahilingan na header saka ito ay nagpapalit sa pamamagitan ng pagpapalit ng teksto.

Humiling ng Katawan ng String

Sa pagakakataon na ito ay 'Tugma na String' ay itinuturing bilang isang string o regex na expression. Kung ito ay nagpapakita sa kahilingan na katawan saka ito ay nagpapalit sa pamamagitan ng pagpapalit ng teksto.

Mga Sagot sa Header (ay magdagdag kung hindi nagpapakita)

Sa pagkakataon ito ang 'Tugmang String' ay itinuturing bilang isang pangalan ng header. Kung ang header ay nagpapakita saka ang halaga nito ay mapapalitan sa pamamagitan ng pagpalit ng teksto. Kung ang pagpapalit ng teksto ay walang laman kung gayon ang header ay matatanggal (kung mayroon). Kung ang header ay hindi nagpapakita at ang pagpalit ng teksto ay hindi walang laman na header ay magdagdag.

Sagot ng Header na String

Sa pagkakaon na ito ay 'Tugma na String' ay itinuturing bilang isang string o regex na expression. Kung ito ay nagpapakita sa sagot na header saka ito ay nagpapalit sa pamamagitan ng pagpapalit ng teksto.

Sagot sa Katawan ng String

Sa pagkakataon na ito ay 'Tugma na String' ay itinuturing bilang isang string o regex na expression. Kung ito ay nagpapakita sa sagot na katawan saka ito ay nagpapalit sa pamamagitan ng pagpapalit ng teksto.

Tugmang String

Ang string na gagamitin para malaman kung ano ang dapat palita - tingnan ang Kapareho na uri para sa mga detalye.

Tugma na RegexMatch Regex

Kung itakda pagkatapos sa tugma na string ay ituturing bilang isang regex na expression. Ang opsyon na ito ay hindi pinagana kapag ang pagkumpara sa kasalukuyang mga header.

Pagpapalit ng String

Ang bagong string na magpapalit sa itinutukoy na pagpipilian.

Paganahin

Kung hindi nakatakda saka ang patakaran ay hindi mag-apply.

Advanced

Kung itakda saka ang dagdag na tab ay makikita na nagbibigay-daan sa iyo para tumukoy kung sino ang 'mga initiator' ang patakaran ay dapat naka-apply. Bawat ZAP na komponent na maaaring magpapadala at tumanggap ng mga mensahe ay isang hiwalay na pagsimula, kaya ito ay nagbibigay sa iyo ng pinaka maayos na butil na kontrol sa eksakto kapag ang patakaran ay dapat ginawa.

Kung ikaw ay nangangailangan nang mas maraming kontrol kapag at papaano ang teksto na dapat mapalitan pagkatapos ikaw ay gumamit ng ZAP Script na console na nagbibigay sayo ng kumpletong kontrol sa ZAP na mga mensahe.
