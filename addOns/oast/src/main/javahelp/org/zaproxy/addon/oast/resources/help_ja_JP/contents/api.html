OAST API

APIでは、以下の操作がサポートされています：

ビュー

  • getActiveScanService: アクティブスキャナーで使用されるサービス (存在する場合) を取得します。
  • getBoastOptions: BOASTオプションを取得します。
  • getCallbackOptions: コールバックオプションを取得します。
  • getDaysToKeepRecords: OASTレコードの保持日数を取得します。
  • getInteractshOptions: Interactshオプションを取得します。
  • getServices: すべてのサービスを取得します。

アクション

  • setActiveScanService (name*): アクティブスキャナーで使用するサービスを設定します。
    • name: サービスの名前
  • setBoastOptions (server* pollInSecs*): BOASTオプションを設定します。
    • server: サーバーURL
    • pollInSecs: ポーリング頻度 (秒)
  • setCallbackOptions (localAddress* remoteAddress* port*): コールバックオプションを設定します。
    • localAddress: ローカルアドレス
    • remoteAddress: リモートアドレス
    • port: リッスン対象のポート
  • setDaysToKeepRecords (days*): OASTレコードの保持日数を設定します。
    • days: 保持する日数
  • setInteractshOptions (server* pollInSecs* authToken*): Interactshオプションを設定します。
    • server: サーバーURL
    • pollInSecs: ポーリング頻度 (秒)
    • authToken: Interactsh認証トークン
