Interactsh

Interactsh — это решение с открытым исходным кодом для внепланового извлечения данных.

Функции

  • DNS/HTTP/SMTP Interaction support
  • Шифрование AES без ведения журнала
  • Поддержка собственного сервера
  • Автоматический TLS с подстановочными знаками на основе ACME с автоматическим продлением

Запросы, сделанные в Interactsh целевым веб-приложением, можно просмотреть в OAST Вкладка «Обратные вызовы» в графическом интерфейсе ZAP.
