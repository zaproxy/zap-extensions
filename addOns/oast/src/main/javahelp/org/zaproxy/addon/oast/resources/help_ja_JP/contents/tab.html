OAST タブ

このタブは、ZAPが発見したアウトオブバンドメッセージの概要を表示します。

メッセージ

各メッセージについて、以下を確認できます：
     ID
     受信したリクエストのタイムスタンプ
     リクエストに関連付けられたメソッド (GET、POSTなど)
     URL
     ハンドラ - コールバックリクエストを捕捉または応答したコンポーネント
     送信元 - 元のリクエスト送信元のアドレス
     リファラー - HTTP Referer ヘッダーの値 (リクエストに含まれていた場合)

ボタン

     メッセージをクリア テーブル内のすべてのメッセージをクリアします。
     今すぐポーリング 登録されているすべてのサービスをポーリングし、次回のポーリングまでの時間をリセットします。 これは、ポーリング頻度が120秒に設定されており、残り時間が30秒の時点でボタンをクリックした場合、次回の自動ポーリングは120秒後に発生することを意味します。

インジケーター

OASTタブには、外部OASTサービスのステータスインジケーターが表示されます。 灰色は、サービスがまだ登録されていないことを示します。 緑色は、サービスが登録されポーリング中であることを示します。 サービスがアクティブスキャンサービスとして登録されている場合、小さな炎のアイコンがインジケータの右下隅に追加されます。 サービス名の横にあるタイムスタンプは、サービスがポーリングされるたびに更新されます。
