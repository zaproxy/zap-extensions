Поддержка инфраструктуры автоматизации OpenAPI

Это дополнение поддерживает Automation Framework.

Надстройка добавит определения OpenAPI, если они будут найдены во время поиска, но рекомендуется добавлять их явно через URL-адрес или локальный файл, если они доступны.

Параметр targetUrl работает одинаково для 'Формата целевого URL'.

Задание: openapi

Задание openapi позволяет импортировать определения OpenAPI через URL-адрес или файл.

It is covered in the video: ZAP Chat 11 Automation Framework Part 5 - APIs. 

  - type: openapi                      # OpenAPI definition import
    parameters:
      apiFile:                         # String: Local file containing the OpenAPI definition, default: null, no definition will be imported
      apiUrl:                          # String: URL containing the OpenAPI definition, default: null, no definition will be imported
      context:                         # String: Context to use when importing the OpenAPI definition, default: null, no context will be used
      targetUrl:                       # String: URL which overrides the target defined in the definition, default: null, the target will not be overridden
