Поддержка инфраструктуры автоматизации OpenAPI

Это дополнение поддерживает Automation Framework.

Надстройка добавит определения OpenAPI, если они будут найдены во время поиска, но рекомендуется добавлять их явно через URL-адрес или локальный файл, если они доступны.

Параметр targetUrl работает одинаково для 'Формата целевого URL'.

Задание: openapi

Задание openapi позволяет импортировать определения OpenAPI через URL-адрес или файл. 
  - type: openapi                      # Импорт определения OpenAPI 
    parameters:
      apiFile:                         # String: Локальный файл, содержащий определение OpenAPI, по умолчанию: null, определение не будет импортировано. 
      apiUrl:                          # String: URL-адрес, содержащий определение OpenAPI, по умолчанию: null, определение не будет импортировано. 
      targetUrl:                       # String: URL-адрес, который переопределяет цель, указанную в определении, по умолчанию: null, цель не будет переопределена.
