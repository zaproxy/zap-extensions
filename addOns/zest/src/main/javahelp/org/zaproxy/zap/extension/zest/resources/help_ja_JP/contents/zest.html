Zest

Zest is an experimental specialized scripting language (also known as a domain-specific language) originally developed by the Mozilla security team and is intended to be used in web oriented security tools.

これは、標準のZAPに含まれています。

Engine Name

The engine is named Mozilla Zest, which should be used when manually/programmatically configuring ZAP.

Zestスクリプトの作り方

様々なZestスクリプトを作成する方法があります。

Zestスクリプトの記録ボタン

  • メインツールバーの「新しいZestスクリプトの記録」ボタンを押します。
  • 「Zestスクリプトの追加」ダイアログでスクリプトの名前を入力します。
  • プレフィックスを選択して特定のリクエストのみを記録するか、空白のままにし、すべてのリクエストを記録します。
  • セーブボタンをクリックします
  • 「新しいZestスクリプトの記録」ボタンが押されたままになり、 「新しいZestスクリプトの記録中」に変わり、赤いアイコンになります。
新しいZestスクリプトは、スクリプトタブに赤い「記録」アイコンで表示されます。
指定されたプレフィックスにマッチするすべてのリクエストが、スクリプトに追加されます。
リクエストの記録を停止するには、もう一度「新しいZestスクリプトの記録」を押してください。
この方法では、「スタンドアロン」 Zestスクリプトしか記録できないことに注意してください。 他のタイプのZestスクリプトを作成する場合は、別のメカニズムを使用する必要があります。

任意のスタンドアロンZestスクリプトを右クリックし、「記録の開始」と「記録の停止」ボタンを使用することもできます。

There are two primary methods for script recording: server-side and client-side. For client-side recording, we leverage the ZAP browser extension and a dedicated client add-on.
This combination allows us to efficiently capture and save user interactions in Zest scripts.

新しいスクリプトボタンをクリック

  • スクリプトツリータブに移動します
  • 新しいスクリプトボタンをクリックします
  • 新しいスクリプトダイアログで適切な名前を入力します
  • スクリプトタイプを選択します(詳細はスクリプトアドオンのヘルプを参照)
  • Zestスクリプトエンジンを選択します
  • テンプレートを一つ選択します(該当する場合)
  • セーブボタンをクリックします
どんなタイプのZestスクリプトもこの方法で作成することができます。

Zestテンプレートを右クリック

  • スクリプトツリータブに移動します
  • 「テンプレート」ノードを展開し、使用するテンプレートを探します。
  • テンプレートを右クリックし、 「新しいスクリプト・・・」を選択します。
  • セーブボタンをクリックします
Any type of Zest script can be created this way.

リクエストを右クリック

  • 「履歴」タブなどのリクエストが表示されているタブに移動します。
  • 1つ、または複数のリクエストを選択します。
  • 選択範囲を右クリックします。
  • 「Zest スクリプトの追加」メニューを選択すると、既存のスタンドアロンスクリプトを選択したり、新しいスタンドアロンスクリプトを作成したりすることができます。
Note that you can only add request to 'Stand Alone' Zest scripts.

Plug-n-Hack

If you are using a recent version of Firefox then you can create Zest scripts from within your browser.
  • ZAP 'クイックスタート'タブの 「Plug-n-Hack」ボタンを押します。
  • Plug-n-Hack Firefoxアドオンをインストールし、すべてのダイアログを受け入れます。
  • Firefoxで 'Shift F2'を押してDeveloper Toolbarにアクセスしてください。
  • 新しいZestスクリプトの記録を開始するために、 「zap record on global」と入力します。
  • ZAPを通じて行ったリクエストがすべてスクリプトに追加されます。
  • 「zap record off global」と入力してスクリプトの記録を停止します。
Note that you can only record 'Stand Alone' Zest scripts in this way. If you want to create other types of Zest script you must use another mechanism.

Zestスクリプトの編集

Zest scripts are edited graphically in the Scripts tree tab.
Each statement is a node in the tree - double click nodes to edit the statement properties.
You can add, move and remove statements via right clicking the Zest nodes.
You can also add requests to 'Stand alone' Zest scripts by right clicking the requests in any of the other tabs.
There are also some right click options available when you select text in the Request or Response tabs.

Zest includes a set of 'built in' variables as well as allowing you to declare your own.
A right click menu is provided (where relevant) in the edit dialogs to allow you to paste in any of the available variable names.
