Обнаружение технологий с помощью Wappalyzer

Дополнение Technology Detection использует правила Wappalyzer для обнаружения технологий, используемых приложениями.

Он работает очень похоже на надстройки браузера Wappalyzer со следующими исключениями:

  • Он не использует «глобальные переменные JavaScript», так как их трудно протестировать без «полного» браузера.
  • Он не показывает уверенность - это еще предстоит сделать
  • Это не соответствует технологиям на основе свойств DOM, так как некоторые свойства устанавливаются JavaScript в браузере после того, как ответ прошел через ZAP.
  • Это позволяет вам увидеть «доказательства», используемые для обнаружения технологий.

Вкладка «Технологии»

На этой вкладке отображаются все обнаруженные технологии для выбранного сайта.
Щелчок правой кнопкой мыши по технологии отобразит меню «Показать доказательства», в котором находятся все регулярные выражения, используемые для ее обнаружения.
При выборе регулярного выражения вы переключитесь на вкладку «Поиск» и выполните поиск по истории для этого регулярного выражения. Примечание. Если выбрано несколько строк меню не будет отображаться.

Рядом с раскрывающимся списком выбора сайта находится кнопка «Экспорт», которую можно использовать для экспорта файла CSV (значения, разделенные запятыми) на основе табличная информация, отображаемая в данный момент. Существует также кнопка-переключатель, которая позволяет пользователям легко включать/отключать Wappalyzer. пассивный сканер.

На панели инструментов есть кнопка-переключатель включения/выключения, которая определяет, работает ли правило пассивного сканирования обнаружения технологий. Это включенное состояние сохраняется между сеансами ZAP.

Составление отчетов

Технологические данные доступны для отчетов через WappalyzerJobResultData класс.
