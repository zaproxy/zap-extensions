インサイト自動化フレームワークジョブ

このアドオンは自動化フレームワークに対応しています。

ジョブ: insights

insightsジョブはインサイトオプションを設定します。 

  - type: insights
    parameters:
      messagesLowThreshold:            # Integer: メッセージ関連のインサイトに使用される低しきい値、デフォルト: 5
      messagesHighThreshold:           # Integer: メッセージ関連のインサイトに使用される高しきい値、デフォルト: 50
      memoryLowThreshold:              # Integer: メモリ関連のインサイトに使用される低しきい値、デフォルト: 80
      memoryHighThreshold:             # Integer: メモリ関連のインサイトに使用される高しきい値、デフォルト: 95
      slowResponse:                    # Integer: 「遅い」と判断されるレスポンス時間 (ミリ秒)、デフォルト: 256
      exitAutoOnHigh:                  # Boolean: true の場合、高レベルのインサイトで自動化プランを終了する、デフォルト: true
