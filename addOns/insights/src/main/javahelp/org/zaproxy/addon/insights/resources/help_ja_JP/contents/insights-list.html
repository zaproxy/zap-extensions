インサイト一覧

現在サポートされているインサイトの完全な一覧は以下の通りです：
キー 説明 サイト タイプ 低 (レベル) 高 (レベル)
insight.database.full データベースが満杯 高 高
insight.diskspace.full ディスクの空き容量が満杯 高 高
insight.memory.usage メモリ使用率 Mem 中 高
insight.log.error ZAPエラーの数 低 低
insight.log.warn ZAP警告の数 低 低
insight.network.failure ネットワーク障害の割合 Msg 低 中
insight.auth.failure 認証失敗の割合 はい Msg 低 中
insight.code.Xxx Xxxレスポンスコードの割合 はい Msg 情報 低
insight.response.slow 低速なレスポンスの割合 はい Msg 情報 低
insight.endpoint.content.CCC Content-Type CCC のレスポンスの割合 はい 情報 情報
insight.endpoint.method.MMM メソッドMMM のレスポンスの割合 はい 情報 情報
insight.endpoint.total エンドポイントの総数 はい 情報 情報

各列はインサイトのフィールドを示しており、以下の追加項目があります：

サイト

インサイトが特定のサイトに固有かどうかを示します (固有の場合はサイトが指定されます)。 固有でない場合、インサイトはサイト固有ではありません。

タイプ

インサイトのタイプでどのしきい値オプションを使用するかを示します：
  • Mem: メモリしきい値を使用します
  • Msg: メッセージしきい値を使用します
  • <空白>: しきい値を使用しません

低 (レベル)

この列には関連する低しきい値を超えた場合に報告されるレベルが表示されます。

「情報」レベルのインサイトは常に報告されます。

高 (レベル)

この列には関連する高しきい値を超えた場合に報告されるレベルが表示されます。
