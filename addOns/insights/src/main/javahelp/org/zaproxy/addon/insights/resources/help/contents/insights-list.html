The columns show the Insight fields with the following additions:

Site

Indicates if the Insight is specific to a site (which will be specified). Otherwise Insights are not site specific.

Low (Threshold : Level)

This column indicates the lowest value at which an Insight will be reported, along with the Level at which it will be reported. A value of 1 indicates that it will be reported as soon as it is greater than zero.

High (Threshold : Level)

This column indicates the value at which an Insight will be reported as a more significant issue, along with the Level at which it will be reported. A value of 1 indicates that it will be reported as soon as it is greater than zero.
