Ang tracker ng Bug

Ang Tracker ng Bug na add-on ay makakatulong sa tagagamit upang maitaas ang mga Alerto na kanilang natanggap sa katawang ng isang isyu o isang bug sa Tracker ng Bug (kagaya ng Bugzilla, Github, etc) habang nagska-skan ng web-app.

Kompigurasyon ng Tracker ng Bug

Ang tagagamit ay maaring makakompigura ng Mga Tracker ng Bug kasama ng kanilang importanteng mga detalye kagaya ng Username, Password, URL ng Repositoryo, etc gamit ang pagpunta sa Mga Pagpipilian->Tracker ng Bug at pagkatapos ay pillin ang akmang Tracker ng Bug.

Pagtataas ng mga Isyu

Ang tagagamit ay maaring makapili ng isa o mas maraming mga Alerto (kahit na ang kanilang mga subnode sa bawat kinakailangan) pagkatapos maka-scan ang tagagamit ay maaring makapili Kontekstong Menu na Item -> Itaas ang isyu sa isang Tracker ng Bug.
Isang bagong diyalogo na popup na magbibigay sa tagagamit ng abilidad na i-edit ang mga field o kahit magpili ng ilang mga field mula sa umiiral na kompigurasyon. Halimbawa: Kung ang kompigurasyon ng Github ay napili ay wala nang pagkakailangan na ilagay ang Username, Password at URL ng Repositoryo basta silang lahat ay tama. Mayroong isang dropdown na tinatawag na "Pumili ng Tracker ng Bug" na kung saan ang tagagamit ay maaring makapili ng Tracker ng Bug upang mapataas ang issue sa.
Ang mga field para sa pagtaas ng isyu ay magdedepende sa Tracker ng Bug at ng kanilang mga implementasyon.
Pagkatapos malagay lahat ng mga field ang tagagamit ay maaring ipindot ang "Save" upang maitaas ang isyu. Isang mensahe ang mapapakita sa tagagamit na magsasabi kung ang kanyang pagtangka na maipataas ang isyu ay tagumpay o hindi, at kung hindi ang rason kung bakit.
