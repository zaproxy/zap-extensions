Pelacak Bug

The Bug Tracker add-on can help the user raise the Alerts that they receive in the form of an issue or a bug on a Bug Tracker (like Bugzilla, Github,etc.) while scanning a web-app.

Konfigurasi Pelacak Bug

The user may configure the Bug Trackers with their important details like Username, Password, Repository URL, etc. by going to Options->Bug Tracker and then selecting the appropriate Bug Tracker.

Meningkatkan Masalah

Pengguna dapat memilih satu atau lebih Alerts (bahkan subnode mereka sesuai kebutuhan) setelah pemindaian sehingga pengguna dapat memilihnya Menu Konteks Item -> Tingkatkan Masalah pada Pelacak Bug.
Sebuah dialog baru akan muncul yang akan memberi pengguna kemampuan untuk mengedit fields atau bahkan memilih beberapa fields dari konfigurasi yang ada. Sebagai contoh: Jika konfigurasi Github dipilih maka tidak perlu mengisi Nama pengguna, Kata sandi, dan Repositori URL asalkan sudah benar. Ada dropdown yang disebut "Memilih Pelacak Bug" di mana pengguna dapat memilih Pelacak Bug untuk mengatasi masalah ini.
Bidang untuk mengangkat permasalahan tergantung pada Pelacak Bug dan pelaksanaannya.
Setelah mengisi kolom pengguna dapat klik "Simpan" untuk mengatasi masalah ini. Pesan kemudian akan ditampilkan ke pengguna yang menyatakan jika ia/dia mencoba untuk mengangkat masalah itu sukses atau tidak, dan jika tidak alasan untuk itu.
