Eventos enviados por el servidor

Los eventos enviados por el servidor pueden ser utilizados por aplicaciones web o sitios web para configurar un canal de comunicación uni direccional (una vía) sobre una sola conexión TCP.

Puede utilizarlo para notificaciones push de un servidor a un cliente de navegador. Los eventos enviados por el servidor a veces proveen una alternativa a usos como Ajax, HTTP Long Polling o Comet.

In the browser you can use Server-Sent Events via the EventSource API. Esta API le pertenece a HTML5 y es estandarizado mediante W3C: http://www.w3.org/TR/eventsource/


ZAP es capaz de:

  • interceptar y mostrar eventos enviados por el servidor
Los eventos enviados por el servidor son mostrados dentro del Server-Sent Events tab.
