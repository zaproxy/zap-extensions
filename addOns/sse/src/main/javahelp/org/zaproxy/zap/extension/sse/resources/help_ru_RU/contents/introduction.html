События, отправленные сервером

События, отправленные сервером, могут использоваться веб-приложениями или веб-сайтами для настройки однонаправленный (односторонний) канал связи по одному TCP-соединению.

Вы можете использовать его для push-уведомлений с сервера на клиент браузера. События, отправленные сервером, иногда представляют собой альтернативу использования Ajax, HTTP Long Polling или Comet.

В браузере вы можете использовать события, отправленные сервером, через EventSource API. Этот API принадлежит HTML5 и стандартизирован W3C: http://www.w3.org/TR/eventsource/


ZAP может:

  • перехватывать и показывать события, отправленные сервером
События, отправленные сервером, отображаются в Вкладка «Отправленные сервером события» .
