Server-Sent Events

Server-Sent Events dapat digunakan oleh aplikasi web atau situs web untuk menetapkan saluran komunikasi uni-directional (satu arah) melalui koneksi TCP tunggal.

Anda dapat menggunakannya untuk menekan pemberitahuan dari server ke peramban klien. Server-Sent Events terkadang memberikan alternatif penggunaan Ajax, HTTP Long Polling atau Comet.

Pada browser anda bisa menggunakan Server-Sent Events melalui EventSource API. API ini termasuk ke HTML5 dan distandarisasikan oleh W3C: http://www.w3.org/TR/eventsource/
