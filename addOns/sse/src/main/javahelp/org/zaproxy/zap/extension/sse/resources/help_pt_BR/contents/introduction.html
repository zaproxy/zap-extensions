Eventos Enviados pelo Servidor

Eventos Enviados pelo Servidor podem ser usados por aplicativos da web ou sites da web para configurar um canal de comunicação unidirecional sobre uma conexão TCP.

Você pode usá-los para fazer push de notificações de um servidor para um navegador. Eventos enviados pelo servidor às vezes são uma alternativa ao uso de Ajax, HTTP Long Polling ou Comet.

No navegador, você pode usar esses eventos pelo EventSource API. Essa API pertence ao HTML5 e é padronizada pelo W3C: http://www.w3.org/TR/eventsource/


O ZAP é capaz de:

  • interceptar e mostrar os eventos Server-Sent
Eventos enviados pelo servidor são exibidos dentro da Aba de Eventos Enviados pelo Servidor.
