Mga pangyayari ng Server-Na[adala

Ang Server-Napadala na mga pangyayari ay pwedeng magamit sa pamamagitan ng mga web aplikasyon o mga web site para i-setup ang isang uni-direksyonal (isang-daan) na komunikasyon channel sa ibabaw ng isang TCP na koneksyon.

Ikaw ay pwedeng gumamit nito para itulak ang mga notipikasyon mula sa isang server hanggang sa browser client. Ang Server-Napadala na mga pangyayari ay minsan nagbibigay ng alternatibong mga pagamit ng Ajax, HTTP Long Polling o Comet.

Sa browser ikaw ay makagamit ng Server-Napadala na mga pangyayari sa pamamagitan ng EventSource API. Ang API na ito ay galing sa HTML5 at naka-standardized gamit ang W3C: http://www.w3.org/TR/eventsource/
