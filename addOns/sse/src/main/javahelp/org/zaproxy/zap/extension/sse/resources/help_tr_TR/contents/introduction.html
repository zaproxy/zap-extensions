Sunucu-Gönderimli Olaylar

Sunucu Tarafından Gönderilmiş Olaylar web uygulamaları tarafından veya tek bir TCP bağlantısı üzerinden tek yönlü iletişim kanalı açan web siteleri tarafından kullanılabilir.

Bir sunucudan bir tarayıcı istemcisine olan anlık bildirimler için kullanabilirsiniz. Sunucu Tarafından Gönderilmiş Olaylar Ajax, HTTP Long Polling ve Comet kullanımına bazen bir alternatif sunabilir.

EventSource API aracılığıyla Sunucu Tarafından Gönderilmiş Olayları tarayıcıda kullanabilirsiniz. Bu API HTML5'e aittir ve W3C tarafından standart hale getirilmiştir: http://www.w3.org/TR/eventsource/


ZAP şunları yapabilir:

  • sunucu Tarafından Gönderilmiş olan Olayları Engelleme ve Gösterme
Sunucu Tarafından Gönderilen Etkinlikler, Sunucu-Gönderimli Olaylar sekmesi.
