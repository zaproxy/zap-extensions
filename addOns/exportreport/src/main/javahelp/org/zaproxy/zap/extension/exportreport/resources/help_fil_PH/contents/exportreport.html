Ang pag Export ng Ulat

Ang pag I-export ng Ulat na modyul ay nag papahintulot sa mga gumagamit para ipasadya ang mga nilalaman at pag i-export sa isang nais na ayos.
  • Nilikha ayon sa JordanGS.
  • Pinakamaliit na Suporta ng Bersyon: Lingguhang Pag-labas ng ZAP_D-2016-09-05
  • Proyekto sa GitHub<0>. Maaari mo ring hanapin ang pinaka maliit na suporta ng .zip na nasa zap-download na polder.
  • Sinuportahan at inilakip sa mga Opisyal na OWASP Zed Attack Proxy Jenkins Plugin<0>.

Talaan ng mga Nilalaman

  • Ang Zap Ui
  • Linya ng Kautusan
  • Mga Tawag sa API

Ang ZAP UI

Pinagmulan ng Inpormasyon

Ang gumagamit ay pumasok at awtomatikong makuha ang mga datos kaugnay sa mga iniulat. Ang bawat field ay tumapat sa mga dugtungan para sa mga kagamitan ng tip.
  • Ang pamagat - Ang mga Default na pangalan sa sesyon
  • Ayon sa
  • Para sa
  • Pag-susuri sa Petsa - ang gumagamit ay pumasok sa petsa ng AScan, mga default sa pangkasalukuyang petsa-oras
  • Ang petsa ng ulat - mga default sa pangkasalukuyang petsa-oras
  • Ang pag-iskan sa Bersyon - Placeholder para sa hinaharap
  • Ang pag-ulat sa Bersyon - Mga Default sa pangkasalukuyang bersyon ng kasangkapan ng ZAP

Ang diyalogo ay makikita lamang sa mga polder at sa mga klase ng file na tinanggap. Ang mga 'Klase ng File' na naibaba na nasa listahan ay salain upang ipakita lamang ang mga polder at mga file na tinukoy na ekstensiyon.

Hanay ng Kautusan

Ang ekstensiyon ay maaari din tumakbo mula sa hanay ng kautusan at nangangailangan ng mga sunmusunod na mga argumento para pumasa sa isang ulat na nililikha.

Ang mga opsyon sa hanay ng kautusan ng OWASP ZAP

  • -sesyon: ang binigay na sesyon ay nagbubukas pagkatapos magsimula ng ZAP
  • -cmd: tumatakbo sa ZAP 'inline', ie na walang simula ng UI o ng isang daemon
Tingnan ang Wiki para sa mga ilang detalye sa mga datihang suportado ng linya ng kautusan sa mga opsiyon.

Ekstensiyon sa pag Export ng Ulat sa mga opsyon ng Hanay ng Kautusan

  • -I-export_I-ulat:

    • Ang Pagsasalarawan:

      • I-seyb sa tinukoy na file pagkatapos isalin ang ibinigay na sesyon.

    • Kasalukuyang magagamit

      • ASCII 1.0 Ang mahigpit na Pag sang-ayon sa XHTML na mga File (.xhtml )
      • Mga File ng XML (.xml )
      • Mga File ng JSON (.json )

    • Ang Milyahe sa Hinaharap

      • Ang mga File ng Bootstrap HTML(.bootstrap.html )
      • Ang mga Dokumento ng PDF (.pdf )
      • Ang mga Dokumento ng Google (.doc )

  • -pinagmulan_impormasyon:

    • Ang Pagsasalarawan:

      • Tumutukoy ang mga sumusunod sa mga ulat:
        1. Pamagat
        2. Ayon sa
        3. Para sa
        4. Suriin ang petsa
        5. I-ulat ang petsa
        6. Suriin ang ver
        7. I-ulat ang ver
        8. Ang paglalarawan

    • Mga Halimbawa:

      • -pinagmulan_impormasyon"kahinaan ng ulat ng MyApp.com;JordanGS;Nawawalang Espiritu; Ika-15 ng Agosto, 2016;Agosto 18, 2016;ZAP_D-2016-08-15;ZAP_D-2016-08-15;Lorem dolor umupo amet, pri corpora ancillae adolescens sa. Ea usu atomorum tincidunt, ne munere regione mayroon. Ne dagat summo tation, et sed nibh nostrum singulis. Nec causae viderer discere eu."

    • Pagbibigay Bisa:

      • Ang nilalaman ay walang check, maaaring pumasok sa mga field na bakante kung ibig mo, ang walong aytem lamang ang lahat na nakalista na nasa kundisyon.

  • -alerto_pag-hihigpit:

    • Ang Pagsasalarawan:

      • Tukuyin kung alin ang mga mahigpit na pag alerto na nakasama sa mga inulat:
        1. Ang Mataas
        2. Ang Katamtaman
        3. Ang Mababa
        4. Ang Impormasyon

    • Paglalagay ng format:

      • Tinatanggap lamang ang isang hanay ng listahan na may ';' delimeter

    • Ang pag input ng limitasyon:

      • Tumatanggap lamang ng 't' at 'f' sa bawat aytem na nasa listahan

    • Halimbawa:

      • -alerto_paghihigpit "t;t;f;t"

    • Pagpapaliwanag:

      • Ang nasa itaas na halimbawa, tanging mataas, katamtaman at mga impormasyon ng pag alerto na nakasama sa mga binubuong ulat.

    • Pagbibigay Bisa:

      • Pagbibigay Bisa: ang nilalaman ay binibigyan ng bisa maging alin man sa 't' o kaya'y 'f' at sa apat na mga aytem na nasa listahan.

  • -alerto_mga detalye:

    • Pagsasalarawan:

      • Tukuyin kung alin sa mga detalye ng alerto ay nakasama sa ulat:
        1. Ang ID ng CWE
        2. Ang ID ng WASC
        3. Ang Pagsasalarawan
        4. Ang iba pang impormasyon
        5. Ang solusyon
        6. Ang reperensiya
        7. Ang kahilingan ng Header
        8. Ang pagtugon ng Header
        9. Ang kahilingan ng Katawan
        10. Ang kahilingan ng Katawan

    • Pag input ng Format:

      • Tanging ang tinatanggap ng isang hanay ng nakalista ay may ';' delimeter

    • Pag input ng limitasyon:

      • Ang tanging tinatanggap lamang ay 't' at 'f' sa bawat item na nakalista

    • Halimbawa:

      • -alerto_mga detalye "t;t;t;t;t;t;f;f;f;f"

    • Pagpapaliwanag:

      • Ang nasa itaas na halimbawa, tanging ang ID ng CWE, ID ng WASC, ang Pagsasalarawan, ang iba pang Impormasyon, ang slusyon at ang reperensiya ay mga alertong detalye na isasama sa pagbuo ng ulat.

    • Pagbibigay bisa:

      • Ang nilalaman ay binibigyan ng bisa maging alin man sa 't' o kaya'y 'f' at sa lahat ng sampung item na nasa listahan.

  • Sama-samang lahat sa paglalagay! (mga Window)

    1. Buksan ang isang command prompt
    2. I-navigate ang lokasyon kung saan ang zap.bat na file ay makikita
    3. isagawa lahat ng mga sumusunod:
      • zap.batexport_ulat "/daan/sa/seyb/ulat.xtml" -pinagmulan_impormasyon"kahinaan na ulat ng MyApp.com;jordanGS;nawawalang espiritu;Agosto 15, 2016;Agosto 18, 2016;ZAP_D-2016-08-15;ZAP_D-2016-08-15;Lorem ipsum dolor sit amet,pri corpora ancillae adolecence in. Ea usu atomorum tincidunt, ne munere regione has. Ne dagat summo tation, et sed nibh nostrum singulis. Nec causae viderer discere eu." -alerto_paghihigpit "t;t;f;t" -alerto_mga detalye "t;t;t;t;t;t;f;f;f;f" -sesyon "/daan/sa/umiiral/sesyon" -cmd

Mga Tawag sa API

Ang ekstensiyon ay maaaaring gawing daan para sa mga tawag ng API at nangangailangan ng mga sumusunod ng argumento para maging pasado sa binubuo na isang ulat.

Ang paghihigpit ay katulad ng nasa itaas na hanay ng command.

Halimbawa ng Pag gamit:


Mapa<String, String> mapa = bagong HashMap<String, String>();
Hanay ng API_KEY = "ZAPROXY-PLUGIN"
Hanay na absolutePath = "/daan/sa/i-seyb/i-ulat.xhtml";
Hanay sa ekstensiyon = "xhtml";
Hanay ng mga sourceDetail = "Kahinaan sa Ulat ng MyApp.com:JordanGS;Nawawalang mga Espiritu;" +
  "Agosto 15, 2016;Agosto 18 2016;ZAP_D-2016-08-15:zap_D-2016-08-15;" +
  "Lorem ipsum dolor sit amet,pri corpora ancillae adolescens in. Ea " + 
   "usu atomorum tincidunt, ne munere regione has. Ne sea summo tatio," +
   "et sed nibh nostrum singulis. Nec causae viderer discere eu.";
Hanay sa alertSeverity = "t;t;f;t";
Hanay sa alertDetails = "t;t;t;t;t;t;f;f;f;f";

map.put("apikey", API_Key);

// Hanay na kumatawan sa tiyak na daan para i-seyb ang destinasyon para sa iniulat
map.put("absolutePath)", absolutePath)

// Hanay na kumakatawan sa ekstensiyon ng mga inulat
map.put("fileExtension", extension);

// Hanay na pinaghiwalay ng ; sa bawat isang ipinasok
map.put("sourceDetails", sourceDetails);

// hanay na pinaghiwalay ng ; sa bawat isang ipinasok
