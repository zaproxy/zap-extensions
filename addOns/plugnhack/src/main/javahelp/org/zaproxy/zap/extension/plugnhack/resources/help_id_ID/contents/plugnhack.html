Plug-n-Hack

Plug-n-Hack adalah standar yang diusulkan dari tim keamanan Mozilla untuk menentukan bagaimana alat keamanan dapat berinteraksi dengan peramban dengan cara yang lebih bermanfaat dan bermanfaat.

Add-on ini menambahkan tombol 'Plug-n-Hack' ke tab Quickstart - mengklik tombol ini akan memungkinkan Anda mengkonfigurasi browser Anda untuk bekerja dengan ZAP dengan cepat dan mudah.

Juga memungkinkan anda untuk mengontrol ZAP dari browser anda. Didalam Firefox ini adalah dukungan melalui Developer Toolbar (Shift+F2).
Ketik 'help pnh' atau 'help zap' pada toolbar Pengembang Firefox untuk memulai.

Plug-n-Hack saat ini didukung oleh Firefox 24.0 dan yang lebih baru.
Untuk rincian selengkapnya tentang Plug-n-Hack termasuk informasi terbaru tentang peramban dan alat yang mendukungnya, lihat: https://developer.mozilla.org/en-US/docs/Plug-n-Hack
