Plug-n-Hack

Plug-n-Hack é um padrão proposto pela equipe de segurança do Mozilla para definir como ferramentas de segurança podem interagir com navegadores de uma forma mais útil e prática.

Este complemento adiciona um botão de 'Plug-n-Hack' na Aba de Início Rápido - clicar neste botão permitirá que você configure seu navegador para trabalhar com o ZAP rápida e facilmente.

Ele também permite que você controle o ZAP por meio do seu navegador. No Firefox o suporte para isso está na barra de ferramentas do desenvolvedor (Shift + F2).
Digite 'help pnh' ou 'help zap' na barra de ferramentas da desenvolvedor do Firefox para começar.

Plug-n-Hack atualmente é suportado pelo Firefox 24,0 e posteriores.
Para mais detalhes sobre o Plug-n-Hack, incluindo as informações mais recentes sobre navegadores e ferramentas de apoio veja: https://developer.mozilla.org/en-US/docs/Plug-n-Hack
