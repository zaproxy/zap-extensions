Поддержка платформы автоматизации SOAP

Это дополнение поддерживает Automation Framework.

Надстройка будет импортировать файлы WSDL, содержащие конечные точки SOAP, если они будут найдены во время поиска, но рекомендуется добавлять их явно через URL-адрес или локальный файл, если они доступны.

Задание:soap

Задание soap позволяет импортировать файлы WSDL локально или по URL-адресу. 
  - type: soap                         # SOAP WSDL импорт
    parameters:
      wsdlFile:                        # String: Путь к локальному файлу WSDL, по умолчанию: null, определение не будет импортировано. 
      wsdlUrl:                         # String: URL-адрес, указывающий на WSDL, по умолчанию: null, определение не будет импортировано.
